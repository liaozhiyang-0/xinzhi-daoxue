# 教师端看板原型

## 页面目标

帮助教师快速掌握班级学习情况，发现共性薄弱知识点，并生成可执行的教学建议。

## 页面模块

### 1. 高频问题 Top 10

- 展示学生提问频次最高的问题。
- 支持按课程、班级和时间范围筛选。
- 字段包括问题摘要、关联知识点、提问次数和最近提问时间。

### 2. 错题类型分布

- 使用饼图或柱状图展示错误类型占比。
- 错误类型包括判断条件遗漏、概念混淆、公式误用、步骤缺失和时序分析错误。

### 3. 薄弱知识点统计

- 使用柱状图展示薄弱知识点排名。
- 支持点击知识点查看典型错题和关联问题。

### 4. 学生活跃度统计

- 展示每日问答次数、错题提交次数和活跃学生数。
- 支持折线图查看趋势。

### 5. 教学建议文本

- 展示教师分析 Agent 生成的建议。
- 内容包括主要风险、课堂讲解建议、作业建议和后续观察指标。

## 页面数据来源

- `/api/v1/teacher/stats/high-frequency-questions`
- `/api/v1/teacher/stats/error-types`
- `/api/v1/teacher/stats/weak-knowledge-points`
- `/api/v1/teacher/stats/activity`
- `/api/v1/teacher/teaching-suggestion`
