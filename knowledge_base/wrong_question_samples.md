# 错题样例

## 样例 1：MOS 管饱和区判断遗漏条件

- 题目：判断 NMOS 是否工作在饱和区。
- 学生答案：只要 `VGS > Vth` 就是饱和区。
- 错误类型：判断条件遗漏。
- 诊断说明：`VGS > Vth` 只能说明 MOS 管导通，还必须判断 `VDS >= VGS - Vth`。
- 建议：复习 MOS 管三个工作区的完整判断流程。

## 样例 2：触发器时序图忽略有效边沿

- 题目：根据 D 触发器输入和时钟绘制输出。
- 学生答案：输入 D 变化时输出立即变化。
- 错误类型：时钟触发机制理解错误。
- 诊断说明：边沿触发 D 触发器只在有效时钟边沿采样输入。
- 建议：结合特性表和时序图训练边沿采样分析。
