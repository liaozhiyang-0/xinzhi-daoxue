# 星辰 Agent 知识库上传计划

## 上传原则

- 先上传课程骨架和总览，后上传章节和题库。
- 先上传 P0 课程，完成问答与错题诊断基本覆盖。
- 每批上传后保留测试问题和测试结果，便于回归验证。
- 旧资料在迁移确认前仅作为参考，不直接作为正式入口。

## 第一批

课程范围：

- 电路理论
- 模拟电子技术
- 数字电子技术
- 半导体器件 / 微电子器件
- CMOS 数字集成电路设计

目标：支撑第一版智能答疑、错题诊断和学习建议演示。

## 第二批

课程范围：

- 信号与系统
- 通信原理
- 数字信号处理
- 数字集成电路设计 / ASIC 设计基础
- 模拟集成电路设计

目标：扩展电子信息主干课程和 IC 设计课程覆盖面。

## 第三批

课程范围：

- EDA 技术与芯片设计流程
- 集成电路版图设计
- 集成电路制造工艺
- 先进封装与测试
- 片上系统 SoC 设计
- 射频集成电路设计
- 功率半导体与功率集成电路

目标：补充工程实践、制造测试和专业深化方向资料。

## 上传后验证

每批上传后至少验证：

- 课程定位问答；
- 章节目录问答；
- 典型题型检索；
- 错题诊断样例匹配；
- 学习建议生成是否能引用课程结构。
