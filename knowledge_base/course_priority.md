# 课程建设优先级

## 第一阶段：课程骨架与 P0 课程总览

目标是建立完整课程目录骨架，并优先补充 P0 课程的课程总览、章节目录、典型题型框架和错题模式框架。

P0 课程：

- 电路理论
- 模拟电子技术
- 数字电子技术
- 信号与系统
- 通信原理
- 半导体物理
- 半导体器件 / 微电子器件
- CMOS 数字集成电路设计

## 第二阶段：P1 课程章节与题库

目标是补充 P1 课程章节级内容，并为 P0 与 P1 课程建立典型题型和错题诊断样例。

P1 课程：

- 数字信号处理
- 高频电子线路
- 信息论与编码
- 集成电路制造工艺
- 模拟集成电路设计
- 数字集成电路设计 / ASIC 设计基础
- 集成电路版图设计
- EDA 技术与芯片设计流程
- Verilog / SystemVerilog
- FPGA 与数字系统设计

## 第三阶段：P2 课程扩展与专业深化

目标是扩展专业深度和工程实践覆盖范围，逐步完善复杂系统、先进工艺和应用方向课程资料。

P2 课程：

- 电磁场与电磁波
- 嵌入式系统
- 集成电路测试
- 先进封装与测试
- 片上系统 SoC 设计
- 射频集成电路设计
- 功率半导体与功率集成电路
