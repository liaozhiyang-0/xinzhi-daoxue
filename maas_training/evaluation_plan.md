# MaaS 评估方案

## 评估目标

评估模型在电子信息课程群中的准确性、解释性、结构化输出能力和教学可用性。

## 评估数据

- 概念问答样例。
- 计算与判断类题目。
- 错题诊断样例。
- 学习建议样例。
- 教师分析样例。

## 关键指标

| 指标 | 说明 |
| --- | --- |
| 知识准确性 | 公式、条件和结论是否正确 |
| 推理完整性 | 是否给出必要步骤和判断依据 |
| 易错点覆盖 | 是否指出常见错误和注意事项 |
| 结构化程度 | 输出是否便于前端展示和入库 |
| 教学可用性 | 建议是否具体、可执行、符合教学场景 |

## 评估流程

1. 固定评估样例集和随机种子。
2. 调用基线模型生成结果。
3. 调用微调模型生成结果。
4. 使用人工评分表对比两组输出。
5. 汇总薄弱题型，继续补充训练数据。
