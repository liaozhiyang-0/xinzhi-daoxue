# MaaS 微调数据格式

## 数据目标

微调数据用于提升模型在电子信息课程群中的垂类问答、错题诊断、学习建议和教师分析能力。第一版仅提供样例格式和数据规范，不包含真实学生隐私数据。

## JSONL 格式

每行一条样例，包含 `instruction`、`input` 和 `output` 三个字段。

```json
{"instruction":"你是电子信息课程群智能导学助手，请回答学生问题。","input":"学生问题或任务上下文","output":"期望模型输出"}
```

## 数据要求

- 不包含真实姓名、学号、手机号、邮箱等隐私信息。
- 问题、答案和诊断过程应符合课程知识。
- 输出应清晰、可解释，适合学生或教师直接阅读。
- 错题诊断类样例应说明错误类型和修正建议。
- 教师分析类样例应给出可执行的教学建议。
