## 3. 知识库组织方式

```text
课程群
→ 课程
→ 章节
→ 知识单元
→ 典型题型
→ 错题模式
→ 学习路径
```

## 4. 每门课程目录规范

```text
课程目录/
├── README.md
├── 00_course_overview.md
├── chapters/
│   └── README.md
├── questions/
│   └── README.md
└── wrong_cases/
    └── README.md
```

## 5. 上传星辰 Agent 知识库建议

1. 先上传课程总览和章节目录，建立课程边界。
2. 再上传章节知识单元，保证文件粒度适合检索。
3. 典型题型和错题样例应与章节知识点建立映射。
4. 文件命名保持稳定，便于后续增量更新。
5. 旧模板资料统一放在 `_legacy/`，迁移完成前不直接作为正式知识库入口。
