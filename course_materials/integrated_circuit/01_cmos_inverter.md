# CMOS 反相器

## 1. 基本概念

CMOS 反相器由一个 PMOS 上拉网络和一个 NMOS 下拉网络组成，是 CMOS 数字集成电路中最基本的逻辑单元。当输入为低电平时，PMOS 导通、NMOS 截止，输出为高电平；当输入为高电平时，NMOS 导通、PMOS 截止，输出为低电平。

## 2. 核心公式或判断条件

- 输入低电平：`Vin = 0`，输出高电平 `Vout ≈ VDD`。
- 输入高电平：`Vin = VDD`，输出低电平 `Vout ≈ 0`。
- 静态功耗理想情况下接近 0，但翻转过程和漏电会产生功耗。

## 3. 分析方法

1. 根据输入电平判断 PMOS 和 NMOS 的导通状态。
2. 根据导通路径判断输出节点连接到 `VDD` 还是 `GND`。
3. 结合电压传输特性分析噪声容限和开关阈值。
4. 结合负载电容分析传播延迟和动态功耗。

## 4. 常见易错点

- 混淆 PMOS 与 NMOS 的导通条件。
- 忽略输入翻转期间可能存在短路电流。
- 只分析逻辑功能，不分析噪声容限和延迟。
- 将静态功耗和动态功耗概念混淆。

## 5. 典型例题

输入为 0 时，PMOS 导通、NMOS 截止，输出被上拉到 `VDD`，逻辑输出为 1。输入为 1 时，NMOS 导通、PMOS 截止，输出被下拉到地，逻辑输出为 0。

## 6. 推荐复习建议

建议结合开关模型、电压传输曲线、功耗构成和传播延迟进行复习，理解 CMOS 反相器为何适合作为数字电路基础单元。
