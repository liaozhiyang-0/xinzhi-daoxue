# 触发器

## 1. 基本概念

触发器是数字电路中能够存储一位二进制状态的基本时序单元。常见类型包括 SR 触发器、D 触发器、JK 触发器和 T 触发器。触发器的输出不仅与当前输入有关，还与电路原有状态有关。

## 2. 核心公式或判断条件

- D 触发器：时钟有效边沿到来后，`Q(n+1) = D`。
- T 触发器：`T = 0` 时保持，`T = 1` 时翻转。
- JK 触发器：`J = K = 0` 保持，`J = 1, K = 0` 置 1，`J = 0, K = 1` 置 0，`J = K = 1` 翻转。

## 3. 分析方法

1. 确认触发器类型和时钟触发方式。
2. 找到有效时钟边沿或有效电平。
3. 根据输入值和特性表推导下一状态。
4. 绘制状态转移或时序波形。

## 4. 常见易错点

- 忽略触发器只在有效边沿或有效电平更新。
- 将锁存器和边沿触发器混淆。
- 时序图中没有考虑传播延迟。
- JK 触发器翻转条件记忆不清。

## 5. 典型例题

D 触发器在上升沿触发。若时钟上升沿到来前 `D = 1`，则上升沿后 `Q` 更新为 1；在两个上升沿之间，`Q` 保持不变。

## 6. 推荐复习建议

建议结合特性表、激励表和时序图复习，重点训练“看时钟边沿再看输入”的分析习惯。
