# Agent 测试用例草稿 README (draft)

## 目录说明

本目录存放《电路理论》课程的**Agent 能力测试用例草稿**，用于验证课程 Agent 在概念问答、公式条件判断、错题诊断和综合追问场景中的表现。

## 当前状态

- 当前版本：**v0.1 draft**
- 文件：
  - `agent_test_cases_v0_1_draft.md` — 测试用例
  - `dataset_generation_report_v0_1.md` — 生成报告

## 内容说明

- 测试用例分为四类：
  1. **概念问答** — 测试 Agent 对基本概念的解释能力；
  2. **公式条件** — 测试 Agent 对公式适用条件的理解；
  3. **错题诊断** — 测试 Agent 识别学生错误并给出反馈的能力；
  4. **综合追问** — 测试 Agent 在多轮对话中保持上下文一致性的能力。

## 使用方式

1. 逐个测试：将用户输入提交给课程 Agent，记录其回答；
2. 对照评价标准打分：每个用例包含具体评价标准；
3. 汇总结果：分析 Agent 在各类场景中的表现和不足；
4. 针对性优化：根据测试结果调整知识库或诊断规则。

## 注意事项

- 当前为草稿，测试用例可后续补充更多章节和场景；
- 评价标准可根据实际 Agent 能力要求调整；
- 建议在正式上线前完成全部测试用例的验证。
