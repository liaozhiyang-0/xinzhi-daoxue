# 电路理论电路结构题设计规范

> 适用范围：`course_materials/01_circuit_theory/questions/` 后续题库生成、人工审核和 Agent 测试集设计。
>
> 核心原则：电路理论题目不能只写纯文字结论或公式条件。凡涉及电路分析、暂态、相量、功率、三相、互感和变压器的题目，都应提供可还原的电路结构。

## 1. 推荐题目结构

每道涉及电路分析的题目建议包含：

- 电路结构图或 ASCII 图；
- 节点编号；
- 电压、电流、功率或磁耦合的参考方向；
- 元件参数；
- 开关动作、激励条件或频率条件；
- 要求计算的目标量；
- 可触发的错因标签；
- 标准答案中的关键方程；
- 可用于诊断的学生错误表现；
- 指向章节文件和题型编号的复习入口。

推荐最小模板：

```markdown
### QCT2-xx-xxx 题目标题

- 所属章节：
- 对应题型：
- 难度：
- 主要知识点：
- 关联错因标签：
- 推荐复习文件：

#### 电路结构

使用 ASCII 图、节点表、支路表、netlist、相量关系或同名端说明之一。

#### 题目

明确目标量和参考方向。

#### 标准答案

先给关键方程，再给结果。

#### 解题思路

1. 识别结构；
2. 建立方程；
3. 代入参数；
4. 检查单位、符号和物理意义。

#### 易错点

列出与错因标签一一对应的错误。

#### 可用于诊断的学生错误表现

说明学生答案中出现什么现象时应触发哪个标签。

#### 推荐反馈语

指出具体错误位置，而不是只说“公式用错”。
```

## 2. 电路表达方式

### 方式 A：ASCII 电路图

```text
        R1=2kΩ
  +---/\/\/\/---o vC(t)
  |             |
 Us=10V        C=100uF
  |             |
  +-------------+
```

适用场景：
- RC/RL/RLC 暂态；
- 开关换路；
- 简单交流串并联；
- 端口等效。

设计要求：
- 标出节点名或端口名；
- 标出状态变量方向，如 `vC(t)`、`iL(t)`；
- 开关题要标出 t<0 与 t>=0 的连接状态。

### 方式 B：节点连接表

```text
节点 A：连接电压源正端、R1 左端
节点 B：连接 R1 右端、C 上端
节点 0：参考地，连接电压源负端、C 下端
```

适用场景：
- 节点法；
- 不容易用 ASCII 图画清楚的网络；
- 受控源或多支路电路。

设计要求：
- 节点名保持唯一；
- 明确参考地；
- 受控源必须写清控制量所在支路。

### 方式 C：支路参数表

```text
| 支路 | 起点 | 终点 | 元件 | 参数 | 参考方向 |
|---|---|---|---|---|---|
| b1 | A | B | R1 | 2kΩ | A -> B |
| b2 | B | 0 | C | 100uF | B -> 0 |
```

适用场景：
- 支路电流法；
- 多支路网络；
- Agent 自动解析或检索。

设计要求：
- 每个目标电压/电流必须能映射到支路；
- 电流方向与标准答案中的 KCL/KVL 保持一致；
- 支路参数单位必须完整。

### 方式 D：netlist 风格

```text
V1 A 0 DC 10V
R1 A B 2k
C1 B 0 100uF
```

适用场景：
- 需要机器解析的题库；
- 节点法/端口等效；
- 可扩展到仿真或自动检查。

设计要求：
- 第一列为元件名；
- 第二、三列为连接节点；
- 后续列为类型、参数、源值或控制关系；
- 受控源建议增加注释说明控制量。

### 方式 E：相量/三相/磁耦合特殊说明

三相相量示例：

```text
三相电源为正序：
Ua = 220∠0° V
Ub = 220∠-120° V
Uc = 220∠120° V
```

磁耦合示例：

```text
L1 与 L2 互感 M，两个线圈上端为同名端。
i1 从 L1 同名端流入，i2 从 L2 非同名端流入。
```

适用场景：
- 正弦稳态；
- 三相线量/相量关系；
- 功率测量；
- 同名端、互感、理想变压器。

设计要求：
- 明确有效值相量还是幅值相量；
- 三相题必须写相序；
- 互感题必须写同名端和电流参考方向；
- 功率题必须写负载吸收功率或电源发出功率的约定。

## 3. 各章节建议电路表达重点

### 第8章：一阶电路的暂态分析

重点表达：
- 开关位置和换路时刻；
- 电容电压或电感电流的参考方向；
- 换路前后的等效电路；
- 从储能元件端口看入的等效电阻；
- 受控源保留和测试源方向。

优先题型：
- 含受控源求时间常数；
- 多次换路；
- 电容电压连续但电容电流可突变；
- 电感电流连续但电感电压可突变。

### 第9章：二阶电路的暂态分析

重点表达：
- RLC 串联或并联结构；
- 状态变量 `uC(t)`、`iL(t)`；
- 初始值和导数初值来源；
- 阻尼参数 `alpha`、`omega0`、`omegad`；
- 冲激源位置和状态跳变条件。

优先题型：
- RLC 串联欠阻尼；
- RLC 并联过阻尼；
- 用初值和导数初值求二阶响应系数。

### 第10章：正弦稳态分析

重点表达：
- 频率或角频率；
- 有效值相量约定；
- 节点法/网孔法中的阻抗或导纳；
- 参考相量；
- 输入/输出端口。

优先题型：
- 含 RLC 的相量网络；
- 交流节点法；
- 相量图判断相位；
- RC 移相电路。

### 第11章：正弦稳态电路的功率

重点表达：
- 负载阻抗；
- 电压电流有效值相量；
- 吸收功率或发出功率约定；
- 复功率计算使用 `S = U * I^*`；
- 功率因数校正电容接入位置；
- 戴维南端口和共轭匹配条件。

优先题型：
- 复功率共轭；
- 多负载复功率相加；
- 功率因数校正电容计算；
- 交流最大功率传输。

### 第12章：三相正弦稳态电路

重点表达：
- 电源相序；
- Y/Delta 连接；
- 线电压、相电压、线电流、相电流的参考；
- 中性点和中线状态；
- 二表法功率表接线和读数符号。

优先题型：
- Y 接负载线电压/相电压；
- Delta 接负载线电流/相电流；
- 不对称三相中性点位移；
- 三相功率测量。

### 第13章：含磁耦合的电路

重点表达：
- 同名端位置；
- 线圈电流参考方向；
- 互感电压参考极性；
- 变压器原边/副边端口；
- 阻抗折算侧；
- 反映阻抗的副边总阻抗。

优先题型：
- 同名端判断互感电压；
- 耦合电感网孔方程；
- 反映阻抗；
- 理想变压器阻抗变换。

## 4. 质量检查清单

生成或审核题目时，应逐项检查：

- 是否可以不看答案也还原电路结构；
- 是否明确所有参考方向；
- 是否明确单位和有效值/幅值约定；
- 是否至少需要两步以上分析；
- 是否有一个以上可能的真实学生错误；
- 错因标签是否能由题干直接触发；
- 反馈语是否能指出具体错误位置；
- 题型编号是否与 `questions/question_type_index.md` 一致；
- 错因标签是否存在于 `wrong_cases/wrong_reason_tags.md`；
- 标准答案是否包含关键方程而不只是最终数值。
