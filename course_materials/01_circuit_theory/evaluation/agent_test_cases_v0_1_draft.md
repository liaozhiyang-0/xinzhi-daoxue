### TCT-001 时间常数的物理意义

- 测试类型：概念问答
- 所属章节：第8章 一阶电路的暂态分析
- 关联题型：CT08-T01 一阶电路识别
- 关联错因标签：time_constant_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 用户输入

"时间常数 τ 是什么意思？τ 越大表示响应越快还是越慢？"

#### 期望 Agent 行为

Agent 应该：
1. 先解释时间常数的物理定义（反映一阶响应指数变化快慢的时域参数）；
2. 说明 τ 越大响应变化越慢，τ 越小变化越快；
3. 结合 RC 和 RL 电路给出具体例子；
4. 说明经过 5τ 后可近似达到稳态。

#### 参考回答要点

- τ = RC 或 τ = L/R
- τ 是响应从初始值变化到终值的 (1-1/e) ≈ 63.2% 所需的时间
- τ 越大，指数项 e^{-t/τ} 衰减越慢，暂态越长
- 约 5τ 后暂态分量衰减到 1% 以下，可认为进入稳态

#### 评价标准

- [ ] 是否准确解释 τ 的物理意义
- [ ] 是否区分 RC 和 RL 电路的时间常数公式
- [ ] 是否能解释 τ 与响应快慢的关系

### TCT-002 阻尼类型和特征根的关系

- 测试类型：概念问答
- 所属章节：第9章 二阶电路的暂态分析
- 关联题型：CT09-T07 特征根与固有频率计算
- 关联错因标签：damping_case_judgment_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 用户输入

"二阶 RLC 电路中什么是过阻尼、临界阻尼和欠阻尼？怎么判断？"

#### 期望 Agent 行为

Agent 应该：
1. 定义三种阻尼类型；
2. 通过比较阻尼系数 α 和固有频率 ω₀ 说明判断方法；
3. 说明三种情况对应的特征根形式；
4. 说明每种情况下响应的基本特征。

#### 参考回答要点

- α > ω₀ → 过阻尼：两个不相等负实根，非振荡响应
- α = ω₀ → 临界阻尼：重根，响应最快进入稳态但不振荡
- α < ω₀ → 欠阻尼：共轭复根，衰减振荡响应
- α 反映电阻引起的能量损耗，ω₀ 反映 LC 固有振荡频率

#### 评价标准

- [ ] 是否清楚说明三种阻尼的判断条件
- [ ] 是否能用特征根形式解释响应特征
- [ ] 是否给出 RLC 串联和并联的 α 公式

### TCT-003 相量法适用条件

- 测试类型：概念问答
- 所属章节：第10章 正弦稳态分析
- 关联题型：CT10-T04 正弦量与相量互换
- 关联错因标签：phasor_conversion_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 用户输入

"相量法在什么条件下能用？为什么不同频率不能用同一个相量？"

#### 期望 Agent 行为

Agent 应该：
1. 说明相量法适用于线性、时不变、同频正弦稳态电路；
2. 解释相量是复数表示，隐含了频率信息，不同频率对应不同的相量模型；
3. 说明不同频率的正弦量叠加必须在时域进行，不能在相量域直接相加；
4. 举反例说明错误使用相量法的后果。

#### 参考回答要点

- 适用条件：线性、时不变、同频正弦稳态
- 相量不包含频率信息，不同频率的相量直接相加会丢失频率差异
- 不同频率源叠加：分别用相量法求解，再转换回时域叠加
- 暂态分析中不能单独用相量法

#### 评价标准

- [ ] 是否完整列出相量法的适用条件
- [ ] 是否能解释不同频率不能直接相量相加的原因
- [ ] 是否给出正确处理不同频率叠加的方法

### TCT-004 三要素法适用范围

- 测试类型：公式条件

- 关联题型：CT08-T06 三要素法
- 关联错因标签：three_element_method_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 用户输入

"三要素法是不是所有换路电路都能用？什么时候不能用？"

#### 期望 Agent 行为

Agent 应该：
1. 说明三要素法适用于直流激励下的一阶线性电路；
2. 说明换路后的电路必须是一阶 RC 或 RL 电路；
3. 说明正弦激励下不能用三要素法（需要另外处理）；
4. 说明非线性电路不能直接套用三要素法；
5. 说明二阶电路不能用三要素法。

#### 参考回答要点

- 适用条件：一阶线性电路 + 直流/阶跃激励
- 不适用场景：正弦激励、二阶或高阶电路、非线性电路
- 三要素法本质是利用一阶响应的指数形式，仅适用于一阶线性系统的响应

### TCT-005 叠加定理在正弦稳态中的使用

- 测试类型：公式条件
- 所属章节：第10章 正弦稳态分析
- 关联题型：CT10-T15 正弦稳态定理应用
- 关联错因标签：ac_theorem_application_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 用户输入

"正弦稳态电路中叠加定理怎么用？不同频率的电源能直接相加吗？"

#### 期望 Agent 行为

Agent 应该：
1. 说明叠加定理在线性电路中成立；
2. 强调同频正弦分量可在相量域相加；
3. 强调不同频率正弦分量必须先转换回时域再叠加；
4. 举例说明正确和错误做法。

#### 参考回答要点

- 每个独立源单独作用时，其他源置零（电压源短路、电流源开路）
- 同频率的响应：在相量域计算结果，可直接相量相加
- 不同频率的响应：每个频率分别求相量响应→转时域→时域相加
- 错误做法：直接将不同频率的相量复数相加

#### 评价标准

- [ ] 是否能区分同频和异频的处理方法
- [ ] 是否说明叠加后转时域的必要性
- [ ] 是否能给出具体算例

### TCT-006 诊断时间常数计算错误

- 测试类型：错题诊断
- 所属章节：第8章 一阶电路的暂态分析
- 关联题型：CT08-T02 RC 零输入响应
- 关联错因标签：time_constant_error, source_zeroing_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 用户输入

"我做了一道 RC 放电题，电容 20μF，放电回路有两个电阻 R1=1kΩ、R2=1kΩ 串联，我算的时间常数是 0.02s，老师说我错了。"

#### 期望 Agent 行为

Agent 应该：
1. 引导学生检查从电容端口看入的等效电阻；
2. 让学生说明自己用了哪个电阻计算；
3. 指出应使用 R1+R2=2kΩ，而不是仅用其中一个；
4. 解释时间常数中 R 的定义。

#### 参考回答要点

- 先判断等效电阻是否从储能元件端口看入
- Req = R1+R2 = 2kΩ，τ = Req·C = 2×10³×20×10⁻⁶ = 0.04s
- 不是 τ=0.02s，可能只用了 R1 或 R2 中的一个

#### 评价标准

- [ ] 是否引导学生自查等效电阻来源
- [ ] 是否准确诊断错误原因
- [ ] 是否给出正确计算过程

### TCT-007 诊断复功率共轭错误

- 测试类型：错题诊断
- 所属章节：第11章 正弦稳态电路的功率
- 关联题型：CT11-T04 复功率计算
- 关联错因标签：complex_power_conjugate_error
- 推荐复习文件：chapters/11_sinusoidal_steady_state_power.md

#### 用户输入

"负载电压 220∠0°V，电流 5∠30°A，我算的复功率是 1100∠30°VA，但答案说是 1100∠-30°VA，为什么？"

#### 期望 Agent 行为

Agent 应该：
1. 指出复功率定义中电流需要取共轭；
2. 让学生用 Ů·Î^* 重新计算；
3. 说明不取共轭会导致无功功率符号错误；
4. 解释取共轭的原因（有功功率应与电压电流相位差余弦有关）。

#### 参考回答要点

- S̅ = Ů · Î^*，不是 Ů · İ
- Î^* = 5∠-30°A，S̅ = 220∠0° × 5∠-30° = 1100∠-30°VA
- 不取共轭会导致无功功率符号相反
- 复功率取共轭是为了保证 P = Re(S̅) = UIcosφ

#### 评价标准

- [ ] 是否准确定位到复功率共轭问题
- [ ] 是否解释清楚共轭运算的含义
- [ ] 是否给出正确的复功率计算过程

### TCT-008 诊断线电压相电压混淆

- 测试类型：错题诊断
- 所属章节：第12章 三相正弦稳态电路
- 关联题型：CT12-T03 线电压与相电压
- 关联错因标签：line_voltage_phase_voltage_error
- 推荐复习文件：chapters/12_three_phase_sinusoidal_circuits.md

#### 用户输入

"对称 Y 形电源相电压 220V，我算的线电压也是 220V，为什么答案是 381V？"

#### 期望 Agent 行为

Agent 应该：
1. 指出 Y 形连接线电压和相电压的关系为 U线 = √3·U相；
2. 画相量图说明为什么是 √3 倍的关系；
3. 说明两相电压相量差为 √3 倍的几何推导；
4. 对比 Δ 形连接的线相电压关系。

#### 参考回答要点

- U线 = UAB = UA - UB（相量差）
- 两个 220V 相差 120° 的相量相减，幅值为 √3 × 220 ≈ 381V
- Y 形：U线 = √3·U相，且超前对应相电压 30°
- Δ 形：U线 = U相

#### 评价标准

- [ ] 是否用相量图正确解释 √3 的来源
- [ ] 是否区分 Y 形和 Δ 形
- [ ] 是否提及相位关系（30°超前）

### TCT-009 一阶暂态完整分析追问

- 测试类型：综合追问
- 所属章节：第8章 一阶电路的暂态分析
- 关联题型：CT08-T06 三要素法
- 关联错因标签：initial_value_error, final_value_error, time_constant_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 用户输入

（模拟多轮对话）

用户第1问："RC 充电电路，电源 10V，R=2kΩ，C=50μF，开关 t=0 闭合，电容初始电压为 0。求 t>0 的 uC(t)。"

用户第2问（假设 Agent 给出答案后）："那如果 t=2τ 时 uC 是多少？"

用户第3问（假设 Agent 给出答案后）："那如果我换一个 R 为 1kΩ，充电速度会变快还是变慢？"

#### 期望 Agent 行为

Agent 应该：
1. 第1问：正确使用三要素法，给出 uC(t)=10(1-e^{-t/0.1})V；
2. 第2问：代入 t=2τ=0.2s，uC=10(1-e^{-2})≈8.65V；
3. 第3问：R 减小使 τ 从 0.1s 变为 0.05s，充电速度变快；
4. 追问中保持一致的变量关系，不出现前后矛盾。

#### 参考回答要点

- 第一问：uC(0⁺)=0V, uC(∞)=10V, τ=2000×50×10⁻⁶=0.1s, uC(t)=10(1-e^{-10t})V
- 第二问：t=0.2s, 代入得 uC=10(1-e^{-2})≈8.65V
- 第三问：R 减半 → τ 减半 → 充电更快（指数项衰减更快）

#### 评价标准

- [ ] 第1问是否完整使用三要素法
- [ ] 第2问是否准确计算指定时刻的响应值
- [ ] 第3问是否正确解释 R 变化对时间常数的影响

### TCT-010 三相电路综合分析追问

- 测试类型：综合追问
- 所属章节：第12章 三相正弦稳态电路
- 关联题型：CT12-T07 对称三相电路功率计算
- 关联错因标签：three_phase_power_formula_error, phase_line_quantity_confusion
- 推荐复习文件：chapters/12_three_phase_sinusoidal_circuits.md

#### 用户输入

（模拟多轮对话）

用户第1问："对称三相 Y 形负载，线电压 380V，每相阻抗 Z=20∠60°Ω，求三相总有功功率。"

用户第2问（假设 Agent 给出答案后）："那如果把负载换成 Δ 形连接，同样的阻抗，有功功率又是多少？"

用户第3问（假设 Agent 给出答案后）："为什么 Δ 形的功率是 Y 形的 3 倍？"

#### 期望 Agent 行为

Agent 应该：
1. 第1问：用 P = √3·U线·I线·cosφ 或 P = 3·U相·I相·cosφ 正确计算；
2. 第2问：正确计算 Δ 形负载下的功率（约为 Y 形的 3 倍）；
3. 第3问：解释因为 Δ 形每相电压是 Y 形的 √3 倍，电流也是 √3 倍，每相功率为 3 倍。

#### 参考回答要点

- 第1问：U相 = 380/√3 ≈ 220V，I相 = 220/20 = 11A，P = 3×220×11×cos60° = 3630W
- 第2问：Δ 形：U相 = 380V，I相 = 380/20 = 19A，I线 = √3×19 ≈ 32.91A，P = √3×380×32.91×0.5 ≈ 10830W
- 第3问：Δ 连接每相电压是 Y 连接的 √3 倍，功率与电压平方成正比，因此每相功率为 3 倍

#### 评价标准

- [ ] 第1问 Y 形功率计算是否正确
- [ ] 第2问 Δ 形功率计算是否正确
- [ ] 第3问是否合理解释功率差异的来源

### TCT-011 理想变压器阻抗变换追问

- 测试类型：综合追问
- 所属章节：第13章 含磁耦合的电路
- 关联题型：CT13-T13 理想变压器阻抗变换
- 关联错因标签：ideal_transformer_impedance_transform_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 用户输入

（模拟多轮对话）

用户第1问："理想变压器匝比 n=5，副边接 50Ω 负载，从原边看入的等效电阻是多少？"

用户第2问（假设 Agent 给出答案后）："为什么是 1250Ω 而不是 10Ω？阻抗变换公式的 n 到底在哪一侧？"

#### 期望 Agent 行为

Agent 应该：
1. 第1问：正确使用 Z₁ = n²·Z₂ = 25×50 = 1250Ω；
2. 第2问：解释 n 是原边匝数与副边匝数之比，阻抗按匝比的平方变换；
3. 说明阻抗从副边折算到原边要乘以 n²，反之除以 n²。

#### 参考回答要点

- Z₁ = n²·Z₂ = 5² × 50 = 1250Ω
- n = N₁/N₂，原边电压 = n·副边电压，原边电流 = 副边电流/n
- 阻抗定义 Z = V/I，因此 Z₁ = (n·V₂)/(I₂/n) = n²·(V₂/I₂) = n²·Z₂

#### 评价标准

- [ ] 是否给出正确的阻抗变换结果
- [ ] 是否用电压电流关系推导阻抗变换
- [ ] 是否解释 n 的位置含义

### TCT-012 功率因数校正综合追问

- 测试类型：综合追问
- 所属章节：第11章 正弦稳态电路的功率
- 关联题型：CT11-T06 功率因数校正
- 关联错因标签：power_factor_correction_error, capacitor_compensation_error
- 推荐复习文件：chapters/11_sinusoidal_steady_state_power.md

#### 用户输入

（模拟多轮对话）

用户第1问："某感性负载 P=5kW，cosφ₁=0.5（滞后），U=220V/50Hz。求补偿到 cosφ₂=0.95 需要的并联电容值。"

用户第2问（假设 Agent 给出答案后）："补偿后负载本身的功率改变了吗？"

用户第3问（假设 Agent 给出答案后）："那电源提供的电流为什么变小了？"

#### 期望 Agent 行为

Agent 应该：
1. 第1问：正确使用 C = P/(ωU²)(tanφ₁ - tanφ₂) 计算；
2. 第2问：说明并联电容只补偿无功，不改变负载本身的有功功率和无功功率；
3. 第3问：解释电源总电流减小是因为电源只需提供负载有功所需电流和部分无功，另一部分无功由电容就地补偿。

#### 参考回答要点

- 第1问：φ₁=60°→tanφ₁=1.732, φ₂=18.19°→tanφ₂=0.329, C = 5000/(314×48400)×(1.732-0.329) ≈ 462μF
- 第2问：负载自身功率不变（负载阻抗和电压不变），变化的是电源提供的总电流
- 第3问：电容可提供容性无功，与感性负载的无功相互抵消一部分，电源只需提供较小的总电流

#### 评价标准

- [ ] 电容补偿量计算是否正确
- [ ] 是否清楚解释补偿不影响负载本身功率
- [ ] 是否正确解释电源电流减小的原因

"同名端是什么？怎么判断两个耦合线圈的互感电压极性？"
