# 章节知识内容目录

本目录存放《电路理论》课程的章节级知识库 Markdown 文件。

当前已完成 13 个章节，覆盖电路模型、电阻电路、电路定理、运算放大器、非线性电阻、动态电路、正弦稳态、三相电路和磁耦合电路。

每个章节文件统一包含：

- 本章定位；
- 知识结构；
- 核心概念；
- 重要公式与适用条件；
- 典型题型；
- 常见学习困难；
- 错题诊断标签；
- Agent 问答样例；
- 学习建议；
- 与其他章节的关系。

章节文件不包含教材原文长段落，也不包含教材例题全文和完整解析，主要用于课程 Agent 检索、答疑和错题诊断。
