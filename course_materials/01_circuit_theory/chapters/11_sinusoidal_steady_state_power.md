## 1. 本章定位

本章是在第10章“正弦稳态分析”的基础上，进一步讨论正弦稳态电路中的功率问题。第10章主要解决“如何用相量法求电压和电流”，本章则回答：

1. 正弦稳态下瞬时功率为什么会随时间振荡；
2. 有功功率、无功功率、视在功率和复功率分别表示什么；
3. 功率因数为什么重要，如何通过并联电容等方式进行校正；
4. 在交流电路中如何实现最大有功功率传输；
5. 实际工程中如何测量有功功率、电能和交流参数。

本章是从“电压、电流分析”过渡到“能量与功率分析”的关键章节。它直接支撑后续交流电路设计、电力系统基础、模拟电路电源分析、电机与电力电子、通信射频电路中的阻抗匹配等内容。

对于课程 Agent 来说，本章适合用于诊断学生在以下方面的错误：

* 把瞬时功率、有功功率、无功功率混为一谈；
* 把视在功率当成实际消耗功率；
* 复功率中电流相量是否取共轭处理错误；
* 感性负载、容性负载无功功率符号判断错误；
* 功率因数校正时电容补偿量求错；
* 最大功率传输条件中“共轭匹配”和“电阻匹配”混淆；
* 功率表、电能表、V-A-W 法测量时仪表连接和读数意义不清。

## 3. 核心概念

### 3.1 瞬时功率

* 概念解释：瞬时功率是某一时刻二端网络吸收或发出的功率，定义为端电压与端电流的瞬时乘积。
* 使用场景：分析正弦电路中能量在电源、负载和储能元件之间的即时交换。
* 与相近概念的区别：
  * 瞬时功率 \(p(t)\) 随时间变化；
  * 有功功率 \(P\) 是瞬时功率在一个周期内的平均值；
  * 无功功率 \(Q\) 描述能量往返交换的幅度，不代表实际消耗的平均能量。
* 常见误区：
  * 认为正弦稳态下功率一定也是正弦量；
  * 把瞬时功率的二倍频振荡项当作有功功率；
  * 忽略电压、电流相位差对功率的影响。

### 3.2 有功功率

* 概念解释：有功功率是瞬时功率在一个周期内的平均值，反映电路中真正被消耗、转换或输出为有效功的功率。
* 使用场景：电阻发热、机械做功、照明、电机输出、电能计量。
* 与相近概念的区别：
  * 有功功率单位是 W；
  * 视在功率单位是 V·A；
  * 无功功率单位是 var。
* 常见误区：
  * 用 \(UI\) 直接当有功功率；
  * 忽略功率因数；
  * 把电感、电容的无功交换误认为有功消耗。

### 3.3 无功功率

* 概念解释：无功功率反映电源与储能元件之间能量往返交换的规模。电感吸收正无功，电容吸收负无功。
* 使用场景：感性负载、容性负载、功率因数校正、电力系统无功补偿。
* 与相近概念的区别：
  * 有功功率对应平均能量消耗；
  * 无功功率对应电场或磁场能量周期性交换；
  * 无功功率不等于“没有用”，它会增加线路电流和设备容量需求。
* 常见误区：
  * 认为无功功率没有工程影响；
  * 忽略感性无功和容性无功的符号；
  * 把 \(Q\) 和 \(S\) 混用。

### 3.4 视在功率

* 概念解释：视在功率是端电压有效值和端电流有效值的乘积，表示电源或设备需要承担的总容量。
* 使用场景：变压器、发电机、电源、UPS、逆变器容量标注。
* 与相近概念的区别：
  * \(S=UI\) 表示容量大小；
  * \(P=UI\cos\varphi\) 表示实际有功；
  * \(Q=UI\sin\varphi\) 表示无功交换。
* 常见误区：
  * 把 kV·A 当 kW；
  * 只看负载有功功率而忽略设备容量；
  * 不理解功率因数低会使电流增大。

### 3.5 功率因数

* 概念解释：功率因数表示有功功率占视在功率的比例。对单一正弦稳态线性电路，功率因数等于电压、电流相位差余弦。
* 使用场景：判断电源利用率、计算线路电流、进行无功补偿。
* 与相近概念的区别：
  * 功率因数是无量纲数；
  * \(\cos\varphi\) 是正弦线性电路中常用的功率因数；
  * 非正弦情况下还会涉及畸变功率因数，本章主要讨论正弦稳态。
* 常见误区：
  * 只写数值不判断超前或滞后；
  * 把功率因数提高理解为增加有功功率；
  * 认为功率因数越高越一定没有无功问题。

### 3.6 复功率

* 概念解释：复功率把有功功率和无功功率统一表示为一个复数，便于用相量法进行功率计算。
* 使用场景：多个交流负载功率相加、负载综合分析、工程电力计算。
* 与相近概念的区别：
  * 复功率 \(\underline{S}=P+jQ\) 是复数；
  * 视在功率 \(S=|\underline{S}|\) 是模值；
  * 有功功率 \(P\) 是实部；
  * 无功功率 \(Q\) 是虚部。
* 常见误区：
  * 计算复功率时忘记对电流相量取共轭；
  * 直接把电压相量和电流相量相乘得到错误相角；
  * 把复功率的实部、虚部、模值混淆。

### 3.7 功率因数校正

* 概念解释：功率因数校正常通过并联电容为感性负载提供部分无功功率，从而减少电源需要提供的无功功率和线路电流。
* 使用场景：电动机、变压器、电力负载、工厂配电系统。
* 与相近概念的区别：
  * 并联电容主要补偿无功，不改变负载消耗的有功功率；
  * 功率因数提高后电源电流减小；
  * 补偿过度可能使系统变为容性。
* 常见误区：
  * 认为并联电容会减少负载本身消耗的有功功率；
  * 把补偿前后的有功功率改掉；
  * 电容无功符号写反；
  * 不检查目标功率因数是滞后还是超前。

### 3.8 最大有功功率传输

* 概念解释：当负载阻抗与电源等效阻抗满足一定匹配条件时，负载吸收的有功功率最大。
* 使用场景：信号传输、通信系统、射频电路、测量系统、某些小功率交流网络。
* 与直流最大功率传输的区别：
  * 直流电阻网络常见条件为 \(R_L=R_{\mathrm{th}}\)；
  * 正弦稳态交流电路若负载阻抗的电阻和电抗都可调，则通常要求共轭匹配 \(Z_L=Z_{\mathrm{th}}^*\)；
  * 若负载电抗不可调，只能对可调参数求最优。
* 常见误区：
  * 把所有交流最大功率题都写成 \(R_L=R_{\mathrm{th}}\)；
  * 忘记共轭匹配中的电抗符号相反；
  * 最大功率传输与最高效率混淆。

### 3.9 有功功率测量

* 概念解释：功率表通过电流线圈和电压线圈共同作用，测量交流电路的平均有功功率。
* 使用场景：实验室交流功率测量、电力系统电能计量、负载测试。
* 与相近概念的区别：
  * 电压表测有效值电压；
  * 电流表测有效值电流；
  * 功率表测有功功率；
  * 电能表测一段时间内的能量。
* 常见误区：
  * 把电流线圈和电压线圈接错；
  * 用电压表、电流表读数乘积代替有功功率；
  * 忽略功率表接法对误差的影响。

## 4. 重要公式与适用条件

### 4.1 瞬时功率定义

公式：

```math
p(t)=u(t)i(t)
```

符号含义：

* \(p(t)\)：瞬时功率；
* \(u(t)\)：端电压瞬时值；
* \(i(t)\)：端电流瞬时值。

适用条件：

* 电压、电流采用关联参考方向时，\(p(t)>0\) 表示网络吸收功率；
* 若参考方向非关联，需要重新判断正负含义。

常见误用：

* 不判断参考方向直接解释吸收或发出；
* 把瞬时功率的某一时刻值当作平均功率。

### 4.2 正弦电压电流下的瞬时功率

若：

```math
u(t)=\sqrt{2}U\cos(\omega t+\varphi_u)
```

```math
i(t)=\sqrt{2}I\cos(\omega t+\varphi_i)
```

则：

```math
p(t)=UI\cos(\varphi_u-\varphi_i)+UI\cos(2\omega t+\varphi_u+\varphi_i)
```

符号含义：

* \(U\)、\(I\)：电压、电流有效值；
* \(\varphi_u\)、\(\varphi_i\)：电压、电流初相；
* \(\varphi=\varphi_u-\varphi_i\)：电压超前电流的相位差。

适用条件：

* 电压、电流均为同频正弦量；
* 使用有效值表示幅值；
* 线性正弦稳态电路。

常见误用：

* 忘记正弦量有效值与最大值之间的 \(\sqrt{2}\) 关系；
* 将二倍频项误认为平均功率；
* 相位差方向写反。

### 4.3 有功功率

公式：

```math
P=\frac{1}{T}\int_{0}^{T}p(t)\,dt
```

对正弦稳态二端网络：

```math
P=UI\cos\varphi
```

符号含义：

* \(P\)：有功功率；
* \(T\)：周期；
* \(\varphi=\varphi_u-\varphi_i\)：端口电压与电流相位差。

适用条件：

* 正弦稳态；
* \(U\)、\(I\) 为有效值；
* 电压电流参考方向按吸收功率约定。

常见误用：

* 使用最大值代入 \(P=UI\cos\varphi\)；
* 用 \(UI\) 代替 \(UI\cos\varphi\)；
* 相位差使用电流超前电压角而导致符号错误。

### 4.4 线性阻抗吸收的有功功率

若：

```math
Z=R+jX
```

则：

```math
P=I^2R
```

也可写为：

```math
P=U^2G
```

其中：

```math
Y=G+jB=\frac{1}{Z}
```

符号含义：

* \(R\)：阻抗实部；
* \(X\)：阻抗虚部；
* \(G\)：导纳实部；
* \(B\)：导纳虚部；
* \(U\)、\(I\)：有效值。

适用条件：

* 单一正弦稳态；
* \(I\) 为流过阻抗的电流有效值；
* \(U\) 为阻抗两端电压有效值。

常见误用：

* 用总电压套入局部电阻公式；
* 用阻抗模值代替电阻实部；
* 把电抗当成有功消耗项。

### 4.5 无功功率

公式：

```math
Q=UI\sin\varphi
```

对阻抗形式：

```math
Q=I^2X
```

对导纳形式：

```math
Q=-U^2B
```

符号含义：

* \(Q\)：无功功率；
* \(X\)：电抗；
* \(B\)：电纳。

适用条件：

* 正弦稳态；
* 关联参考方向；
* \(Q>0\) 通常表示感性负载吸收无功，\(Q<0\) 通常表示容性负载吸收负无功。

常见误用：

* 忽略电感、电容无功符号相反；
* 把 \(|Q|\) 当作有符号的 \(Q\)；
* 将 \(Q\) 和 \(S\) 混用。

### 4.6 电阻、电感、电容的功率特性

电阻：

```math
P_R=I^2R=\frac{U_R^2}{R}
```

```math
Q_R=0
```

电感：

```math
P_L=0
```

```math
Q_L=I^2X_L=\frac{U_L^2}{X_L}>0
```

电容：

```math
P_C=0
```

```math
Q_C=I^2X_C=-\frac{U_C^2}{|X_C|}<0
```

符号含义：

* \(X_L=\omega L\)：感抗；
* \(X_C=-1/(\omega C)\)：容抗。

适用条件：

* 理想电阻、电感、电容；
* 正弦稳态。

常见误用：

* 认为理想电感或电容消耗有功功率；
* 容性无功符号写成正；
* 忽略实际电感、电容可能有损耗。

### 4.7 视在功率

公式：

```math
S=UI
```

也有：

```math
S=\sqrt{P^2+Q^2}
```

符号含义：

* \(S\)：视在功率，单位 V·A；
* \(P\)：有功功率，单位 W；
* \(Q\)：无功功率，单位 var。

适用条件：

* 单一正弦稳态；
* \(U\)、\(I\) 为有效值；
* 功率三角形适用于线性正弦稳态负载。

常见误用：

* 把 \(S\) 当作有功功率；
* 单位写成 W；
* 只用 \(P\) 选择设备容量，忽略视在功率。

### 4.8 功率因数

公式：

```math
\lambda=\frac{P}{S}
```

单一正弦稳态线性负载中：

```math
\lambda=\cos\varphi
```

符号含义：

* \(\lambda\)：功率因数；
* \(\varphi\)：电压超前电流的相位差。

适用条件：

* 对正弦线性电路，功率因数等于相位差余弦；
* 对非正弦情况需考虑畸变影响，本章主要不展开。

常见误用：

* 只写 \(\lambda\) 的数值，不说明滞后或超前；
* 把提高功率因数理解为改变负载有功功率；
* 误认为功率因数为 1 时电路一定无损耗。

### 4.9 复功率

公式：

```math
\underline{S}=\underline{U}\,\underline{I}^{*}
```

其中：

```math
\underline{S}=P+jQ
```

```math
|\underline{S}|=S
```

符号含义：

* \(\underline{U}\)：电压相量；
* \(\underline{I}^{*}\)：电流相量的共轭；
* \(P\)：有功功率；
* \(Q\)：无功功率。

适用条件：

* 相量均采用有效值相量；
* 电压、电流参考方向为关联参考方向；
* 适用于正弦稳态功率计算。

常见误用：

* 写成 \(\underline{U}\underline{I}\) 而漏掉共轭；
* 相量用最大值而不是有效值；
* 把复功率的模和实部混淆。

### 4.10 复功率守恒

公式：

```math
\sum_k \underline{S}_k=0
```

若将负载吸收功率和电源提供功率分开，也可写成：

```math
\sum \underline{S}_{\text{source}}=\sum \underline{S}_{\text{load}}
```

符号含义：

* \(\underline{S}_k\)：第 \(k\) 个元件或支路的复功率。

适用条件：

* 同一电路中所有支路采用统一参考方向；
* 正弦稳态相量计算；
* 功率符号体系一致。

常见误用：

* 只让有功功率守恒，不检查无功功率；
* 电源功率正负号与负载功率符号混用；
* 多负载复功率相加时直接加视在功率模值。

### 4.11 功率因数校正中的补偿无功

若负载有功功率 \(P\) 不变，校正前功率因数角为 \(\varphi_1\)，校正后目标功率因数角为 \(\varphi_2\)，则需要电容提供的无功补偿量为：

```math
Q_C=P(\tan\varphi_2-\tan\varphi_1)
```

常按所需电容无功大小写为：

```math
|Q_C|=P(\tan\varphi_1-\tan\varphi_2)
```

并联电容无功：

```math
|Q_C|=\omega C U^2
```

所以：

```math
C=\frac{P(\tan\varphi_1-\tan\varphi_2)}{\omega U^2}
```

符号含义：

* \(P\)：负载有功功率；
* \(\varphi_1\)：补偿前功率因数角；
* \(\varphi_2\)：补偿后功率因数角；
* \(U\)：并联电容两端电压有效值。

适用条件：

* 感性负载功率因数校正；
* 并联电容与负载端电压相同；
* 有功功率近似不变；
* 目标功率因数仍为滞后时，补偿不过量。

常见误用：

* \(\tan\varphi_1\) 和 \(\tan\varphi_2\) 顺序写反；
* 把电容串联到负载中；
* 用线电压、相电压混淆；
* 补偿后把负载有功功率也改变。

### 4.12 负载阻抗可调时的最大有功功率传输

若电源等效为戴维南形式：

```math
Z_{\mathrm{th}}=R_{\mathrm{th}}+jX_{\mathrm{th}}
```

负载阻抗为：

```math
Z_L=R_L+jX_L
```

当 \(R_L\) 与 \(X_L\) 均可调时，最大有功功率传输条件为：

```math
Z_L=Z_{\mathrm{th}}^{*}=R_{\mathrm{th}}-jX_{\mathrm{th}}
```

最大功率为：

```math
P_{\max}=\frac{U_{\mathrm{oc}}^2}{4R_{\mathrm{th}}}
```

符号含义：

* \(Z_{\mathrm{th}}\)：电源内网络从负载端口看入的等效阻抗；
* \(Z_L\)：负载阻抗；
* \(U_{\mathrm{oc}}\)：端口开路电压有效值。

适用条件：

* 正弦稳态；
* 负载阻抗实部和虚部均可调；
* \(R_{\mathrm{th}}>0\)；
* \(U_{\mathrm{oc}}\) 为有效值。

常见误用：

* 写成 \(Z_L=Z_{\mathrm{th}}\) 而不是共轭；
* 忘记电抗符号相反；
* 用最大值电压代入有效值公式。

### 4.13 负载电抗不可调时的最优负载电阻

若负载电抗 \(X_L\) 已固定，只能调节 \(R_L\)，则最优 \(R_L\) 由对负载有功功率求极值得到。常见形式为：

```math
R_L=\sqrt{R_{\mathrm{th}}^2+(X_{\mathrm{th}}+X_L)^2}
```

符号含义：

* \(R_L\)：可调负载电阻；
* \(X_L\)：固定负载电抗；
* \(R_{\mathrm{th}}+jX_{\mathrm{th}}\)：端口等效内阻抗。

适用条件：

* 只有 \(R_L\) 可调；
* \(X_L\) 已知且不可调；
* 正弦稳态线性端口。

常见误用：

* 仍使用 \(R_L=R_{\mathrm{th}}\)；
* 忽略固定电抗对最佳电阻的影响；
* 把 \(X_{\mathrm{th}}-X_L\) 与 \(X_{\mathrm{th}}+X_L\) 混淆。

### 4.14 有功功率表读数

功率表读数本质上为一个周期内瞬时功率平均值：

```math
P=\frac{1}{T}\int_0^T u(t)i(t)\,dt
```

对正弦稳态：

```math
P=UI\cos\varphi
```

符号含义：

* \(u(t)\)：电压线圈两端电压；
* \(i(t)\)：电流线圈电流；
* \(\varphi\)：这两个量之间的相位差。

适用条件：

* 功率表电流线圈串联在被测支路；
* 电压线圈并联在被测端口；
* 接线极性正确。

常见误用：

* 电流线圈和电压线圈接错；
* 忽略表内线圈功耗；
* 只用电压表和电流表乘积代替功率表读数。

## 5. 典型题型

### 题型 1：由正弦电压、电流求瞬时功率

考查知识点：

* 正弦量三要素；
* 瞬时功率；
* 积化和差公式；
* 二倍频功率项。

解题关键：

1. 将电压、电流统一为余弦形式；
2. 判断使用的是最大值还是有效值；
3. 写出 \(p(t)=u(t)i(t)\)；
4. 分解为平均项和二倍频振荡项；
5. 根据符号判断网络吸收或发出功率。

常见错误：

* 相位差写反；
* 有效值与最大值混用；
* 误把瞬时功率平均项前的系数写错；
* 忽略二倍频项。

对应错因标签：

* instantaneous_power_expression_error
* rms_peak_confusion
* phase_lead_lag_error
* double_frequency_term_missing

### 题型 2：判断 R、L、C 元件的瞬时功率特征

考查知识点：

* 电阻耗能；
* 电感储放磁场能量；
* 电容储放电场能量；
* 瞬时功率正负变化。

解题关键：

1. 写出元件电压电流相位关系；
2. 分析 \(p(t)\) 是否有平均值；
3. 判断是否存在能量往返交换；
4. 区分有功消耗和无功交换。

常见错误：

* 认为电感、电容没有功率；
* 把电感、电容的瞬时功率平均值写成非零；
* 对负瞬时功率物理意义解释错误。

对应错因标签：

* reactive_power_sign_error
* energy_exchange_confusion
* active_power_error

### 题型 3：计算有功功率和无功功率

考查知识点：

* 有功功率；
* 无功功率；
* 电压电流相位差；
* 阻抗实部与虚部。

解题关键：

1. 求端口电压、电流有效值；
2. 求相位差 \(\varphi\)；
3. 用 \(P=UI\cos\varphi\)；
4. 用 \(Q=UI\sin\varphi\)；
5. 判断感性或容性。

常见错误：

* 用 \(UI\) 直接作为 \(P\)；
* 将 \(Q\) 的符号取反；
* 误用阻抗模值代替 \(R\) 或 \(X\)。

对应错因标签：

* active_power_error
* reactive_power_sign_error
* impedance_model_error

### 题型 4：利用阻抗或导纳计算功率

考查知识点：

* 阻抗；
* 导纳；
* \(P=I^2R\)；
* \(Q=I^2X\)；
* \(P=U^2G\)；
* \(Q=-U^2B\)。

解题关键：

1. 明确已知的是支路电流还是支路电压；
2. 若用阻抗，采用 \(I^2R\)、\(I^2X\)；
3. 若用导纳，采用 \(U^2G\)、\(-U^2B\)；
4. 检查 \(R,X,G,B\) 对应的是同一端口。

常见错误：

* 用总电流计算局部支路功率；
* 用 \(U^2B\) 时忘记负号；
* 阻抗和导纳表达混用。

对应错因标签：

* reactive_power_sign_error
* branch_power_variable_mismatch
* impedance_admittance_confusion

### 题型 5：视在功率与功率因数计算

考查知识点：

* 视在功率；
* 功率三角形；
* 功率因数；
* 感性/容性负载。

解题关键：

1. 先求 \(P\)、\(Q\) 或 \(U\)、\(I\)；
2. 计算 \(S=UI\) 或 \(S=\sqrt{P^2+Q^2}\)；
3. 计算功率因数 \(\lambda=P/S\)；
4. 根据 \(Q\) 符号判断滞后或超前。

常见错误：

* 把视在功率单位写成 W；
* 只给功率因数数值，不说明感性或容性；
* 把 \(\lambda\) 和 \(\varphi\) 混淆。

对应错因标签：

* apparent_power_error
* power_factor_error
* power_triangle_error

### 题型 6：复功率计算

考查知识点：

* 复功率；
* 电流相量共轭；
* 有功与无功；
* 相量有效值。

解题关键：

1. 将电压、电流写成有效值相量；
2. 明确参考方向是否关联；
3. 使用 \(\underline{S}=\underline{U}\underline{I}^*\)；
4. 分离实部和虚部；
5. 由模值求视在功率。

常见错误：

* 忘记电流取共轭；
* 使用最大值相量；
* 相量方向不统一；
* 将复功率模值和实部混淆。

对应错因标签：

* complex_power_conjugate_error
* complex_power_conjugate_error
* complex_power_conjugate_error

### 题型 7：多负载复功率合成

考查知识点：

* 复功率守恒；
* 多支路功率相加；
* 电源电流计算；
* 负载功率因数。

解题关键：

1. 分别求各负载复功率；
2. 直接相加复功率，而不是相加视在功率模值；
3. 得到总 \(P\)、总 \(Q\)；
4. 求总 \(S\)、总功率因数和电源电流。

常见错误：

* 直接相加各负载 \(S\) 的数值；
* 忽略容性无功可抵消感性无功；
* 电源提供功率与负载吸收功率符号混乱。

对应错因标签：

* complex_power_addition_error
* complex_power_addition_error
* passive_sign_convention_error

### 题型 8：功率因数校正

考查知识点：

* 感性负载；
* 并联电容补偿；
* 无功功率；
* 功率因数角。

解题关键：

1. 根据补偿前功率因数求 \(\varphi_1\)；
2. 根据目标功率因数求 \(\varphi_2\)；
3. 用 \(P(\tan\varphi_1-\tan\varphi_2)\) 求补偿无功大小；
4. 用 \(|Q_C|=\omega C U^2\) 求电容；
5. 检查补偿后是否过补偿。

常见错误：

* 把 \(\tan\varphi_1\) 与 \(\tan\varphi_2\) 写反；
* 把电容无功写成正；
* 误认为校正改变负载有功功率；
* 电压有效值选错。

对应错因标签：

* power_factor_correction_error
* capacitor_compensation_error
* power_factor_correction_error
* capacitor_compensation_error

### 题型 9：最大有功功率传输

考查知识点：

* 戴维南等效；
* 共轭匹配；
* 最大有功功率；
* 可调负载条件。

解题关键：

1. 从负载端口求 \(U_{\mathrm{oc}}\) 和 \(Z_{\mathrm{th}}\)；
2. 判断负载的哪些参数可调；
3. 若 \(R_L\)、\(X_L\) 均可调，用共轭匹配；
4. 若只有 \(R_L\) 可调，按极值条件求最优 \(R_L\)；
5. 计算最大有功功率。

常见错误：

* 没有移去负载就求等效；
* 把共轭匹配写成相等匹配；
* 最大功率公式使用峰值电压；
* 忽略受控源情况下的测试源法。

对应错因标签：

* max_ac_power_transfer_error
* max_ac_power_transfer_error
* thevenin_port_error
* rms_value_error

### 题型 10：功率表测量有功功率

考查知识点：

* 功率表接线；
* 电流线圈；
* 电压线圈；
* 平均功率。

解题关键：

1. 判断电流线圈是否串联在被测支路；
2. 判断电压线圈是否并联在被测端口；
3. 明确功率表读数为有功功率；
4. 根据接法考虑表内功耗误差。

常见错误：

* 把功率表读数当 \(UI\)；
* 电流线圈和电压线圈接错；
* 忽略低功率因数下功率表读数较小。

对应错因标签：

* wattmeter_connection_error
* wattmeter_connection_error
* wattmeter_connection_error

### 题型 11：电能计量与电费计算

考查知识点：

* 有功电能；
* 电能表；
* kW·h；
* 功率因数与计费。

解题关键：

1. 区分功率和电能；
2. 用有功功率乘以时间求电能；
3. 根据电价计算费用；
4. 若涉及功率因数奖惩，按题目给定规则处理。

常见错误：

* 把 kW 与 kW·h 混淆；
* 用视在功率计算有功电能；
* 忽略功率因数低带来的额外费用或容量限制。

对应错因标签：

* power_unit_confusion
* energy_meter_reading_error
* apparent_power_error

### 题型 12：交流参数测量

考查知识点：

* 电压表；
* 电流表；
* 功率表；
* 阻抗模值；
* 功率因数角。

解题关键：

1. 用 \(Z=U/I\) 求阻抗模值；
2. 用 \(P=UI\cos\varphi\) 求功率因数；
3. 判断 \(R=Z\cos\varphi\)；
4. 根据感性或容性判断 \(X\) 符号；
5. 写出负载阻抗。

常见错误：

* 只由 \(U/I\) 得到阻抗模值却当成电阻；
* 无法判断电抗符号；
* 忘记功率表读数是有功功率。

对应错因标签：

* ac_measurement_error
* reactance_sign_error
* ac_measurement_error

## 6. 常见学习困难

* 瞬时功率与平均功率混淆：学生容易把 \(p(t)\) 的表达式直接当成 \(P\)。
* 有效值和最大值混用：正弦功率公式通常使用有效值，若代入峰值会差一个系数。
* 相位差方向不统一：\(\varphi=\varphi_u-\varphi_i\) 与 \(\varphi_i-\varphi_u\) 混用会导致 \(Q\) 符号错误。
* \(P\)、\(Q\)、\(S\) 单位混乱：W、var、V·A 经常混写。
* 无功功率物理意义模糊：不理解无功是能量往返交换，而不是“完全没有意义的功率”。
* 电感、电容无功符号错误：电感吸收正无功，电容吸收负无功。
* 复功率中电流共轭遗漏：这是本章最常见的相量计算错误之一。
* 多负载功率合成错误：视在功率不能直接代数相加，复功率可以代数相加。
* 功率因数校正公式机械套用：不知道补偿的是无功，不改变原负载有功功率。
* 最大功率传输条件误用：交流情况下常需要共轭匹配，不是简单 \(Z_L=Z_{\mathrm{th}}\)。
* 功率测量仪表含义不清：电压表、电流表、功率表、电能表测量对象不同。
* 工程题单位错误：kW、kvar、kV·A、kW·h、F、μF 等单位换算容易出错。

## 7. 错题诊断标签
### instantaneous_power_expression_error

中文名称：瞬时功率表达式错误  
适用场景：由正弦电压、电流求 \(p(t)\)  
典型表现：漏掉平均项或二倍频项，积化和差公式错误  
诊断提示：先统一正弦函数形式，再使用三角恒等式展开

### rms_peak_confusion

中文名称：有效值与最大值混淆  
适用场景：所有正弦功率计算  
典型表现：把峰值代入 \(P=UI\cos\varphi\) 或 \(S=UI\)  
诊断提示：功率公式默认使用有效值；若给出峰值，先除以 \(\sqrt{2}\)

### phase_lead_lag_error

中文名称：相位差方向错误  
适用场景：有功、无功、功率因数、瞬时功率计算  
典型表现：把 \(\varphi_u-\varphi_i\) 写成 \(\varphi_i-\varphi_u\)  
诊断提示：明确题目规定的是电压超前电流还是电流超前电压

### double_frequency_term_missing

中文名称：二倍频功率项遗漏  
适用场景：瞬时功率表达式  
典型表现：只写 \(UI\cos\varphi\)，没有写随 \(2\omega\) 变化的部分  
诊断提示：正弦电压电流相乘会产生直流分量和二倍频分量

### active_power_error

中文名称：有功功率公式错误  
适用场景：平均功率计算  
典型表现：把 \(P=UI\cos\varphi\) 写成 \(UI\) 或 \(UI\sin\varphi\)  
诊断提示：有功功率对应与电压同相的电流分量

### reactive_power_sign_error

中文名称：无功功率符号错误  
适用场景：感性、容性负载判断  
典型表现：电感无功写负，电容无功写正  
诊断提示：按 \(Q=UI\sin(\varphi_u-\varphi_i)\) 判断符号

### impedance_model_error

中文名称：阻抗实部虚部混淆  
适用场景：用阻抗计算 \(P\)、\(Q\)  
典型表现：用 \(|Z|\) 代替 \(R\)，或用 \(R\) 计算无功  
诊断提示：有功对应阻抗实部，无功对应阻抗虚部

### apparent_power_error

中文名称：视在功率与有功功率混淆  
适用场景：设备容量、负载功率计算  
典型表现：把 kV·A 当 kW  
诊断提示：视在功率是容量，有功功率是实际平均功率

### power_factor_error

中文名称：功率因数超前滞后判断错误  
适用场景：功率因数计算和说明  
典型表现：只给数值，不判断感性滞后或容性超前  
诊断提示：根据 \(Q\) 符号或电流相位判断超前/滞后

### power_triangle_error

中文名称：功率三角形错误  
适用场景：由 \(P,Q,S\) 相互计算  
典型表现：三角形边对应关系写错，或 \(Q\) 方向画错  
诊断提示：横轴通常为 \(P\)，纵轴为 \(Q\)，斜边为 \(S\)

### complex_power_conjugate_error

中文名称：复功率共轭错误  
适用场景：复功率计算  
典型表现：写成 \(\underline{S}=\underline{U}\underline{I}\)  
诊断提示：复功率应使用电流相量共轭 \(\underline{I}^*\)

### complex_power_addition_error

中文名称：视在功率直接相加错误  
适用场景：多个负载功率合成  
典型表现：直接把各负载 \(S\) 的数值相加作为总视在功率  
诊断提示：先相加复功率，再由总 \(P,Q\) 求总 \(S\)

### passive_sign_convention_error

中文名称：电源负载功率符号混乱  
适用场景：复功率守恒  
典型表现：电源提供功率和负载吸收功率使用同一正号体系导致不守恒  
诊断提示：先规定所有元件按吸收功率为正，或明确区分提供与吸收

### power_factor_correction_error

中文名称：功率因数校正公式错误  
适用场景：并联电容补偿  
典型表现：补偿无功公式中 \(\tan\varphi_1\)、\(\tan\varphi_2\) 顺序错误  
诊断提示：补偿后无功应小于补偿前无功

### capacitor_compensation_error

中文名称：补偿电容无功符号错误  
适用场景：功率因数校正  
典型表现：把并联电容提供的无功写成正感性无功  
诊断提示：理想电容吸收负无功，等效为向感性负载提供容性无功

### max_ac_power_transfer_error

中文名称：交流最大功率条件错误  
适用场景：最大有功功率传输  
典型表现：所有交流题都用 \(R_L=R_{\mathrm{th}}\)  
诊断提示：负载复阻抗可调时应使用共轭匹配

### rms_value_error

中文名称：功率公式峰值误用  
适用场景：最大功率和复功率计算  
典型表现：把开路电压峰值代入 \(P_{\max}=U_{\mathrm{oc}}^2/(4R)\)  
诊断提示：该公式中的 \(U_{\mathrm{oc}}\) 应为有效值

### wattmeter_connection_error

中文名称：功率表接线错误  
适用场景：有功功率测量  
典型表现：电流线圈未串联，电压线圈未并联，极性端接反  
诊断提示：检查电流线圈、电压线圈和同名端连接

### power_unit_confusion

中文名称：功率与电能单位混淆  
适用场景：电能计量、电费计算  
典型表现：把 kW 和 kW·h 混用  
诊断提示：功率乘以时间才是电能

### energy_meter_reading_error

中文名称：电能表读数错误  
适用场景：电量计量  
典型表现：不理解电能表累计的是有功电能  
诊断提示：电能表读数通常表示累计有功电能

### ac_measurement_error

中文名称：阻抗模值与电阻混淆  
适用场景：V-A-W 法测交流参数  
典型表现：由 \(U/I\) 得到的阻抗模值直接当 \(R\)  
诊断提示：还需用有功功率求功率因数，从而分解 \(R\) 和 \(X\)

### reactance_sign_error

中文名称：电抗符号判断错误  
适用场景：交流参数测量  
典型表现：无法判断负载是感性还是容性  
诊断提示：结合相位、功率因数滞后/超前或电路结构判断 \(X\) 正负

## 8. Agent 问答样例

### Q1：正弦稳态下瞬时功率也是正弦量吗？

所属知识点：瞬时功率  
可能错因：double_frequency_term_missing  
回答：  
不一定。若电压和电流是同频正弦量，它们相乘后会得到一个恒定分量和一个二倍频分量。因此瞬时功率通常不是与电压同频的正弦量，而是围绕平均功率上下波动。

### Q2：为什么有功功率公式是 \(P=UI\cos\varphi\)，不是 \(P=UI\)？

所属知识点：有功功率  
可能错因：active_power_error  
回答：  
\(UI\) 表示视在功率，反映电压电流共同形成的容量。有功功率只对应电流中与电压同相的分量，所以要乘以 \(\cos\varphi\)。当电压电流同相时，\(\cos\varphi=1\)，这时 \(P=UI\)。

### Q3：无功功率是不是没有用？

所属知识点：无功功率  
可能错因：reactive_power_sign_error  
回答：  
不能简单说“没有用”。无功功率表示电源与电感、电容等储能元件之间往返交换的能量。它不消耗为平均有功功率，但会增加线路电流和设备容量需求，所以在工程中需要关注和补偿。

### Q4：电感和电容的无功功率符号怎么判断？

所属知识点：无功功率符号  
可能错因：reactive_power_sign_error  
回答：  
按常用吸收功率约定，电感吸收正无功，电容吸收负无功。也可以用 \(Q=UI\sin(\varphi_u-\varphi_i)\) 判断：感性负载电流滞后电压，\(Q>0\)；容性负载电流超前电压，\(Q<0\)。

### Q5：复功率为什么要用电流相量的共轭？

所属知识点：复功率  
可能错因：complex_power_conjugate_error  
回答：  
因为复功率要得到 \(P+jQ\)，其中相角应体现电压相位减去电流相位。若直接用 \(\underline{U}\underline{I}\)，相角会变成相位相加，不符合功率相位差关系。所以应使用 \(\underline{S}=\underline{U}\underline{I}^{*}\)。

### Q6：多个负载的视在功率可以直接相加吗？

所属知识点：复功率守恒  
可能错因：complex_power_addition_error  
回答：  
一般不能。因为不同负载的无功功率可能方向不同，视在功率只是复功率的模值，不能直接代数相加。正确做法是先分别求 \(\underline{S}_k=P_k+jQ_k\)，再把复功率相加，最后求总视在功率。

### Q7：功率因数低为什么不好？

所属知识点：功率因数  
可能错因：power_factor_error  
回答：  
在有功功率一定时，功率因数越低，所需视在功率和线路电流越大。这会增加线路损耗、变压器和电源容量占用，并可能引起电压降。因此工程中常通过无功补偿提高功率因数。

### Q8：并联电容校正功率因数会减少负载有功功率吗？

所属知识点：功率因数校正  
可能错因：power_factor_correction_error  
回答：  
理想情况下不会。并联电容主要提供容性无功，用来抵消感性负载所需的一部分无功功率。原负载消耗的有功功率基本不变，但电源侧电流和视在功率会减小。

### Q9：交流最大功率传输为什么不是简单 \(Z_L=Z_{\mathrm{th}}\)？

所属知识点：最大有功功率传输  
可能错因：max_ac_power_transfer_error  
回答：  
交流电路中阻抗有电阻和电抗两部分。为了让负载得到最大有功功率，需要电抗部分相互抵消，使总电抗为零，同时电阻匹配。因此当负载阻抗可调时，条件是 \(Z_L=Z_{\mathrm{th}}^{*}\)，即共轭匹配。

### Q10：功率表读数是什么功率？

所属知识点：有功功率测量  
可能错因：wattmeter_connection_error  
回答：  
普通功率表读数是有功功率，即一个周期内瞬时功率的平均值。它不是视在功率，也不是瞬时功率。正弦稳态下读数等于 \(UI\cos\varphi\)。

### Q11：用电压表和电流表读数相乘能得到有功功率吗？

所属知识点：功率测量  
可能错因：apparent_power_error  
回答：  
不一定。电压表和电流表读数相乘得到的是视在功率 \(S=UI\)。只有当功率因数为 1 时，它才等于有功功率。若负载有感性或容性成分，还需要乘以功率因数，或直接用功率表测量。

### Q12：V-A-W 法测阻抗时，为什么还需要功率表？

所属知识点：交流参数测量  
可能错因：ac_measurement_error  
回答：  
电压表和电流表只能给出阻抗模值 \(|Z|=U/I\)，不能区分电阻和电抗。功率表给出有功功率 \(P\)，可以求出功率因数和电阻部分，再结合 \(|Z|\) 求电抗。

## 9. 学习建议

### 9.1 概念薄弱型

适用表现：  
混淆瞬时功率、有功功率、无功功率、视在功率和复功率。

学习建议：

1. 先画一张功率概念表，分别写出符号、单位、物理意义；
2. 重点记住：\(P\) 是平均消耗，\(Q\) 是能量交换，\(S\) 是容量；
3. 每做一道题都先判断要求的是 \(p(t)\)、\(P\)、\(Q\)、\(S\) 还是 \(\underline{S}\)。

### 9.2 公式误用型

适用表现：  
会背公式，但经常把有效值、最大值、相角、符号搞错。

学习建议：

1. 功率公式默认使用有效值；
2. 计算 \(Q\) 前先固定 \(\varphi=\varphi_u-\varphi_i\)；
3. 计算复功率一定写 \(\underline{S}=\underline{U}\underline{I}^{*}\)；
4. 使用导纳公式时特别注意 \(Q=-U^2B\)。

### 9.3 解题流程混乱型

适用表现：  
多负载功率、功率因数校正、最大功率传输题中不知道先求什么。

学习建议：

1. 多负载题：先求每个负载的复功率，再相加；
2. 功率因数校正题：先求补偿前后无功差，再求电容；
3. 最大功率传输题：先从负载端口求戴维南等效，再判断可调条件；
4. 测量题：先明确每个仪表读数代表的物理量。

### 9.4 综合应用困难型

适用表现：  
遇到工程题、仪表题和含受控源最大功率题时容易失去方向。

学习建议：

1. 把电源和负载分开，明确谁提供功率、谁吸收功率；
2. 用复功率守恒检查 \(P\) 和 \(Q\) 是否平衡；
3. 含受控源端口求等效阻抗时优先用测试源法；
4. 工程计算中统一单位：W、kW、var、kvar、V·A、kV·A、F、μF；
5. 结果出来后检查数量级，例如补偿电容是否过大、功率因数是否超过 1。
