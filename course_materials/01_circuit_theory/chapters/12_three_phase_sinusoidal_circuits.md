## 1. 本章定位

本章是在正弦稳态相量分析、交流功率分析的基础上，进一步讨论**三相电源、三相负载、三相连接方式、三相功率与不对称三相电路**。三相电路是电力系统、工业供电、电机拖动、变频器、电力电子、楼宇配电和实验测量中的基础模型。

本章主要解决以下问题：

1. 如何描述对称三相电压、电流及其相序；
2. 如何区分相电压、线电压、相电流、线电流；
3. 如何分析 Y 形和 Δ 形三相负载；
4. 如何把对称三相电路化简为单相等效电路；
5. 如何计算三相有功功率、无功功率、视在功率和功率因数；
6. 如何处理不对称负载、中性点位移和中线电流；
7. 如何理解三相功率测量与三相配电工程应用。

本章与第10章正弦稳态相量法、第11章正弦稳态功率密切相关。后续学习电机学、电力电子、供配电技术、通信电源、嵌入式电源系统和自动化控制时，都会反复使用本章的三相电路语言。

## 3. 核心概念

### 3.1 三相电源

* 概念解释：三相电源由三个频率相同、幅值通常相等、相位依次相差 $120^\circ$ 的正弦电压源组成。
* 使用场景：发电机、电网供电、电机驱动、三相整流、电力电子变换器。
* 与单相电源的区别：单相系统只有一个正弦电压；三相系统有三个相互关联的正弦电压，可以形成旋转磁场并提高输电效率。
* 常见误区：
  * 把三相电源理解成三个互不相关的单相电源；
  * 只记住“相差 $120^\circ$”，但不判断正序或负序；
  * 把相电压和线电压混为一谈。

### 3.2 对称三相电源

* 概念解释：三个相电压幅值相等、频率相同、相位互差 $120^\circ$，且代数和为零的三相电源称为对称三相电源。
* 使用场景：理想三相电网、三相发电机、三相对称负载分析。
* 与不对称电源的区别：不对称电源可能幅值不等、相位差不为 $120^\circ$ 或相序异常。
* 常见误区：
  * 忽略相量和为零的性质；
  * 正序、负序判断错误；
  * 认为所有三相电路都可以直接按对称电路处理。

### 3.3 相序

* 概念解释：相序表示三相电压或电流达到正最大值的先后顺序，常见有正序和负序。
* 使用场景：电机转向判断、三相相量图、电源接线检查。
* 与相位差的区别：相位差说明两个量之间的角度关系；相序说明三个相量整体旋转时的先后顺序。
* 常见误区：
  * 只根据字母顺序判断相序，不看相量角度；
  * 接线改变后仍沿用原来的相序；
  * 负序情况下仍套用正序线电压公式。

### 3.4 相电压与线电压

* 概念解释：相电压是某相端点与中性点之间的电压；线电压是任意两相线之间的电压。
* 使用场景：三相电源、Y 形负载、三相四线制电路计算。
* 与相近概念的区别：
  * 相电压通常记作 $U_A, U_B, U_C$ 或 $U_{AN}, U_{BN}, U_{CN}$；
  * 线电压通常记作 $U_{AB}, U_{BC}, U_{CA}$。
* 常见误区：
  * 把线电压直接当相电压使用；
  * 忽略线电压相位相对于相电压有 $30^\circ$ 偏移；
  * 不区分电源侧中性点和负载侧中性点。

### 3.5 相电流与线电流

* 概念解释：相电流是流过某一相负载的电流；线电流是流过输电线的电流。
* 使用场景：Y 形负载、Δ 形负载、电流测量、线路损耗计算。
* 与相近概念的区别：
  * Y 形负载中，线电流等于相电流；
  * Δ 形负载中，线电流是两个相支路电流的相量差。
* 常见误区：
  * 在 Δ 形负载中误认为线电流等于相电流；
  * 忽略线电流与相电流之间的 $30^\circ$ 相位差；
  * 用标量相减代替相量相减。

### 3.6 Y 形连接

* 概念解释：三相电源或负载的一端连接到公共点，另一端分别接到 A、B、C 三相线，这种连接称为星形连接或 Y 形连接。
* 使用场景：三相四线制供电、低压配电、照明负载、电机定子绕组。
* 与 Δ 形连接的区别：Y 形存在中性点，可以引出中性线；Δ 形三个支路首尾相连，一般没有中性点。
* 常见误区：
  * 误把 Y 形负载的相电压看成线电压；
  * 中性点未连接时仍认为负载中性点电位必为电源中性点电位；
  * 不对称 Y 形负载中忽略中性点位移。

### 3.7 Δ 形连接

* 概念解释：三个相负载首尾相接形成闭合三角形，三相线分别接到三个连接点，这种连接称为三角形连接或 Δ 形连接。
* 使用场景：三相电机绕组、三相工业负载、三角形加热负载。
* 与 Y 形连接的区别：Δ 形负载每相直接承受线电压；线电流不是单个相支路电流，而是相邻两相支路电流的相量差。
* 常见误区：
  * 把 Δ 形相电压误写成线电压除以 $\sqrt{3}$；
  * 把线电流和相电流相等；
  * Δ-Y 等效时阻抗换算比例写反。

### 3.8 对称三相电路

* 概念解释：电源对称、负载三相阻抗相同、线路参数对称的三相电路称为对称三相电路。
* 使用场景：三相电机、对称三相负载、电力系统近似分析。
* 与不对称三相电路的区别：对称三相电路可只分析一相；不对称三相电路通常必须按完整三相电路列方程。
* 常见误区：
  * 只看到电源对称就认为整个电路对称；
  * 忽略负载阻抗是否相同；
  * 不对称负载仍用分相计算法。

### 3.9 分相计算法

* 概念解释：在对称三相电路中，只需取一相建立单相等效电路，求出一相电压、电流后，再根据相序推出另外两相。
* 使用场景：Y-Y、Y-Δ、Δ-Y、Δ-Δ 对称三相电路分析。
* 与一般三相方程法的区别：分相法依赖对称条件；一般方程法不要求对称。
* 常见误区：
  * 忽略三相对称条件；
  * 选错单相等效电压；
  * Δ 形负载未先等效为 Y 形就直接分相分析。

### 3.10 中性线与中性点位移

* 概念解释：中性线连接电源中性点与负载中性点；若负载中性点电位相对电源中性点发生变化，称为中性点位移。
* 使用场景：三相四线制、三相不平衡负载、家庭和楼宇配电。
* 与对称三相的区别：对称三相负载中性线电流为零，中性点位移为零；不对称三相负载中可能出现中性点位移和中线电流。
* 常见误区：
  * 认为中线永远没有电流；
  * 三相三线制不对称负载仍令负载中性点等于电源中性点；
  * 不区分工作中性线和保护接地线。

### 3.11 三相功率

* 概念解释：三相负载总功率等于三相各相功率之和。对称三相负载中，可用一相功率乘以 3。
* 使用场景：电机功率、工业负载功率、配电系统容量、功率因数补偿。
* 与单相功率的区别：三相功率既可以按相量逐相计算，也可以在对称条件下使用线电压、线电流公式。
* 常见误区：
  * 直接把单相公式用于总三相功率；
  * $\sqrt{3}$ 系数漏写或误用；
  * 功率因数角取错。

### 3.12 三相功率测量

* 概念解释：三相有功功率可以通过功率表测量。根据负载是否对称、是否有中线以及可接入测量仪表的条件，可采用一表法、二表法或三表法。
* 使用场景：实验测量、工业配电、三相负载功率检测。
* 与理论计算的区别：功率表接法必须满足电压线圈和电流线圈的参考方向要求，否则读数符号可能为负。
* 常见误区：
  * 二表法中把两个读数简单看作两相功率；
  * 负读数时直接取绝对值而不解释；
  * 不区分对称负载和不对称负载的测量条件。

## 4. 重要公式与适用条件

### 4.1 对称三相相电压

公式：

```math
\dot U_A = U\angle 0^\circ
```

```math
\dot U_B = U\angle -120^\circ
```

```math
\dot U_C = U\angle 120^\circ
```

符号含义：

* $\dot U_A,\dot U_B,\dot U_C$：三相相电压相量；
* $U$：相电压有效值。

适用条件：

* 对称三相电源；
* 采用正序 $A\to B\to C$；
* $A$ 相取为参考相。

常见误用：

* 负序电源仍按正序公式；
* 把峰值当有效值；
* 相角顺序写反。

### 4.2 对称三相相量和

公式：

```math
\dot U_A+\dot U_B+\dot U_C=0
```

符号含义：

* $\dot U_A,\dot U_B,\dot U_C$：三相对称相量。

适用条件：

* 三相幅值相等；
* 相位互差 $120^\circ$。

常见误用：

* 不对称三相仍使用该式；
* 对瞬时值与相量混用。

### 4.3 Y 形连接中线电压与相电压关系

公式：

```math
\dot U_{AB}=\dot U_A-\dot U_B
```

```math
U_l=\sqrt{3}U_p
```

正序条件下常用相角关系：

```math
\dot U_{AB}=\sqrt{3}\dot U_A\angle 30^\circ
```

符号含义：

* $U_l$：线电压有效值；
* $U_p$：相电压有效值；
* $\dot U_{AB}$：A 线相对于 B 线的线电压。

适用条件：

* 对称三相 Y 形电源或 Y 形负载；
* 正序相量关系明确。

常见误用：

* 只写幅值关系，忽略线电压超前对应相电压 $30^\circ$；
* 把 $U_l=\sqrt{3}U_p$ 用于 Δ 形相电压；
* 线电压方向写反。

### 4.4 Y 形连接中线电流与相电流关系

公式：

```math
I_l=I_p
```

符号含义：

* $I_l$：线电流有效值；
* $I_p$：相电流有效值。

适用条件：

* Y 形负载；
* 每一相负载与对应线电流串联。

常见误用：

* 把该关系误用于 Δ 形负载；
* 忽略不对称负载时三相电流相角和幅值可能不同。

### 4.5 Δ 形连接中相电压与线电压关系

公式：

```math
U_p=U_l
```

符号含义：

* $U_p$：每一相 Δ 支路电压；
* $U_l$：线电压。

适用条件：

* Δ 形负载；
* 每个负载支路直接接在线与线之间。

常见误用：

* 误写成 $U_p=U_l/\sqrt{3}$；
* 不注意支路电压方向与线电压方向是否一致。

### 4.6 Δ 形连接中线电流与相电流关系

公式：

```math
I_l=\sqrt{3}I_p
```

正序对称条件下，相位还存在 $30^\circ$ 差异。

符号含义：

* $I_l$：线电流有效值；
* $I_p$：Δ 支路相电流有效值。

适用条件：

* 对称 Δ 形负载；
* 三相支路阻抗相同。

常见误用：

* 不对称 Δ 形负载仍直接套 $\sqrt{3}$ 关系；
* 把线电流当作某一支路相电流；
* 忽略线电流是两个相邻支路电流的相量差。

### 4.7 Δ-Y 阻抗等效

对称三相负载中，若 Δ 形每相阻抗为 $Z_\Delta$，等效 Y 形每相阻抗为：

```math
Z_Y=\frac{Z_\Delta}{3}
```

反向变换：

```math
Z_\Delta=3Z_Y
```

符号含义：

* $Z_\Delta$：对称三角形每相阻抗；
* $Z_Y$：等效星形每相阻抗。

适用条件：

* 三相阻抗对称；
* 只保持三相端口对外等效。

常见误用：

* 非对称 Δ 形负载使用简单 $1/3$ 关系；
* 把电阻网络中的一般 Y-Δ 公式和对称特殊公式混淆；
* 变换后忘记负载相电压、电流定义改变。

### 4.8 对称 Y 形负载相电流

公式：

```math
\dot I_A=\frac{\dot U_{AN}}{Z_Y}
```

符号含义：

* $\dot I_A$：A 相线电流，也是 A 相负载电流；
* $\dot U_{AN}$：A 相相电压；
* $Z_Y$：每相负载阻抗。

适用条件：

* 对称 Y 形负载；
* 中性点不发生位移，或可采用分相等效。

常见误用：

* 把线电压 $\dot U_{AB}$ 直接除以 Y 形相阻抗；
* 不判断电源和负载相序是否一致。

### 4.9 对称 Δ 形负载相电流

公式：

```math
\dot I_{AB}=\frac{\dot U_{AB}}{Z_\Delta}
```

符号含义：

* $\dot I_{AB}$：Δ 形 AB 支路电流；
* $\dot U_{AB}$：AB 线电压；
* $Z_\Delta$：每相 Δ 支路阻抗。

适用条件：

* 对称或给定支路阻抗的 Δ 形负载；
* 支路电流方向与电压方向一致。

常见误用：

* 用相电压 $U_{AN}$ 计算 Δ 支路电流；
* 忘记线电流要由支路电流相量差得到。

### 4.10 Δ 形负载线电流的相量关系

公式：

```math
\dot I_A=\dot I_{AB}-\dot I_{CA}
```

```math
\dot I_B=\dot I_{BC}-\dot I_{AB}
```

```math
\dot I_C=\dot I_{CA}-\dot I_{BC}
```

符号含义：

* $\dot I_A,\dot I_B,\dot I_C$：线电流；
* $\dot I_{AB},\dot I_{BC},\dot I_{CA}$：Δ 支路相电流。

适用条件：

* Δ 形负载；
* 支路电流方向按相邻线之间统一规定。

常见误用：

* 使用代数差而非相量差；
* 支路电流方向选取后没有保持一致；
* 误把 $I_A=I_{AB}$。

### 4.11 中线电流

公式：

```math
\dot I_N=-(\dot I_A+\dot I_B+\dot I_C)
```

若按相电流流入负载为正，也常写成：

```math
\dot I_N=\dot I_A+\dot I_B+\dot I_C
```

具体符号取决于中线电流参考方向。

符号含义：

* $\dot I_N$：中线电流；
* $\dot I_A,\dot I_B,\dot I_C$：三相线电流。

适用条件：

* 三相四线制；
* 中线参考方向明确。

常见误用：

* 对对称负载外的情况仍令中线电流为零；
* 忽略相量求和，直接把有效值相加；
* 中线参考方向与 KCL 方程不一致。

### 4.12 不对称 Y 形负载的中性点电压

若负载中性点为 $N'$，电源中性点为 $N$，各相导纳为 $Y_A,Y_B,Y_C$，则可用节点法写：

```math
\dot U_{N'N}=\frac{Y_A\dot U_A+Y_B\dot U_B+Y_C\dot U_C}{Y_A+Y_B+Y_C}
```

符号含义：

* $\dot U_{N'N}$：负载中性点相对电源中性点的位移电压；
* $Y_A,Y_B,Y_C$：三相负载导纳；
* $\dot U_A,\dot U_B,\dot U_C$：电源相电压。

适用条件：

* 三相三线制 Y 形不对称负载；
* 无中线或中线开路；
* 电源三相相电压已知。

常见误用：

* 认为负载中性点永远等于电源中性点；
* 分母漏掉某相导纳；
* 把阻抗直接代入导纳公式。

### 4.13 不对称 Y 形负载相电压

公式：

```math
\dot U_{AN'}=\dot U_A-\dot U_{N'N}
```

```math
\dot U_{BN'}=\dot U_B-\dot U_{N'N}
```

```math
\dot U_{CN'}=\dot U_C-\dot U_{N'N}
```

符号含义：

* $\dot U_{AN'},\dot U_{BN'},\dot U_{CN'}$：实际加在各相负载上的相电压；
* $\dot U_{N'N}$：负载中性点位移电压。

适用条件：

* 中性点位移已知或可求；
* 负载为 Y 形不对称负载。

常见误用：

* 直接把电源相电压作为负载相电压；
* 中性点位移方向写反；
* 不画相量图导致符号错误。

### 4.14 对称三相有功功率

公式：

```math
P=3U_p I_p\cos\varphi
```

也可写为：

```math
P=\sqrt{3}U_l I_l\cos\varphi
```

符号含义：

* $P$：三相总有功功率；
* $U_p,I_p$：相电压、相电流有效值；
* $U_l,I_l$：线电压、线电流有效值；
* $\varphi$：每相负载电压与电流之间的相位差。

适用条件：

* 对称三相负载；
* 三相功率因数相同；
* 电压、电流使用有效值。

常见误用：

* 不对称负载直接使用该公式；
* $\varphi$ 取成线电压与线电流的夹角；
* 漏写 $\sqrt{3}$ 或把 $U_p,I_p$ 与 $U_l,I_l$ 混用。

### 4.15 对称三相无功功率

公式：

```math
Q=3U_p I_p\sin\varphi
```

也可写为：

```math
Q=\sqrt{3}U_l I_l\sin\varphi
```

符号含义：

* $Q$：三相总无功功率。

适用条件：

* 对称三相正弦稳态电路；
* 按统一符号规定，感性负载通常 $Q>0$，容性负载通常 $Q<0$。

常见误用：

* 感性、容性符号判断错误；
* 用功率因数角的绝对值导致无功符号丢失；
* 把无功功率单位写成 W。

### 4.16 对称三相视在功率

公式：

```math
S=3U_p I_p=\sqrt{3}U_l I_l
```

复功率形式：

```math
\dot S=P+jQ
```

符号含义：

* $S$：视在功率，单位 VA；
* $\dot S$：复功率；
* $P$：有功功率；
* $Q$：无功功率。

适用条件：

* 对称三相电路；
* 正弦稳态有效值。

常见误用：

* 把 $S$ 当作有功功率；
* 不区分 VA、W、var；
* 复功率方向与电压电流参考方向不一致。

### 4.17 三相功率因数

公式：

```math
\lambda=\frac{P}{S}=\cos\varphi
```

符号含义：

* $\lambda$：功率因数；
* $P$：有功功率；
* $S$：视在功率。

适用条件：

* 对称三相正弦稳态电路；
* $\varphi$ 为每相负载电压与电流夹角。

常见误用：

* 把线电压与线电流的相角差直接当 $\varphi$；
* 忽略功率因数超前或滞后；
* 功率因数校正时只改有功功率。

### 4.18 三相电路线路损耗

公式：

```math
P_{\mathrm{loss}}=3I_l^2R_l
```

符号含义：

* $P_{\mathrm{loss}}$：三相线路总铜耗；
* $I_l$：线电流；
* $R_l$：每根相线电阻。

适用条件：

* 三相对称线路；
* 每根相线电阻相同；
* 中线电流为零或中线损耗另计。

常见误用：

* 只计算一根线的损耗；
* 将相电流和线电流混用；
* 忽略不平衡时各相电流不同。

### 4.19 三相功率因数校正

校正前后有功功率不变，若负载由功率因数 $\cos\varphi_1$ 提高到 $\cos\varphi_2$，所需补偿无功功率为：

```math
Q_C=P(\tan\varphi_1-\tan\varphi_2)
```

若三相并联电容为 Y 形连接，每相电容可由：

```math
Q_C=3\omega C U_p^2
```

若三相并联电容为 Δ 形连接，每相电容可由：

```math
Q_C=3\omega C U_l^2
```

符号含义：

* $Q_C$：需要补偿的容性无功功率总量；
* $P$：负载有功功率；
* $\omega$：角频率；
* $C$：每相补偿电容。

适用条件：

* 正弦稳态；
* 负载有功功率近似不变；
* 电容并联在负载端。

常见误用：

* 把补偿后的无功功率直接设为零，但题目要求只提高到某一功率因数；
* Y 接和 Δ 接电容电压取错；
* 忽略电容提供的是容性无功。

### 4.20 二表法测量三相有功功率

公式：

```math
P=P_1+P_2
```

对于对称三相负载，还可用两个表读数求功率因数角：

```math
\tan\varphi=\sqrt{3}\frac{P_1-P_2}{P_1+P_2}
```

符号含义：

* $P_1,P_2$：两只功率表读数；
* $P$：三相总有功功率；
* $\varphi$：负载功率因数角。

适用条件：

* 三相三线制；
* 二表法接线正确；
* 功率因数角公式通常用于对称负载。

常见误用：

* 把 $P_1$、$P_2$ 分别当作某两相功率；
* 某表反偏或负读数时不处理符号；
* 不对称负载时仍用功率因数角公式。

## 5. 典型题型

### 题型 1：判断三相电源是否对称及相序

考查知识点：

* 三相相量；
* 幅值、频率、相位；
* 正序与负序。

解题关键：

1. 将三相电压统一写成相量形式；
2. 检查幅值是否相等、频率是否相同；
3. 判断相位是否互差 $120^\circ$；
4. 根据相角先后判断正序或负序。

常见错误：

* 只看相位差，不看幅值是否相等；
* 正序、负序判断反；
* 峰值与有效值混用。

对应错因标签：

* balanced_three_phase_condition_error
* phase_sequence_error
* rms_peak_confusion

### 题型 2：相电压与线电压换算

考查知识点：

* Y 形连接；
* 相电压；
* 线电压；
* 相量差。

解题关键：

1. 明确给定电压是相电压还是线电压；
2. 确定相序；
3. 使用 $U_l=\sqrt{3}U_p$；
4. 保留线电压相对相电压的 $30^\circ$ 相位关系。

常见错误：

* 幅值会换算但相位漏掉；
* 把 $U_{AB}$ 与 $U_{BA}$ 混淆；
* Δ 形连接也误用 Y 形线相关系。

对应错因标签：

* phase_line_quantity_confusion
* line_voltage_phase_voltage_error
* voltage_polarity_error

### 题型 3：Y 形对称负载电流计算

考查知识点：

* Y 形负载；
* 线电流等于相电流；
* 分相计算法。

解题关键：

1. 将电源电压换成对应相电压；
2. 用 $\dot I_A=\dot U_{AN}/Z_Y$ 求一相电流；
3. 根据相序写出另外两相；
4. 判断中线电流是否为零。

常见错误：

* 直接用线电压除以 Y 形相阻抗；
* 忽略阻抗角；
* 对称负载下仍计算出非零中线电流。

对应错因标签：

* line_voltage_phase_voltage_error
* impedance_model_error
* neutral_current_error

### 题型 4：Δ 形对称负载电流计算

考查知识点：

* Δ 形负载；
* 相电流；
* 线电流；
* 相量差。

解题关键：

1. 每个 Δ 支路承受线电压；
2. 先求支路相电流；
3. 再用相量差求线电流；
4. 对称时可用 $I_l=\sqrt{3}I_p$，但仍要注意相位。

常见错误：

* 误认为 Δ 形线电流等于相电流；
* 忽略线电流与相电流之间的相位差；
* 支路电流方向定义混乱。

对应错因标签：

* line_current_phase_current_error
* line_current_phase_current_error
* line_current_phase_current_error

### 题型 5：Δ-Y 等效后分相计算

考查知识点：

* 对称 Δ-Y 等效；
* 分相计算法；
* 端口等效。

解题关键：

1. 判断三相负载是否对称；
2. 将 Δ 形负载换算为等效 Y 形负载；
3. 取一相单相等效电路计算；
4. 根据相量关系还原线电压、线电流、相电流。

常见错误：

* 非对称 Δ 形负载使用 $Z_Y=Z_\Delta/3$；
* 等效后忘记原 Δ 支路电流不是等效 Y 支路电流；
* 端口相量关系没有还原。

对应错因标签：

* star_delta_connection_error
* star_delta_connection_error
* line_current_phase_current_error

### 题型 6：对称三相电路功率计算

考查知识点：

* 三相有功功率；
* 三相无功功率；
* 视在功率；
* 功率因数。

解题关键：

1. 判断是否对称；
2. 确定使用相电压相电流还是线电压线电流；
3. 正确取电压与电流夹角 $\varphi$；
4. 用 $P,Q,S$ 的三相公式计算。

常见错误：

* 漏掉 $\sqrt{3}$ 或 3；
* 用线电压与线电流的相角差当负载阻抗角；
* 感性、容性无功符号判断错误。

对应错因标签：

* three_phase_power_formula_error
* power_factor_error
* reactive_power_sign_error

### 题型 7：三相功率因数校正

考查知识点：

* 功率因数；
* 无功补偿；
* 并联电容；
* Y/Δ 电容连接。

解题关键：

1. 根据原功率因数求原无功；
2. 根据目标功率因数求补偿后无功；
3. 两者差值为电容需要提供的容性无功；
4. 根据电容连接方式选取相电压或线电压。

常见错误：

* 目标功率因数理解错误；
* Y 接和 Δ 接电容电压取错；
* 将补偿电容的无功符号写成感性。

对应错因标签：

* power_factor_correction_error
* capacitor_compensation_error
* reactive_power_sign_error

### 题型 8：不对称 Y 形负载中性点位移

考查知识点：

* 不对称负载；
* 中性点位移；
* 节点电压法；
* 中线电流。

解题关键：

1. 判断负载中性点是否接到电源中性点；
2. 若无中线，求负载中性点位移电压；
3. 用位移电压求各相负载实际相电压；
4. 再求各相电流。

常见错误：

* 无中线时仍令负载中性点电位为零；
* 阻抗和导纳混用；
* 实际负载相电压未修正。

对应错因标签：

* neutral_shift_error
* admittance_model_error
* neutral_shift_error

### 题型 9：三相四线制中线电流计算

考查知识点：

* 三相四线制；
* 不对称负载；
* KCL；
* 相量和。

解题关键：

1. 按各相实际相电压分别求相电流；
2. 用 KCL 求中线电流；
3. 中线电流必须按相量求和；
4. 判断中线电流方向。

常见错误：

* 把三个电流有效值直接相加；
* 对称与不对称条件混淆；
* 中线参考方向写反。

对应错因标签：

* neutral_current_error
* neutral_current_error
* symmetry_condition_error

### 题型 10：不对称 Δ 形负载分析

考查知识点：

* Δ 形负载；
* 支路电流；
* 线电流；
* 不对称相量计算。

解题关键：

1. 每个支路直接使用对应线电压；
2. 分别求三个支路电流；
3. 用 KCL 求线电流；
4. 不使用对称条件下的 $\sqrt{3}$ 关系。

常见错误：

* 不对称时仍用 $I_l=\sqrt{3}I_p$；
* 支路电流方向混乱；
* 线电流相量差写错。

对应错因标签：

* unbalanced_three_phase_error
* line_current_phase_current_error
* line_current_phase_current_error

### 题型 11：二表法测三相功率

考查知识点：

* 二表法；
* 三相三线制；
* 功率表读数；
* 功率因数。

解题关键：

1. 判断是否适合二表法；
2. 根据接线确定两个功率表读数符号；
3. 总有功功率为两个读数代数和；
4. 对称负载时可进一步由读数求功率因数。

常见错误：

* 把两个读数分别看成两相功率；
* 忽略负读数；
* 不对称负载仍使用功率因数角公式。

对应错因标签：

* three_phase_power_measurement_error
* three_phase_power_measurement_error
* three_phase_power_measurement_error

### 题型 12：三相配电工程应用判断

考查知识点：

* 三相四线制；
* 中性线；
* 保护接地；
* 负载平衡；
* 配电安全。

解题关键：

1. 区分相线、中性线、保护线；
2. 判断单相负载在三相中的分配是否均衡；
3. 分析中性线电流和电压偏移风险；
4. 结合额定电压、功率和保护装置判断安全性。

常见错误：

* 把中性线和保护线混为一谈；
* 忽略单相负载不均衡导致的中线电流；
* 不检查额定功率和额定电流。

对应错因标签：

* distribution_safety_check_missing
* distribution_safety_check_missing
* distribution_safety_check_missing

## 6. 常见学习困难

* 相量关系不熟：学生容易只记幅值关系，不会处理 $30^\circ$ 相移。
* 相电压和线电压混淆：尤其在 Y 形与 Δ 形切换时容易用错电压。
* 相电流和线电流混淆：尤其在 Δ 形负载中容易直接把支路电流当线电流。
* 正序与负序混淆：相量图画反后，线电压、线电流相位都会错。
* 分相计算法条件不清：不对称负载仍强行只算一相。
* Δ-Y 等效误用：非对称负载使用对称三相的特殊等效公式。
* 中性点位移理解困难：无中线不对称 Y 形负载中，负载相电压不再等于电源相电压。
* 中线电流计算错误：把相量和当标量和。
* 三相功率公式混乱：$3U_pI_p$ 与 $\sqrt{3}U_lI_l$ 混用，功率因数角取错。
* 功率因数校正不熟：不知道电容接成 Y 或 Δ 时电压不同，导致电容值差三倍。
* 二表法读数理解不足：不能正确解释一个表负读数或两个表读数与功率因数的关系。
* 工程接线概念薄弱：不区分相线、中性线、保护线和接地。

## 7. 错题诊断标签
### balanced_three_phase_condition_error

中文名称：三相对称条件判断错误  
适用场景：判断三相电源或三相负载是否对称  
典型表现：只检查相位差，不检查幅值和频率；或只看负载数量，不看三相阻抗是否相同  
诊断提示：依次检查幅值、频率、相位差和三相参数是否相同

### phase_sequence_error

中文名称：相序判断错误  
适用场景：正序、负序相量判断，线电压线电流相位关系  
典型表现：把 $A\to B\to C$ 与 $A\to C\to B$ 混淆  
诊断提示：画出相量图，按逆时针方向或时间先后判断相序

### phase_line_quantity_confusion

中文名称：相电压与线电压混淆  
适用场景：Y 形电源、Y 形负载、线电压换算  
典型表现：Y 形负载中直接用线电压计算相电流  
诊断提示：先判断电压两端是“相线-中性点”还是“两根相线”

### line_voltage_phase_voltage_error

中文名称：线电压相位偏移错误  
适用场景：由相电压求线电压  
典型表现：只乘 $\sqrt{3}$，漏掉 $30^\circ$ 相位差  
诊断提示：线电压是两个相电压的相量差，必须保留相位信息

### line_current_phase_current_error

中文名称：Δ 形线电流与相电流混淆  
适用场景：Δ 形负载电流计算  
典型表现：把 Δ 支路电流直接当成线电流  
诊断提示：Δ 形线电流是相邻两个支路电流的相量差

### star_delta_connection_error

中文名称：Δ-Y 等效条件错误  
适用场景：对称三相负载等效  
典型表现：非对称 Δ 形负载直接使用 $Z_Y=Z_\Delta/3$  
诊断提示：只有三相阻抗相等时才能使用对称特殊公式

### three_phase_power_formula_error

中文名称：三相功率公式错误  
适用场景：对称三相有功、无功、视在功率计算  
典型表现：漏写 3 或 $\sqrt{3}$，相量和标量混用  
诊断提示：先明确使用相量 $U_p,I_p$ 还是线量 $U_l,I_l$

### power_factor_error

中文名称：功率因数角错误  
适用场景：三相功率因数、有功无功功率计算  
典型表现：把线电压和线电流夹角当作每相负载阻抗角  
诊断提示：功率因数角是负载相电压与相电流的夹角

### reactive_power_sign_error

中文名称：无功功率符号错误  
适用场景：感性、容性负载无功功率计算  
典型表现：容性无功写成正，或感性无功写成负  
诊断提示：结合课程约定判断感性吸收无功、容性提供无功

### power_factor_correction_error

中文名称：功率因数校正错误  
适用场景：三相并联电容补偿  
典型表现：补偿无功计算方向错误，或校正目标理解错误  
诊断提示：补偿前后有功功率不变，用 $Q_C=P(\tan\varphi_1-\tan\varphi_2)$ 判断

### capacitor_compensation_error

中文名称：补偿电容连接方式错误  
适用场景：Y 接或 Δ 接三相补偿电容计算  
典型表现：Y 接电容使用线电压，或 Δ 接电容使用相电压  
诊断提示：先判断每只电容实际承受的是相电压还是线电压

### neutral_shift_error

中文名称：中性点位移忽略  
适用场景：无中线不对称 Y 形负载  
典型表现：直接令负载中性点电压等于电源中性点电压  
诊断提示：若无中线且负载不对称，应先求中性点位移

### admittance_model_error

中文名称：导纳与阻抗混淆  
适用场景：不对称三相节点法、中性点位移公式  
典型表现：公式需要导纳却直接代入阻抗  
诊断提示：节点法中支路电流通常用导纳乘以电压差表示

### neutral_current_error

中文名称：中线电流相量和错误  
适用场景：三相四线制中线电流计算  
典型表现：直接把三个相电流有效值相加  
诊断提示：中线电流必须按相量求和

### unbalanced_three_phase_error

中文名称：不对称 Δ 形电流计算错误  
适用场景：不对称 Δ 形负载  
典型表现：对每个支路仍套对称电流关系  
诊断提示：不对称 Δ 形应逐支路求相电流，再由 KCL 求线电流

### three_phase_power_measurement_error

中文名称：二表法理解错误  
适用场景：三相三线制功率测量  
典型表现：把两个功率表读数当成两相功率  
诊断提示：二表法两个读数是测量组合量，总功率为代数和

### distribution_safety_check_missing

中文名称：中性线与保护线混淆  
适用场景：三相四线制、低压配电、家庭用电  
典型表现：把工作中性线当作保护接地线  
诊断提示：中性线参与工作电流回路，保护线用于故障保护，功能不同

## 8. Agent 问答样例

### Q1：三相电源为什么要相差 120°？

所属知识点：三相电源  
可能错因：phase_sequence_error  
回答：  
三个相量相差 $120^\circ$ 且幅值相等时，相量和为零，系统更平衡，能形成旋转磁场，也便于电机运行和远距离输电。对称三相系统中任一时刻总功率也更平稳。

### Q2：相电压和线电压怎么区分？

所属知识点：相电压与线电压  
可能错因：phase_line_quantity_confusion  
回答：  
相电压是相线对中性点的电压，如 $U_{AN}$；线电压是两根相线之间的电压，如 $U_{AB}$。Y 形对称电路中 $U_l=\sqrt{3}U_p$，并且线电压相对对应相电压有 $30^\circ$ 相位差。

### Q3：Y 形负载中线电流和相电流相等吗？

所属知识点：Y 形连接  
可能错因：line_voltage_phase_voltage_error  
回答：  
相等。Y 形负载中，每一根相线和对应的一相负载串联，所以线电流就是该相负载电流。但相电压不是线电压，计算电流时要用负载相电压除以相阻抗。

### Q4：Δ 形负载中为什么线电流不是相电流？

所属知识点：Δ 形连接  
可能错因：line_current_phase_current_error  
回答：  
Δ 形负载中，每根相线连接到两个支路的交点，线电流等于相邻两个支路电流的相量差。只有在对称 Δ 形负载中，线电流有效值才满足 $I_l=\sqrt{3}I_p$，但相位仍相差 $30^\circ$。

### Q5：什么时候可以只分析三相电路中的一相？

所属知识点：分相计算法  
可能错因：balanced_three_phase_condition_error  
回答：  
只有在电源对称、负载对称、线路参数也对称时，才可以只分析一相。求出一相后，其余两相可通过相差 $120^\circ$ 得到。不对称负载不能直接用分相计算法。

### Q6：没有中线的不对称 Y 形负载为什么要考虑中性点位移？

所属知识点：中性点位移  
可能错因：neutral_shift_error  
回答：  
无中线时，负载中性点没有被强制固定在电源中性点电位上。若三相负载不相等，三相电流不平衡，负载中性点电位会偏移，导致各相负载实际电压不再等于电源相电压。

### Q7：三相总功率为什么可以写成 $\sqrt{3}U_lI_l\cos\varphi$？

所属知识点：三相功率  
可能错因：three_phase_power_formula_error  
回答：  
对称三相负载总有功功率为三相功率之和，即 $P=3U_pI_p\cos\varphi$。利用 Y 或 Δ 对称关系都可以化成 $P=\sqrt{3}U_lI_l\cos\varphi$。这里的 $\varphi$ 是每相负载电压与相电流的夹角。

### Q8：功率因数校正时为什么常并联电容？

所属知识点：功率因数校正  
可能错因：power_factor_correction_error  
回答：  
很多工业负载呈感性，会吸收感性无功，使功率因数降低。并联电容可以提供容性无功，抵消一部分感性无功，使电源提供的无功减少，线路电流下降，功率因数提高。

### Q9：二表法中两个功率表读数分别是两相功率吗？

所属知识点：二表法  
可能错因：three_phase_power_measurement_error  
回答：  
不是。二表法中两个功率表读数是由特定电压线圈和电流线圈组合得到的测量量，不等于简单的两相功率。三相三线制中，总有功功率等于两个读数的代数和。

### Q10：三相四线制中中线一定没有电流吗？

所属知识点：中线电流  
可能错因：neutral_current_error  
回答：  
不一定。对称三相负载时三相电流相量和为零，中线电流为零；不对称负载时三相电流相量和通常不为零，中线会有电流。

### Q11：为什么家庭用电会涉及三相电？

所属知识点：三相配电应用  
可能错因：distribution_safety_check_missing  
回答：  
居民楼和建筑物常由三相四线制供电，再把单相负载分配到不同相上。这样可以提高供电容量并尽量平衡三相负载。实际接线中还要区分相线、中性线和保护线。

## 9. 学习建议

### 9.1 概念薄弱型

适用表现：  
分不清相电压、线电压、相电流、线电流；不知道 Y 形和 Δ 形连接的区别。

学习建议：

1. 先画出 Y 形和 Δ 形连接图；
2. 在图中标出“相量”和“线量”；
3. 用表格对比 Y 形与 Δ 形的电压、电流关系；
4. 每题先判断连接方式，再选公式。

### 9.2 相量关系薄弱型

适用表现：  
会算幅值，但相角经常错；正序、负序、$30^\circ$ 相位差经常混淆。

学习建议：

1. 每次线相换算都画一个简化相量图；
2. 不要只记 $\sqrt{3}$，还要记线电压和线电流的相角偏移；
3. 先统一选 A 相为参考，再推 B、C 相。

### 9.3 公式误用型

适用表现：  
对称和不对称都套同一组公式，功率公式中 $3$、$\sqrt{3}$ 经常写错。

学习建议：

1. 每次使用三相功率公式前先判断是否对称；
2. 明确公式使用的是相电压相电流，还是线电压线电流；
3. 对不对称负载，优先逐相计算后再求和。

### 9.4 不对称电路困难型

适用表现：  
遇到不对称 Y 形负载、无中线电路、中性点位移就不会列方程。

学习建议：

1. 把负载中性点单独设为未知节点；
2. 用节点电压法求中性点位移；
3. 再求各相实际负载电压和电流；
4. 中线电流必须用相量和，不要用有效值相加。

### 9.5 工程应用薄弱型

适用表现：  
会做计算题，但无法解释功率因数校正、三相配电、中性线和保护线等工程意义。

学习建议：

1. 把三相功率和配电容量联系起来理解；
2. 理解并联电容不是“增加有功功率”，而是补偿无功；
3. 学会从安全角度检查额定电流、导线容量和接地保护。

## 10. 与其他章节的关系

* 与第10章正弦稳态分析的关系：三相电路的核心计算仍然依赖相量法、阻抗、导纳、相量图和正弦稳态 KCL/KVL。
* 与第11章正弦稳态功率的关系：三相有功、无功、视在功率和功率因数直接建立在单相交流功率基础上。
* 与第2章等效变换的关系：Δ-Y 等效本质上是三端网络等效在对称三相负载中的特殊应用。
* 与第3章电路分析方法的关系：不对称三相电路通常需要节点法、网孔法或相量方程完整求解。
* 与第4章电路定理的关系：三相电路中有时可对每相或每个端口应用戴维南等效、叠加等方法，但必须保持相量和端口定义一致。
* 与电机学的关系：三相对称电流可以产生旋转磁场，是三相异步电机、同步电机和电机控制的基础。
* 与电力电子的关系：三相整流、三相逆变、PWM 变换、并网控制都需要三相相量和功率概念。
* 与供配电技术的关系：三相四线制、中性线、功率因数补偿、配电箱和保护接地都是实际供配电系统的核心内容。
* 与实验测量的关系：三相功率测量、二表法、功率因数测量和电能计量都依赖本章知识。
