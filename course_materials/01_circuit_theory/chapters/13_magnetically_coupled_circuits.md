## 1. 本章定位

本章讨论电路中不同线圈之间通过磁场发生相互影响时的建模与分析方法。前面章节主要处理电阻、电容、电感之间通过导线连接形成的电路，而本章引入“磁耦合”这一新的连接机制：两个线圈即使没有电气导通，也可以通过共同磁通产生感应电压。

本章是正弦稳态电路、交流功率、三相电路之后的重要扩展，主要服务于以下能力：

1. 理解耦合电感、互感、同名端、耦合系数的物理意义；
2. 掌握含耦合电感电路的 KVL、网孔法和相量分析；
3. 掌握去耦等效、反映阻抗、理想变压器和实际变压器模型；
4. 能够分析变压器中的电压变换、电流变换、阻抗变换和功率传输；
5. 了解磁耦合在无线充电、隔离变换、电能传输和测量中的工程应用。

对课程 Agent 来说，本章是错题诊断中非常容易出现“符号方向错误”的章节。学生常见问题包括：同名端判断错误、互感电压正负号写错、把互感项漏掉、理想变压器电流比方向写反、阻抗折算到错误侧、把实际变压器和理想变压器模型混用等。

## 3. 核心概念

### 3.1 磁耦合

* 概念解释：当一个线圈中的电流变化产生的磁通同时穿过另一个线圈时，第二个线圈中会产生感应电压，这种通过磁场发生的相互作用称为磁耦合。
* 使用场景：变压器、互感线圈、无线能量传输、电流互感器、电压互感器、耦合谐振电路。
* 与相近概念的区别：
  * 普通电感只考虑自身电流产生的自感电压；
  * 耦合电感还要考虑其他线圈电流变化产生的互感电压。
* 常见误区：
  * 只把线圈当作普通电感，漏掉互感项；
  * 认为只有导线连接才会影响电路；
  * 忽略同名端导致互感项符号写反。

### 3.2 自感与互感

* 概念解释：自感反映线圈自身电流变化对自身磁链的影响；互感反映一个线圈电流变化对另一个线圈磁链和感应电压的影响。
* 使用场景：列写耦合电感的时域方程、相量方程、储能表达式。
* 与相近概念的区别：
  * 自感系数通常记为 \(L_1,L_2\)；
  * 互感系数通常记为 \(M\)，描述两个线圈之间的耦合强弱。
* 常见误区：
  * 把互感 \(M\) 当成第三个独立电感；
  * 忽略 \(M\) 的符号由同名端和参考方向共同决定；
  * 认为互感电压只与本线圈电流有关。

### 3.3 同名端

* 概念解释：同名端用于表示两个线圈绕向和磁通方向之间的相对关系。若两个电流同时从各自同名端流入，所产生的磁通相互加强；若一个从同名端流入、另一个从异名端流入，则磁通相互削弱。
* 使用场景：判断互感项正负号、串联耦合电感等效电感、理想变压器电压电流参考方向。
* 与相近概念的区别：
  * 同名端是磁耦合方向的标记，不是实际电气连接点；
  * 同名端符号通常用点标法表示。
* 常见误区：
  * 看到点端就机械加正号，不结合电流参考方向；
  * 把同名端当成必须接在一起的端子；
  * 判断串联加强/削弱时只看线圈位置，不看电流进入端。

### 3.4 加强耦合与削弱耦合

* 概念解释：若两个线圈电流产生的磁通方向相同，则为加强耦合；若磁通方向相反，则为削弱耦合。
* 使用场景：耦合电感串联等效、等效电感计算、储能表达式判断。
* 与相近概念的区别：
  * 加强或削弱不是由电流数值正负单独决定，而是由电流参考方向、同名端和实际电流符号共同决定；
  * 在公式中常体现为 \(+2M\) 或 \(-2M\) 项。
* 常见误区：
  * 把“同名端相连”简单等同于加强耦合；
  * 串联后没有重新判断电流进入同名端的情况；
  * 把 \(L_1+L_2+2M\) 和 \(L_1+L_2-2M\) 混用。

### 3.5 耦合系数

* 概念解释：耦合系数 \(k\) 表示两个线圈磁耦合的紧密程度，反映互感相对于自感几何平均值的大小。
* 使用场景：判断耦合强弱、理想变压器近似条件、无线能量传输效率分析。
* 与相近概念的区别：
  * \(M\) 是互感系数，具有亨利单位；
  * \(k\) 是无量纲系数，范围通常为 \(0\le k\le 1\)。
* 常见误区：
  * 认为 \(k=1\) 在实际磁路中普遍成立；
  * 把弱耦合系统直接当作理想变压器；
  * 忽略漏磁导致的等效误差。

### 3.6 耦合电感储能

* 概念解释：耦合电感的磁场储能不仅包含各线圈自身储能，还包含与互感有关的交叉项。
* 使用场景：判断耦合电感参数是否物理可行、分析初始储能、推导耦合系数约束。
* 与相近概念的区别：
  * 普通独立电感储能为 \(\frac12Li^2\)；
  * 两个耦合电感还会出现 \(\pm M i_1 i_2\) 形式的互能项。
* 常见误区：
  * 储能公式中漏掉互感项；
  * 互感项符号与同名端方向不一致；
  * 得到不满足 \(M^2\le L_1L_2\) 的参数仍继续计算。

### 3.7 去耦等效

* 概念解释：去耦等效是将含互感的电路转换为不含显式互感、但端口关系等效的普通电感网络，以便用常规电路分析方法求解。
* 使用场景：含多个耦合电感的网孔分析、正弦稳态复杂电路、互感线圈共端连接网络。
* 与相近概念的区别：
  * 去耦等效不是随意删除互感；
  * 去耦后的等效电感可能出现 \(L\pm M\) 或与多个互感组合的项。
* 常见误区：
  * 没有共端条件就套某一种去耦公式；
  * 去耦后改变端口连接关系；
  * 忽略互感方向导致等效电感符号错误。

### 3.8 反映阻抗与映射阻抗

* 概念解释：耦合线圈中一个回路的负载会通过互感作用反映到另一个回路中，表现为原边看到的附加阻抗，这类阻抗称为反映阻抗或映射阻抗。
* 使用场景：线性变压器输入阻抗、负载折算、最大功率传输、阻抗匹配。
* 与相近概念的区别：
  * 普通串联阻抗直接在电气回路中；
  * 反映阻抗通过磁耦合间接出现在另一侧。
* 常见误区：
  * 把负载阻抗原样搬到原边；
  * 漏掉 \((\omega M)^2\) 或变比平方项；
  * 阻抗折算到错误侧。

### 3.9 线性变压器

* 概念解释：线性变压器用两个或多个耦合电感及其绕组电阻来描述实际变压器在小信号或线性磁化区的行为。
* 使用场景：实际铁心变压器近似分析、输入阻抗计算、最大功率传输、电压和电流变换。
* 与理想变压器的区别：
  * 线性变压器保留绕组电阻、漏感、互感不完全等实际因素；
  * 理想变压器忽略损耗、漏磁和励磁电流。
* 常见误区：
  * 用理想变压器关系分析明显含损耗或漏感的电路；
  * 没有将副边负载折算到原边；
  * 忽略绕组电阻对功率传输的影响。

### 3.10 理想变压器

* 概念解释：理想变压器是一种满足全耦合、绕组无损耗、励磁电流可忽略的理想二端口元件。它只改变电压、电流和阻抗的比例关系，不消耗功率。
* 使用场景：交流电压变换、阻抗匹配、隔离电路、三相变压器等效分析。
* 与线性变压器的区别：
  * 理想变压器直接使用变比关系；
  * 线性变压器需要保留 \(L_1,L_2,M,R_1,R_2\) 等参数。
* 常见误区：
  * 电压比和电流比都写成同一方向的 \(n\)；
  * 忽略点端导致极性关系错误；
  * 把功率放大误解为理想变压器功能。

### 3.11 阻抗变换

* 概念解释：理想变压器可将一侧负载阻抗按变比平方折算到另一侧。
* 使用场景：最大功率传输、阻抗匹配、扬声器匹配、电源输出匹配。
* 与电源变换的区别：
  * 电源变换保持同一端口 \(u-i\) 关系；
  * 变压器阻抗变换是通过匝数比改变两侧电压电流比例。
* 常见误区：
  * 折算阻抗时少平方；
  * 将折算方向弄反；
  * 忽略负载是复阻抗时实部和虚部都要折算。

### 3.12 三相变压器

* 概念解释：三相变压器由三组单相变压器或三相铁心结构组成，用于三相电压等级变换和电能传输。
* 使用场景：配电系统、牵引供电、工业供电、相序和线电压变换。
* 与单相理想变压器的区别：
  * 需要同时处理相电压、线电压、相序和连接组别；
  * Y、Δ 连接会引入 \(\sqrt{3}\) 倍关系和相位移。
* 常见误区：
  * 只按匝数比处理，不考虑 Y/Δ 连接引入的线相量关系；
  * 忽略相序和相位偏移；
  * 把相电压和线电压混用。

## 4. 重要公式与适用条件

### 4.1 自感磁链关系

公式：

```math
\psi = N\Phi = Li
```

符号含义：

* \(\psi\)：磁链；
* \(N\)：线圈匝数；
* \(\Phi\)：穿过线圈的磁通；
* \(L\)：自感；
* \(i\)：线圈电流。

适用条件：

* 线圈处于线性磁化范围；
* 磁链与电流近似成正比。

常见误用：

* 非线性铁心饱和时仍使用固定 \(L\)；
* 忽略磁链方向与绕向有关。

### 4.2 自感电压

公式：

```math
u_L = L\frac{di}{dt}
```

相量形式：

```math
\dot U_L = j\omega L\dot I
```

符号含义：

* \(u_L\)：自感电压；
* \(L\)：自感；
* \(i\)：电感电流；
* \(\omega\)：角频率。

适用条件：

* 电压、电流采用关联参考方向；
* 正弦稳态下可用相量形式。

常见误用：

* 非关联参考方向下直接套正号；
* 把 \(j\omega L\) 写成 \(1/(j\omega L)\)。

### 4.3 互感磁链关系

公式：

```math
\psi_{12}=M_{12}i_2,\qquad \psi_{21}=M_{21}i_1
```

在线性互易磁路中常有：

```math
M_{12}=M_{21}=M
```

符号含义：

* \(\psi_{12}\)：电流 \(i_2\) 在线圈 1 中产生的互磁链；
* \(\psi_{21}\)：电流 \(i_1\) 在线圈 2 中产生的互磁链；
* \(M\)：互感系数。

适用条件：

* 两线圈之间存在磁耦合；
* 介质可近似线性，且互易关系成立。

常见误用：

* 将互感方向和大小混在一起；
* 忽略同名端导致符号错误。

### 4.4 互感电压

公式：

```math
u_{M1}=\pm M\frac{di_2}{dt},\qquad u_{M2}=\pm M\frac{di_1}{dt}
```

相量形式：

```math
\dot U_{M1}=\pm j\omega M\dot I_2,\qquad \dot U_{M2}=\pm j\omega M\dot I_1
```

符号含义：

* \(u_{M1}\)：线圈 2 电流在线圈 1 中引起的互感电压；
* \(u_{M2}\)：线圈 1 电流在线圈 2 中引起的互感电压；
* \(M\)：互感。

适用条件：

* 已明确电流参考方向和同名端；
* 互感项正负号由“电流进入同名端关系”和“电压参考极性”共同决定。

常见误用：

* 只看点端，不看电流方向；
* 只看电流方向，不看电压参考极性；
* 漏写互感电压。

### 4.5 两耦合电感的端口方程

公式：

```math
\begin{cases}
u_1=L_1\dfrac{di_1}{dt}\pm M\dfrac{di_2}{dt}\\[4pt]
u_2=L_2\dfrac{di_2}{dt}\pm M\dfrac{di_1}{dt}
\end{cases}
```

相量形式：

```math
\begin{cases}
\dot U_1=j\omega L_1\dot I_1\pm j\omega M\dot I_2\\
\dot U_2=j\omega L_2\dot I_2\pm j\omega M\dot I_1
\end{cases}
```

符号含义：

* \(L_1,L_2\)：两个线圈自感；
* \(M\)：互感；
* \(\dot I_1,\dot I_2\)：两个线圈电流相量。

适用条件：

* 线圈可视为线性耦合电感；
* 需要先确定同名端和参考方向。

常见误用：

* 把两个方程的互感项符号随意设为相同；
* 未按绕向重新判断正负号；
* 混淆时域和相量域形式。

### 4.6 串联耦合电感的等效电感

加强耦合：

```math
L_{eq}=L_1+L_2+2M
```

削弱耦合：

```math
L_{eq}=L_1+L_2-2M
```

符号含义：

* \(L_{eq}\)：串联后的等效电感；
* \(M\)：互感。

适用条件：

* 两线圈串联且电流相同；
* 加强或削弱由同名端和电流进入端判断。

常见误用：

* 同名端相连时直接判断为加强；
* 忽略互感项的正负；
* 对并联或非串联结构误用串联公式。

### 4.7 耦合系数

公式：

```math
k=\frac{M}{\sqrt{L_1L_2}}
```

符号含义：

* \(k\)：耦合系数；
* \(M\)：互感；
* \(L_1,L_2\)：自感。

适用条件：

* 两线圈磁耦合系统；
* 通常满足 \(0\le k\le 1\)。

常见误用：

* 得到 \(k>1\) 后不检查参数；
* 把 \(k\) 当成有单位量；
* 弱耦合时仍按理想变压器处理。

### 4.8 耦合电感储能

公式：

```math
w=\frac12L_1i_1^2+\frac12L_2i_2^2\pm Mi_1i_2
```

符号含义：

* \(w\)：磁场储能；
* \(L_1,L_2\)：自感；
* \(M\)：互感；
* \(i_1,i_2\)：两个线圈电流。

适用条件：

* 线性磁路；
* 符号由磁通加强或削弱决定。

常见误用：

* 漏掉交叉项；
* 互感项符号与电压方程不一致；
* 忽略储能必须非负所带来的 \(M^2\le L_1L_2\) 约束。

### 4.9 正弦稳态下耦合电感阻抗矩阵

公式：

```math
\begin{bmatrix}
\dot U_1\\
\dot U_2
\end{bmatrix}
=
\begin{bmatrix}
j\omega L_1 & \pm j\omega M\\
\pm j\omega M & j\omega L_2
\end{bmatrix}
\begin{bmatrix}
\dot I_1\\
\dot I_2
\end{bmatrix}
```

符号含义：

* 对角项：自感阻抗；
* 非对角项：互感阻抗；
* 正负号：由同名端和参考方向决定。

适用条件：

* 正弦稳态；
* 线性耦合电感。

常见误用：

* 把互感阻抗写成实数 \(\omega M\)；
* 非对角项漏掉 \(j\)；
* 两个非对角项符号前后不一致。

### 4.10 反映阻抗的典型表达

在两个耦合线圈中，副边总阻抗为 \(Z_2\) 时，原边可能看到的反映阻抗常写成：

```math
Z_r=\frac{(\omega M)^2}{Z_2}
```

符号含义：

* \(Z_r\)：折算到原边的反映阻抗；
* \(M\)：互感；
* \(Z_2\)：副边回路总阻抗。

适用条件：

* 典型线性耦合回路；
* 具体符号和等效形式需结合参考方向和方程形式确定。

常见误用：

* 把 \(Z_2\) 只取负载，不含副边自阻抗；
* 忘记平方 \((\omega M)^2\)；
* 折算方向错误。

> 注：反映阻抗表达式在不同教材符号约定下可能呈现为不同代数形式，使用时需结合原电路方程人工校验。

### 4.11 理想变压器电压比

公式：

```math
\frac{\dot U_1}{\dot U_2}=\pm\frac{N_1}{N_2}=\pm n
```

符号含义：

* \(N_1,N_2\)：原、副边匝数；
* \(n=N_1/N_2\)：变比；
* 正负号：由同名端和电压参考方向决定。

适用条件：

* 理想变压器；
* 全耦合、无损耗、励磁电流可忽略。

常见误用：

* 不看点端直接写正号；
* 把相电压、线电压与绕组电压混用；
* 变比方向写反。

### 4.12 理想变压器电流比

公式：

```math
\frac{\dot I_1}{\dot I_2}=\mp\frac{N_2}{N_1}=\mp\frac{1}{n}
```

符号含义：

* \(\dot I_1,\dot I_2\)：原、副边电流相量；
* \(n=N_1/N_2\)：变比。

适用条件：

* 理想变压器；
* 电流参考方向需与电压参考方向、点端约定保持一致。

常见误用：

* 电压比和电流比都写成 \(n\)；
* 忽略电流方向与功率守恒的关系；
* 把电流放大误认为功率放大。

### 4.13 理想变压器功率守恒

公式：

```math
\dot U_1\dot I_1^*+\dot U_2\dot I_2^*=0
```

或在无损条件下理解为：

```math
S_1+S_2=0
```

符号含义：

* \(S_1,S_2\)：两侧复功率；
* 星号表示共轭。

适用条件：

* 理想无损变压器；
* 两侧功率符号按吸收功率约定统计。

常见误用：

* 认为理想变压器可产生功率；
* 只检查电压比，不检查功率关系；
* 忽略参考方向导致功率符号不平衡。

### 4.14 理想变压器阻抗变换

公式：

```math
Z_{in}=n^2Z_L
```

其中：

```math
n=\frac{N_1}{N_2}
```

符号含义：

* \(Z_{in}\)：从原边看入的等效阻抗；
* \(Z_L\)：副边负载阻抗；
* \(n\)：原边对副边的匝数比。

适用条件：

* 理想变压器；
* 负载折算到原边；
* 若从副边看原边，应使用相反折算方向。

常见误用：

* 忘记平方；
* 把 \(n^2\) 写成 \(1/n^2\)；
* 只折算电阻，不折算电抗。

### 4.15 线性变压器最大功率匹配思想

公式形式可概括为：

```math
Z_L \text{ 经变压器折算后应与电源内阻抗满足共轭匹配条件}
```

对纯电阻情形可简化理解为：

```math
R_{in}=R_s
```

符号含义：

* \(Z_L\)：负载阻抗；
* \(R_s\)：等效电源内阻；
* \(R_{in}\)：经变压器折算后从源端看到的输入电阻。

适用条件：

* 目标是负载获得最大有功功率；
* 需要先明确变压器模型是理想还是线性实际模型。

常见误用：

* 直接让负载等于某个绕组电阻；
* 不折算阻抗就套最大功率传输；
* 忽略负载含电抗时的共轭匹配条件。

## 5. 典型题型

### 题型 1：同名端与互感符号判断

考查知识点：

* 同名端；
* 点标法；
* 互感电压正负号；
* 加强/削弱耦合。

解题关键：

1. 标出两个线圈的同名端；
2. 判断电流是进入点端还是离开点端；
3. 判断两个电流产生的磁通是加强还是削弱；
4. 根据电压参考极性确定互感项正负。

常见错误：

* 只看点端，不看电流方向；
* 只看电流方向，不看电压极性；
* 把同名端当成实际连接点。

对应错因标签：

* dot_convention_error
* mutual_voltage_polarity_error
* reference_direction_error
* dot_convention_error

### 题型 2：耦合电感端口方程列写

考查知识点：

* 自感电压；
* 互感电压；
* 时域方程；
* 相量方程。

解题关键：

1. 选定每个线圈电压和电流参考方向；
2. 写出自感项；
3. 根据同名端和电流方向写互感项；
4. 必要时转换为相量形式。

常见错误：

* 互感项漏写；
* 互感项符号错误；
* 时域形式和相量形式混用。

对应错因标签：

* mutual_inductance_definition_error
* mutual_voltage_polarity_error
* coupled_inductor_phasor_error

### 题型 3：串联耦合电感等效

考查知识点：

* 串联耦合；
* 加强耦合；
* 削弱耦合；
* 等效电感。

解题关键：

1. 判断两个线圈是否真正串联；
2. 判断同一电流进入同名端的情况；
3. 选择 \(L_1+L_2+2M\) 或 \(L_1+L_2-2M\)；
4. 检查等效电感是否物理合理。

常见错误：

* 加强与削弱判断反；
* 把并联结构当串联结构；
* 漏掉 \(2M\)。

对应错因标签：

* coupled_series_equivalent_error
* dot_convention_error
* mutual_inductance_definition_error

### 题型 4：耦合系数与参数可行性判断

考查知识点：

* 互感；
* 耦合系数；
* 参数范围；
* 储能非负性。

解题关键：

1. 用 \(k=M/\sqrt{L_1L_2}\) 计算耦合系数；
2. 检查是否满足 \(0\le k\le 1\)；
3. 若题中给出参数矛盾，应指出物理不可行；
4. 根据 \(k\) 判断可否近似为强耦合或理想变压器。

常见错误：

* 认为任意 \(M\) 都可取；
* \(k>1\) 后仍继续计算；
* 把 \(M\) 与 \(k\) 混淆。

对应错因标签：

* coupling_coefficient_error
* coupling_coefficient_error
* coupled_inductor_energy_error

### 题型 5：耦合电感储能计算

考查知识点：

* 自感储能；
* 互感储能项；
* 加强/削弱耦合；
* 初始能量。

解题关键：

1. 分别写出两个自感储能；
2. 判断互感交叉项符号；
3. 代入电流初值；
4. 检查能量是否非负。

常见错误：

* 漏掉互感能量项；
* 互感项符号写反；
* 用电压代入储能公式。

对应错因标签：

* coupled_inductor_energy_error
* energy_storage_error
* dot_convention_error

### 题型 6：含耦合电感电路的网孔法

考查知识点：

* 网孔电流；
* 互感阻抗；
* KVL；
* 相量法。

解题关键：

1. 将正弦稳态电路转化为相量模型；
2. 统一网孔电流方向；
3. 在 KVL 中加入 \(\pm j\omega M I\) 项；
4. 联立求解电流、电压或功率。

常见错误：

* 互感阻抗符号错误；

对应错因标签：

* mesh_analysis_error

* phasor_model_error

### 题型 7：去耦等效电路分析

考查知识点：

解题关键：

1. 判断是否满足可去耦的结构条件；

常见错误：

对应错因标签：

## 6. 常见学习困难

* 同名端理解困难：学生容易把点端看成普通端点，不理解其表示的是磁耦合极性。
* 互感电压符号混乱：互感项的正负同时受电流方向、电压极性和同名端影响。
* 加强耦合与削弱耦合判断错误：串联耦合电感等效时容易把 \(+2M\) 和 \(-2M\) 混淆。
* 互感项遗漏：列 KVL 时只写 \(j\omega L I\)，漏掉 \(j\omega M I\)。
* 时域与相量域混淆：把 \(Mdi/dt\) 与 \(j\omega MI\) 混写。
* 耦合系数物理意义不清：不检查 \(k\le 1\)，导致不合理参数仍被使用。
* 储能公式错误：漏掉互感交叉项或交叉项符号错误。
* 去耦等效条件不清：不判断共端结构就套去耦公式。
* 反映阻抗折算错误：折算方向、变比平方、复阻抗整体折算容易出错。
* 理想变压器条件不清：把实际变压器、线性变压器和理想变压器混用。
* 电压比和电流比混淆：电压按匝数比变换，电流按反比变换，并且要注意参考方向。
* 三相变压器线量相量关系复杂：相电压、线电压、相位移和连接方式容易混淆。
* 工程应用建模过度理想化：忽略漏磁、损耗、饱和、频率、功率等级和安全隔离要求。

## 7. 错题诊断标签
### dot_convention_error

中文名称：同名端判断错误  
适用场景：互感电压符号、理想变压器极性、串联耦合判断  
典型表现：点端极性与电流方向没有对应，或把同名端当成实际连接点  
诊断提示：让学生分别标出电流进入端和电压正端，再根据点端判断互感项符号

### mutual_voltage_polarity_error

中文名称：互感电压符号错误  
适用场景：耦合电感端口方程、网孔法、相量模型  
典型表现：\(\pm Mdi/dt\) 或 \(\pm j\omega M I\) 的符号写反  
诊断提示：检查同名端、电流进入端和电压参考极性是否同时考虑

### mutual_inductance_definition_error

中文名称：互感项遗漏  
适用场景：含耦合电感的 KVL、网孔法、相量分析  
典型表现：只写自感项，漏写互感产生的电压项  
诊断提示：看到耦合电感符号或同名端标记时，应检查每个线圈是否受到另一个线圈电流影响

### coupled_inductor_kvl_error

中文名称：耦合电感方程建立错误  
适用场景：两线圈或多线圈耦合电感建模  
典型表现：自感项、互感项、参考方向或矩阵位置错误  
诊断提示：先写磁链关系，再对时间求导，最后代入 KVL

### coupled_inductor_phasor_error

中文名称：时域与相量域混淆  
适用场景：正弦稳态含互感电路  
典型表现：把 \(Mdi/dt\) 与 \(j\omega MI\) 混用，或同一方程同时出现时域量和相量  
诊断提示：先确定题目是在时域暂态还是正弦稳态，再统一表达形式

### coupling_coefficient_error

中文名称：耦合系数错误  
适用场景：互感参数判断、理想变压器近似、无线能量传输  
典型表现：\(k\) 计算错误，或出现 \(k>1\) 未检查  
诊断提示：使用 \(k=M/\sqrt{L_1L_2}\)，并检查 \(0\le k\le 1\)

### coupled_inductor_energy_error

中文名称：互感储能项遗漏  
适用场景：耦合电感储能计算  
典型表现：只写两个自感储能，漏掉 \(\pm Mi_1i_2\)  
诊断提示：含耦合电感的能量不仅有自能项，还有互能项

### decoupling_equivalent_error

中文名称：去耦等效错误  
适用场景：含耦合电感电路化简  
典型表现：不满足去耦条件仍套公式，或去耦后端口关系改变  
诊断提示：先确认耦合线圈的连接结构，再按同名端确定等效电感

### reflected_impedance_error

中文名称：反映阻抗错误  
适用场景：线性变压器输入阻抗、负载折算  
典型表现：反映阻抗公式写错，或只折算负载而漏掉副边回路阻抗  
诊断提示：从两侧 KVL 联立消元推导，而不是直接搬移阻抗

### linear_transformer_model_error

中文名称：理想变压器误用  
适用场景：实际变压器、线性变压器、弱耦合电路分析  
典型表现：存在绕组电阻、漏感、不完全耦合时直接使用理想变压器关系  
诊断提示：先判断是否满足无损、全耦合、励磁电流可忽略等条件

### ideal_transformer_ratio_error

中文名称：理想变压器变比错误  
适用场景：电压比、电流比、匝数比计算  
典型表现：电压比和电流比方向写反，或把两者都写成 \(n\)  
诊断提示：电压按匝数比，电流按反比，并用功率守恒检查

### ideal_transformer_current_direction_error

中文名称：变压器电流方向错误  
适用场景：理想变压器电流比、功率平衡  
典型表现：电流方向与电压参考方向不匹配，功率守恒不成立  
诊断提示：用复功率守恒检查电流方向是否正确

### ideal_transformer_impedance_transform_error

中文名称：变压器阻抗变换错误  
适用场景：理想变压器负载折算、输入阻抗  
典型表现：阻抗折算少平方或方向反  
诊断提示：阻抗折算按匝数比平方，先明确折算到哪一侧

### transformer_power_relation_error

中文名称：变压器阻抗匹配错误  
适用场景：最大功率传输、负载匹配  
典型表现：没有通过变比折算就直接匹配负载和源内阻  
诊断提示：先把负载折算到源侧，再应用最大功率传输条件

### three_phase_transformer_connection_error

中文名称：三相变压器连接错误  
适用场景：Y/Δ 变压器、三相变比、相位移  
典型表现：原副边连接方式判断错误，导致线电压相量关系错误  
诊断提示：先画出每相绕组电压，再转换为线电压

### phase_line_quantity_confusion

中文名称：线电压与相电压混淆  
适用场景：三相变压器和三相电路  
典型表现：把绕组电压直接当线电压，或漏掉 \(\sqrt3\) 关系  
诊断提示：Y 接时线电压与相电压相差 \(\sqrt3\) 和相位角；Δ 接时绕组电压等于线电压

## 8. Agent 问答样例

### Q1：为什么两个线圈没有导线相连，也会互相影响？

所属知识点：磁耦合  
可能错因：mutual_inductance_definition_error  
回答：  
因为一个线圈中的电流变化会产生变化的磁通，如果这部分磁通穿过另一个线圈，就会在另一个线圈中感应出电压。这种影响不是通过导线连接完成的，而是通过磁场耦合完成的。

### Q2：同名端到底是什么意思？

所属知识点：同名端  
可能错因：dot_convention_error  
回答：  
同名端表示两个线圈绕向和磁通方向之间的相对关系。如果两个电流同时从各自同名端流入，它们产生的磁通相互加强；如果一个从同名端流入、另一个从异名端流入，则磁通相互削弱。同名端不是实际短接点，只是极性标记。

### Q3：互感电压的正负号怎么判断？

所属知识点：互感电压  
可能错因：mutual_voltage_polarity_error  
回答：  
先看引起互感电压的那个电流是进入点端还是离开点端，再看被感应线圈电压参考正端是否在点端。两者方向关系一致时取正，不一致时取负。不能只看到点端就机械加正号。

### Q4：串联耦合电感什么时候是 \(L_1+L_2+2M\)？

所属知识点：串联耦合电感  
可能错因：dot_convention_error  
回答：  
当串联电流使两个线圈产生的磁通相互加强时，等效电感为 \(L_1+L_2+2M\)。若磁通相互削弱，则为 \(L_1+L_2-2M\)。判断时要看电流是否同时进入两个线圈的同名端，或同时离开两个同名端。

### Q5：为什么耦合系数不能大于 1？

所属知识点：耦合系数  
可能错因：coupling_coefficient_error  
回答：  
耦合系数 \(k=M/\sqrt{L_1L_2}\) 表示两个线圈磁耦合的紧密程度。实际中一个线圈产生的磁通不可能全部“超过完全耦合”地穿过另一个线圈，所以通常有 \(0\le k\le 1\)。如果算出 \(k>1\)，说明参数、单位或计算过程有问题。

### Q6：含耦合电感的网孔法和普通网孔法有什么不同？

所属知识点：含耦合电感网孔法  
可能错因：coupled_inductor_mesh_error  
回答：  
普通网孔法只需要写本回路元件的电压降；含耦合电感时，还要在相关线圈电压中加入由另一个网孔电流产生的互感电压项，即 \(\pm j\omega M I\)。符号要由同名端和电流方向判断。

### Q7：去耦等效是不是把互感删掉？

所属知识点：去耦等效  
可能错因：decoupling_equivalent_error  
回答：  
不是。去耦等效是把含互感的电路转化为没有显式互感的等效电感网络，端口电压电流关系仍要保持一致。互感的影响会进入等效支路中的 \(L\pm M\) 等参数，而不是被简单删除。

### Q8：理想变压器为什么电压变大时电流会变小？

所属知识点：理想变压器  
可能错因：ideal_transformer_ratio_error  
回答：  
理想变压器不消耗功率，也不产生功率。若副边电压因匝数比升高，在负载吸收功率一定的情况下，副边电流会相应变小。电压按匝数比变化，电流按反比变化，这是功率守恒的结果。

### Q9：阻抗通过变压器折算为什么要平方？

所属知识点：阻抗变换  
可能错因：ideal_transformer_impedance_transform_error  
回答：  
阻抗是电压与电流的比值。理想变压器中，电压按匝数比 \(n\) 变化，而电流按 \(1/n\) 变化，所以电压与电流之比会按 \(n^2\) 变化。因此折算阻抗时出现变比平方。

### Q10：什么时候不能把实际变压器看成理想变压器？

所属知识点：实际变压器与理想变压器  
可能错因：linear_transformer_model_error  
回答：  
当绕组电阻、漏磁、铁心损耗、磁饱和、励磁电流或频率特性不能忽略时，就不能直接使用理想变压器模型。低损耗、强耦合、工作在额定频率和线性磁化区时，实际变压器才可近似为理想变压器。

### Q11：三相变压器为什么不能只按匝数比算线电压？

所属知识点：三相变压器  
可能错因：phase_line_quantity_confusion  
回答：  
三相变压器的绕组电压是相电压，而外部常用的是线电压。Y 接和 Δ 接的线电压与相电压关系不同，还可能引入 \(30^\circ\) 等相位移。因此三相变压器要同时考虑匝数比、连接方式、相序和线相量关系。

### Q12：无线充电为什么和磁耦合有关？

所属知识点：工程应用  
可能错因：linear_transformer_model_error  
回答：  
无线充电中，发射线圈产生交变磁场，接收线圈通过磁耦合获得感应电压和能量。由于两个线圈之间存在气隙，耦合系数通常小于 1，因此不能简单看成理想变压器，常需要考虑漏磁、频率、补偿网络和效率。

## 9. 学习建议

### 9.1 概念薄弱型

适用表现：  
不理解磁耦合、互感、同名端、理想变压器之间的关系。

学习建议：

1. 先用“电流变化 → 磁通变化 → 另一线圈感应电压”的因果链理解互感；
2. 把同名端理解为“磁通方向关系标记”，不要理解成实际接线点；
3. 对比普通电感、耦合电感、线性变压器和理想变压器的模型差异。

### 9.2 符号方向错误型

适用表现：  
互感项正负号、理想变压器电压极性、电流方向经常写错。

学习建议：

1. 每道题先标出电流参考方向和电压参考极性；
2. 再标同名端，判断电流是否进入点端；
3. 不要直接背“点端取正”，而要结合电压参考方向判断；
4. 用功率守恒或储能非负性检查结果。

### 9.3 公式误用型

适用表现：  
会背变比、阻抗折算、串联耦合公式，但经常用错条件。

学习建议：

1. 使用 \(L_1+L_2\pm 2M\) 前先确认是串联耦合；
2. 使用 \(Z_{in}=n^2Z_L\) 前先确认是理想变压器，并明确折算方向；
3. 使用反映阻抗公式前先写两侧 KVL，避免漏掉副边回路阻抗；
4. 计算耦合系数后检查是否满足 \(0\le k\le 1\)。

### 9.4 解题流程混乱型

适用表现：  
遇到含多个耦合线圈、变压器、三相连接时不知道先做什么。

学习建议：

1. 先判断使用哪种模型：耦合电感、线性变压器还是理想变压器；
2. 若是耦合电感，优先写端口方程或网孔方程；
3. 若是理想变压器，先写电压比、电流比和阻抗折算；
4. 若是三相变压器，先每相分析，再转化为线量。

### 9.5 综合应用困难型

适用表现：  
会做单个公式题，但不会分析工程场景中的变压器、无线充电或测量电路。

学习建议：

1. 工程题先判断模型理想化程度；
2. 关注频率、功率、效率、漏磁、绝缘和安全约束；
3. 对无线充电、电流互感器、电压互感器等应用，先明确输入量、输出量和耦合路径；
4. 最后用功率关系和单位量级检查结果是否合理。

## 10. 与其他章节的关系

* 与第7章电容、电感及动态电路的关系：本章扩展了单个电感的概念，引入由多个线圈共同决定的磁链和储能。
* 与第8章一阶电路暂态分析的关系：含耦合电感电路在时域中可能出现联立微分方程，初始电流和储能仍需遵守电感连续性思想。
* 与第9章二阶电路暂态分析的关系：多个耦合电感和电容组合可能形成高阶暂态系统，互感会改变自然频率和响应形式。
* 与第10章正弦稳态分析的关系：本章大量使用相量法，将互感电压表示为 \(\pm j\omega M I\)。
* 与第11章正弦稳态功率的关系：变压器分析中需要使用有功功率、无功功率、复功率和最大功率传输思想。
* 与第12章三相正弦稳态电路的关系：三相变压器需要结合三相相量、Y/Δ 连接、相电压/线电压关系和相序分析。
* 与模拟电子技术的关系：变压器耦合、隔离、阻抗匹配在音频放大、隔离电源、信号耦合中常见。
* 与通信原理和高频电子线路的关系：互感耦合、谐振耦合、阻抗匹配和无线能量传输是高频系统的重要基础。
* 与电力电子和电力系统的关系：变压器、电流互感器、电压互感器、隔离变换和三相配电是工程应用核心内容。
