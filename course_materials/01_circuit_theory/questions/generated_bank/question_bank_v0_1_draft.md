## 第8章 一阶电路的暂态分析

### QCT-08-001 RC放电时间常数计算

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T02（RC零输入响应）
- 难度：基础
- 主要知识点：
  - RC零输入响应
  - 时间常数定义
  - 初始电压确定
- 关联错因标签：
  - time_constant_error
  - initial_value_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

一个已充电的电容 C = 10 μF 通过电阻 R = 5 kΩ 放电。已知换路前电容电压 u_C(0⁻) = 12 V。求：
（1）时间常数 τ；
（2）写出 t ≥ 0 时电容电压 u_C(t) 的表达式；
（3）求 t = τ 时刻的电容电压值。

#### 标准答案

（1）τ = RC = 5 × 10³ × 10 × 10⁻⁶ = 0.05 s = 50 ms
（2）u_C(t) = 12 e^{−t/0.05} V = 12 e^{−20t} V（t ≥ 0）
（3）u_C(τ) = 12 e^{−1} ≈ 12 × 0.3679 ≈ 4.41 V

#### 解题思路

1. 确认零输入响应条件：外部独立源置零，仅由初始储能引起响应；
2. 从电容两端看入求等效电阻 R = 5 kΩ；
3. 时间常数 τ = RC；
4. 一阶 RC 零输入响应形式为 u_C(t) = u_C(0⁺)e^{−t/τ}；
5. 电容电压连续：u_C(0⁺) = u_C(0⁻) = 12 V。

#### 易错点

- 时间常数公式写成 τ = R/C 或 τ = 1/(RC)；
- 只记得 τ = RC 但未从储能元件端看入求等效电阻（此题简单，但复杂电路容易错）；
- 把 u_C(t) 表达式中的指数符号写成正指数。

#### 可用于诊断的学生错误表现

- 若写成 τ = 5 × 10³ × 10 × 10⁻⁶ = 0.05 但单位写为 s⁻¹，可判断为 time_constant_error（时间常数物理意义不清晰）；
- 若写成 u_C(t) = 12 − 12e^{−t/τ}，可判断为 zero_input_response_confusion（零输入与零状态响应混淆）。

#### 推荐反馈语

你这里把零输入响应和阶跃响应的形式搞混了。零输入响应是储能元件放电，形式为 u_C(t) = u_C(0⁺)e^{−t/τ}，终值为零。建议先确认换路后外部电源是否被置零。

### QCT-08-002 RL电路零输入响应

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T03（RL零输入响应）
- 难度：基础
- 主要知识点：
  - RL零输入响应
  - 时间常数
  - 初始电流确定
- 关联错因标签：
  - time_constant_error
  - inductor_current_continuity_error
  - initial_value_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

RL 电路中，L = 2 H，R = 8 Ω。换路前电感电流 i_L(0⁻) = 3 A。换路后电源被断开，R 与 L 形成闭合回路。求：
（1）时间常数 τ；
（2）t ≥ 0 时的电感电流 i_L(t)；
（3）t ≥ 0 时的电阻电压 u_R(t)。

#### 标准答案

（1）τ = L/R = 2/8 = 0.25 s
（2）i_L(t) = 3 e^{−t/0.25} = 3 e^{−4t} A（t ≥ 0）
（3）u_R(t) = R · i_L(t) = 8 × 3 e^{−4t} = 24 e^{−4t} V（t ≥ 0），参考方向关联

#### 解题思路

1. 换路后仅 R-L 回路，独立源为零，属于零输入响应；
2. 电感电流连续：i_L(0⁺) = i_L(0⁻) = 3 A；
3. 时间常数 τ = L/R；
4. RL 零输入响应形式：i_L(t) = i_L(0⁺)e^{−t/τ}。

#### 易错点

- 把 RL 时间常数写成 τ = RL 而不是 L/R；
- 忘记电感电流连续，用换路后稳态重新求初始值；
- 求电阻电压时参考方向未与电流方向关联。

#### 可用于诊断的学生错误表现

- 若 τ = 2 × 8 = 16 s，可判断为 time_constant_error（RC与RL公式混淆）；
- 若 i_L(0⁺) 直接取为 0，可判断为 inductor_current_continuity_error；
- 若 u_R(t) = 24 V 常数，可判断为零输入响应类型误判。

#### 推荐反馈语

RL一阶电路的时间常数是 L/R，不是 R×L。建议从储能元件（L）两端看入求等效电阻，再代入 τ = L/R。另外电感电流在换路瞬间不能突变，请先用换路前电路求 i_L(0⁻)。

### QCT-08-003 RC阶跃响应三要素法

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T04（RC阶跃响应）
- 难度：标准
- 主要知识点：
  - RC零状态响应
  - 三要素法
  - 终值和初值
- 关联错因标签：
  - three_element_method_error
  - final_value_error
  - time_constant_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

RC 串联电路中，R = 2 kΩ，C = 50 μF，初始时电容未充电（u_C(0⁻) = 0）。t = 0 时刻开关闭合，连接一个 10 V 直流电压源。求 t ≥ 0 时的 u_C(t) 和 i_C(t)。

#### 标准答案

初始值：u_C(0⁺) = u_C(0⁻) = 0 V
终值：u_C(∞) = 10 V
时间常数：τ = RC = 2 × 10³ × 50 × 10⁻⁶ = 0.1 s
u_C(t) = 10(1 − e^{−10t}) V（t ≥ 0）
i_C(t) = C · du_C/dt = (10/R) e^{−t/τ} = 5 e^{−10t} mA（t ≥ 0）

#### 解题思路

1. 零状态：u_C(0⁺) = 0；
2. 稳态后电容开路，u_C(∞) = 10 V（分压于 C两端）；
3. τ = RC；
4. 三要素公式：u_C(t) = u_C(∞) + [u_C(0⁺) − u_C(∞)] e^{−t/τ}；
5. 电容电流由电压求导得到。

#### 易错点

- 终值只考虑电阻分压而忘记电容开路后两端电压等于电源电压；
- 求电容电流时导数计算错误或漏掉绝对值；
- 逻辑混乱：对零状态响应仍写成 e^{−t/τ} 衰减形式不加终值。

#### 可用于诊断的学生错误表现

- 若写成 u_C(t) = 10 e^{−10t}，可判断为 zero_state_response_confusion（把零状态响应写成零输入形式）；
- 若 τ = 2 × 10³ × 50 × 10⁻⁶ = 0.1 但 u_C(∞) 取为 0，可判断为 final_value_error。

#### 推荐反馈语

你写的 u_C(t) 看起来更像是零输入响应的形式。阶跃响应中电容从 0 V 充电到电源电压，正确形式应包含 (1 − e^{−t/τ}) 因子。请先用三要素法明确写出初始值、终值和时间常数。

### QCT-08-004 一阶RL阶跃响应

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T05（RL阶跃响应）
- 难度：标准
- 主要知识点：
  - RL零状态响应
  - 时间常数计算
  - 电感电流与电压关系
- 关联错因标签：
  - three_element_method_error
  - final_value_error
  - inductor_current_continuity_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

RL 串联电路中，R = 4 Ω，L = 2 H。t = 0 时开关闭合接入 12 V 直流源。初始 i_L(0⁻) = 0。
求 t ≥ 0 时的 i_L(t) 和 u_L(t)。

#### 标准答案

初始值：i_L(0⁺) = i_L(0⁻) = 0
终值：i_L(∞) = 12/4 = 3 A
τ = L/R = 2/4 = 0.5 s
i_L(t) = 3(1 − e^{−2t}) A（t ≥ 0）
u_L(t) = L · di_L/dt = 2 × 3 × 2 e^{−2t} = 12 e^{−2t} V（t ≥ 0）

#### 解题思路

1. 零状态响应，确定初始条件；
2. 终值时电感相当于短路；
3. 代入三要素公式。

#### 易错点

- 求 i_L(∞) 时忘记稳态后电感短路导致 R 两端电压就是电源电压；
- 对 u_L(t) 直接应用三要素公式而非求导。

#### 可用于诊断的学生错误表现

- 若 i_L(∞) = 12/(4+2)=2 A（将电阻、电感误作串联分压），可判断为 final_value_error；
- 若 u_L(t) = 12(1 − e^{−2t})，可判断为电感电流和电压关系错误。

#### 推荐反馈语

稳态时电感相当于短路，不是普通电阻。因此稳态电流 i_L(∞) = 12/4 = 3 A。电压 u_L(t) 不能直接用三要素法套，要通过电流求导得到。

### QCT-08-005 含受控源的一阶电路时间常数

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T09（含受控源一阶电路）
- 难度：综合
- 主要知识点：
  - 含受控源的等效电阻
  - 时间常数计算
  - 独立源置零规则
- 关联错因标签：
  - time_constant_error
  - dependent_source_zeroed_error
  - test_source_direction_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

电路含一个受控电流源（控制量为 R₁ 两端电压 u_R1），C = 0.1 F，R₁ = 10 Ω，R₂ = 5 Ω，受控源 i = 2 u_R1。开关在 t = 0 时动作后，求 τ。

#### 标准答案

求时间常数时，独立源置零，受控源保留。断开 C，从 C 端口看入：
设端口电压为 u，端口电流为 i（方向从正端流入）。
KCL：i = u/R₂ + i_受控 = u/R₂ + 2 u_R1
其中 u_R1 = u（因 R₁ 与端口并联）
→ i = u/5 + 2u = 0.2u + 2u = 2.2u
→ R_eq = u/i = 1/2.2 ≈ 0.4545 Ω
τ = R_eq · C ≈ 0.04545 s

#### 解题思路

1. 求时间常数需从储能元件端口看入求等效电阻；
2. 独立源置零（电压源短路、电流源开路），但受控源保留；
3. 用端口加测试源法求 R_eq。

#### 易错点

- 把受控源也置零（相当于删除），导致等效电阻错误；
- 直接从固定电阻并联求等效，忽略受控源作用；
- 端口测试源方向与 KCL/KVL 不一致。

#### 可用于诊断的学生错误表现

- 若 R_eq = R₁ ∥ R₂ = 10∥5 ≈ 3.33 Ω，说明误将受控源置零，可判断为 dependent_source_zeroed_error；
- 若计算正确但公式写成 τ = L/R 类型，可判断为 time_constant_error。

#### 推荐反馈语

含受控源的一阶电路求时间常数时，受控源必须保留（它代表电路中有源特性）。独立源置零后，从电容端看进去加测试源列方程。你目前只算了无源电阻的并联，漏掉了受控源提供的额外路径。

### QCT-08-006 分段换路响应

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T08（分段换路响应）
- 难度：综合
- 主要知识点：
  - 多次换路
  - 分段初值
  - 时间平移
- 关联错因标签：
  - piecewise_switching_error
  - time_origin_shift_error
  - dynamic_initial_condition_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

RC 电路中 R = 10 kΩ，C = 20 μF，u_C(0⁻) = 0。t = 0 时开关接到 +10V 直流源；t = 2 ms 时开关突然接到 0 V（接地）。求 t ≥ 0 全程的 u_C(t) 表达式。

#### 标准答案

第一阶段（0 ≤ t < 2 ms）：
  τ₁ = RC = 10×10³×20×10⁻⁶ = 0.2 s
  u_C(0⁺) = 0，u_C(∞) = 10 V
  u_C(t) = 10(1 − e^{−t/0.2}) V
  在 t = 2 ms 时：u_C(2 ms) = 10(1 − e^{−0.002/0.2}) = 10(1 − e^{−0.01}) ≈ 10(1 − 0.99005) = 0.0995 V

第二阶段（t ≥ 2 ms）：
  重新设时间原点 τ' = t − 2 ms
  u_C(τ'=0⁺) = 0.0995 V
  终值 u_C(∞) = 0 V
  τ₂ = 0.2 s（电阻、电容不变）
  u_C(τ') = 0.0995 e^{−τ'/0.2} V（τ' ≥ 0）
  即 u_C(t) = 0.0995 e^{−(t−0.002)/0.2} V（t ≥ 2 ms）

#### 解题思路

1. 分段处理：每段各自确定初值和终值；
2. 第一段末值作为第二段初值；
3. 第二段的时间原点重新设在该段换路时刻。

#### 易错点

- 第二段重新从 0 V 开始而不用第一段末值；
- 时间平移时没有重新设 t = 0，仍用绝对时间；
- 第一段充电时间极短（2 ms << 0.2 s），但仍有微小电压，不能忽略。

#### 可用于诊断的学生错误表现

- 若第二段 u_C 从 0 V 开始衰减，可判断为 piecewise_switching_error；
- 若第二段写成 u_C(t) = 0.0995 e^{−t/0.2}（t ≥ 0 而不是 t ≥ 2 ms），可判断为 time_origin_shift_error。

#### 推荐反馈语

分段换路时，下一段的初值来自上一段的末值，不是回到原始初始条件。另外时间原点要重新设在换路时刻，指数项也应写成 e^{−(t−t₀)/τ}。

### QCT-08-007 一阶电路微分方程法

- 所属章节：第8章 一阶电路的暂态分析
- 对应题型：CT08-T06（三要素法）
- 难度：标准
- 主要知识点：
  - 一阶微分方程的建立与求解
  - 三要素法与方程法对照
- 关联错因标签：
  - three_element_method_error
  - dynamic_equation_setup_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

图中有 R = 3 Ω，C = 2 F，串联后与 6 V 直流源构成回路。初始 u_C(0⁻) = 0。试建立 u_C(t) 的微分方程并求解 t ≥ 0 时的 u_C(t)。

#### 标准答案

KVL：RC·du_C/dt + u_C = 6
即 3×2·du_C/dt + u_C = 6 → 6 du_C/dt + u_C = 6
→ du_C/dt + (1/6)u_C = 1
齐次解：u_Ch = K e^{−t/6}
特解：u_Cp = 6（稳态解）
全解：u_C(t) = 6 + K e^{−t/6}
代入 u_C(0⁺) = 0 → 0 = 6 + K → K = −6
∴ u_C(t) = 6(1 − e^{−t/6}) V

三要素法直接验证：τ = RC = 6 s，u_C(0⁺)=0，u_C(∞)=6

#### 解题思路

1. 由 KVL 列微分方程；
2. 齐次解 + 特解 → 全解；
3. 用初始条件确定积分常数；
4. 对比三要素法结果，理解两种方法的一致性。

#### 易错点

- 微分方程列写时电容伏安关系 i_C = C du_C/dt 代入 KVL 后漏掉元件数；
- 特解直接假设为常数 C 但求解时忘记代入方程。

#### 可用于诊断的学生错误表现

- 若将 KVL 写成 3·i_C + u_C = 6 但 i_C 没有用 C du_C/dt 替代，可判断为 dynamic_equation_setup_error；
- 若特解设为 0，可判断为零输入响应与阶跃响应混淆。

#### 推荐反馈语

列微分方程时，电容的伏安关系要用 i_C = C du_C/dt 替换，不能把电容当成开路或短路来列方程。特解对应稳态解，对于直流源是常数。

## 第9章 二阶电路的暂态分析

### QCT-09-001 RLC串联零输入响应——阻尼类型判断

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T02（RLC串联零输入响应）
- 难度：标准
- 主要知识点：
  - 特征方程与特征根
  - 阻尼类型判断
  - 零输入响应形式
- 关联错因标签：
  - characteristic_root_error
  - damping_case_judgment_error
  - second_order_equation_setup_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 串联电路中，R = 4 Ω，L = 1 H，C = 1/4 F。初始条件：u_C(0⁻)=10 V，i_L(0⁻)=0。求特征根，判断阻尼类型，写出零输入响应 u_C(t) 的表达式形式。

#### 标准答案

RLC 串联零输入响应方程：d²u_C/dt² + (R/L) du_C/dt + (1/LC) u_C = 0
代入：d²u_C/dt² + 4 du_C/dt + 4 u_C = 0
特征方程：s² + 4s + 4 = 0 → (s + 2)² = 0 → s₁ = s₂ = −2（重根）
阻尼系数 α = R/(2L) = 4/(2×1) = 2，固有角频率 ω₀ = 1/√(LC) = 1/√(1×0.25) = 2
α = ω₀ → 临界阻尼
响应形式：u_C(t) = (A₁ + A₂t)e^{−2t} V

#### 解题思路

1. 列 RLC 串联特征方程；
2. 求特征根，判断是过阻尼（两相异实根）、临界阻尼（重根）还是欠阻尼（共轭复根）；
3. 根据阻尼类型写出响应基函数形式。

#### 易错点

- 特征方程系数写错（R/L 与 1/LC 顺序颠倒或符号错误）；
- 阻尼系数 α = R/(2L) 与固有频率 ω₀ 混淆；
- 临界阻尼响应写成 (A₁ + A₂) e^{st} 或漏掉 t 因子。

#### 可用于诊断的学生错误表现

- 若特征方程写成 s² + 4s + 1 = 0（忘记除以 LC），可判断为 second_order_equation_setup_error；
- 若响应写成 u_C = A₁ e^{s₁t} + A₂ e^{s₂t}（两个独立指数项），可判断为 damping_case_judgment_error（应使用临界阻尼形式）。

#### 推荐反馈语

你的特征方程系数和响应形式与阻尼类型不匹配。该电路是临界阻尼（重根），响应形式是 (A₁ + A₂t)e^{−αt}，不是两个独立指数项的和。请重新检查 α 与 ω₀ 的比较。

### QCT-09-002 RLC串联过阻尼零输入响应

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T04（过阻尼响应）
- 难度：标准
- 主要知识点：
  - 过阻尼特征根
  - 两个初值定系数
  - 零输入响应
- 关联错因标签：
  - damping_case_judgment_error
  - second_order_initial_condition_error
  - initial_derivative_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 串联电路中 R = 6 Ω，L = 1 H，C = 1 F。初始 u_C(0⁻)=5 V，i_L(0⁻)=0。求 t ≥ 0 时的 u_C(t)。

#### 标准答案

特征方程：s² + (R/L)s + 1/(LC) = s² + 6s + 1 = 0
s₁,₂ = −3 ± √(9 − 1) = −3 ± √8 ≈ −3 ± 2.828
s₁ ≈ −0.172，s₂ ≈ −5.828（两相异实根 → 过阻尼）
u_C(t) = A₁ e^{s₁t} + A₂ e^{s₂t}

初始条件：
u_C(0⁺) = u_C(0⁻) = 5 → A₁ + A₂ = 5
du_C/dt(0⁺) = i_L(0⁺)/C = 0/1 = 0
du_C/dt = s₁A₁ e^{s₁t} + s₂A₂ e^{s₂t} → 0 = s₁A₁ + s₂A₂

解得：A₁ = 5·s₂/(s₂−s₁) ≈ 5·(−5.828)/(−5.828+0.172) ≈ 5.15
A₂ = 5 − A₁ ≈ −0.15
u_C(t) ≈ 5.15 e^{−0.172t} − 0.15 e^{−5.828t} V

#### 解题思路

1. 建立特征方程，求特征根；
2. 判断阻尼类型（两相异实根→过阻尼）；
3. 确定两个初始条件：u_C(0⁺) 和 du_C/dt(0⁺)；
4. 用初值解待定系数。

#### 易错点

- 过阻尼响应只写了一个指数项；
- 导数初值 du_C/dt(0⁺) 没有通过 i_L(0⁺)/C 得到，而是直接假设为 0 没有依据；
- 特征根符号写错（正根与负根颠倒）。

#### 可用于诊断的学生错误表现

- 若只写 A₁ e^{s₁t}（漏掉一个指数项），可判断为 damping_case_judgment_error；
- 若只用 u_C(0⁺)=5 一个条件解两个系数，可判断为 second_order_initial_condition_error。

#### 推荐反馈语

过阻尼响应由两个指数项组成，不能只写一个。另外二阶电路需要两个初始条件：状态初值和导数初值。导数初值 du_C/dt(0⁺) = i_L(0⁺)/C，不能凭空假设。

### QCT-09-003 RLC串联欠阻尼响应

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T06（欠阻尼响应）
- 难度：综合
- 主要知识点：
  - 欠阻尼特征根
  - 衰减振荡
  - 阻尼振荡角频率
- 关联错因标签：
  - damping_case_judgment_error
  - natural_frequency_error
  - characteristic_root_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 串联电路中，R = 2 Ω，L = 1 H，C = 1 F。初始 u_C(0⁻)=10 V，i_L(0⁻)=0。求特征根并判断阻尼类型。

#### 标准答案

α = R/(2L) = 2/(2×1) = 1
ω₀ = 1/√(LC) = 1/√(1×1) = 1
特征方程：s² + 2αs + ω₀² = s² + 2s + 1 = 0
判别式：Δ = 4 − 4 = 0 → 临界阻尼（此处 R=2 使 Δ=0）

更正说明：如果 R = 1 Ω，则 α = 0.5，ω₀ = 1，α < ω₀ → 欠阻尼
特征根：s₁,₂ = −α ± j√(ω₀² − α²) = −0.5 ± j√(1 − 0.25) = −0.5 ± j0.866
响应形式：u_C(t) = e^{−0.5t}(A₁ cos 0.866t + A₂ sin 0.866t)

#### 解题思路

1. 计算 α 和 ω₀；
2. 比较 α 与 ω₀ 判断阻尼类型；
3. 欠阻尼时写共轭复根和衰减振荡响应形式。

#### 易错点

- 把 ω₀ 直接当成欠阻尼实际振荡角频率，没考虑阻尼衰减修正（ω_d = √(ω₀² − α²)）；
- 欠阻尼响应漏写指数衰减因子 e^{−αt}；
- 特征根虚部符号写错。

#### 可用于诊断的学生错误表现

- 若写成 u_C(t) = A₁ cos ω₀t + A₂ sin ω₀t（漏指数衰减），可判断为 natural_frequency_error（把固有频率当阻尼频率）；
- 若 s = −0.5 ± j1（未计算 ω_d），可判断为 damping_case_judgment_error。

#### 推荐反馈语

欠阻尼响应中实际振荡频率 ω_d = √(ω₀² − α²) 小于固有频率 ω₀。你的表达式漏掉了指数衰减因子 e^{−αt}，正确形式应为 e^{−αt}(A₁ cos ω_d t + A₂ sin ω_d t)。

### QCT-09-004 RLC并联零输入响应

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T03（RLC并联零输入响应）
- 难度：综合
- 主要知识点：
  - RLC并联电路方程
  - 并联与串联对偶关系
  - 特征根计算
- 关联错因标签：
  - characteristic_root_error
  - conductance_resistance_confusion
  - rlc_duality_confusion
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 并联电路中 R = 4 Ω，L = 2 H，C = 0.5 F。初始 i_L(0⁻)=2 A，u_C(0⁻)=0。求零输入响应 i_L(t) 的表达式形式。

#### 标准答案

对并联 RLC，以 i_L 为变量：
C·d²u_C/dt² + (1/R)·du_C/dt + (1/L)·u_C = 0 不直接适用。
以 i_L 为响应时：由 KCL：
u_C/R + i_L + C·du_C/dt = 0
且 u_C = L·di_L/dt
代入：L/R·di_L/dt + i_L + LC·d²i_L/dt² = 0
即 LC·d²i_L/dt² + (L/R)·di_L/dt + i_L = 0
特征方程：LC·s² + (L/R)·s + 1 = 0
→ 0.5×2 s² + (2/4)s + 1 = s² + 0.5s + 1 = 0
→ s² + 0.5s + 1 = 0
s₁,₂ = −0.25 ± j√(1 − 0.0625) = −0.25 ± j0.968（欠阻尼）
i_L(t) = e^{−0.25t}(A₁ cos 0.968t + A₂ sin 0.968t)

#### 解题思路

1. 对并联 RLC，以电感电流为变量推导微分方程；
2. 注意对偶关系：串联 RLC 中 R→并联 RLC 中 G(=1/R)；
3. 求解特征方程，判断阻尼类型。

#### 易错点

- 直接把串联 RLC 的公式套用到并联电路（忘记对偶替换）；
- 并联电路以电感电流为主变量时微分方程系数不同；
- α 与 ω₀ 的表达式在并联和串联中不同。

#### 可用于诊断的学生错误表现

- 若将 RLC 串联方程直接用于并联，可判断为 rlc_duality_confusion；
- 若特征方程系数明显错误，可判断为 conductance_resistance_confusion（电导、电阻混用）。

#### 推荐反馈语

RLC 并联电路不能直接套串联公式。对偶关系中串联的电阻对应并联的电导。请重新推导以电感电流为变量的微分方程，注意 KCL 和元件约束的正确代入。

### QCT-09-005 二阶直流激励全响应

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T09（直流激励下二阶响应）
- 难度：综合
- 主要知识点：
  - 直流激励下的全响应
  - 稳态分量与暂态分量
  - 初始条件确定
- 关联错因标签：
  - second_order_forced_response_error
  - response_component_confusion
  - second_order_initial_condition_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 串联电路，R = 3 Ω，L = 0.5 H，C = 0.2 F。t = 0 时接入 10 V 直流电源。初始 u_C(0⁻)=2 V，i_L(0⁻)=0。求 t ≥ 0 时的 u_C(t)。

#### 标准答案

特征方程：s² + (R/L)s + 1/(LC) = s² + 6s + 10 = 0
s₁,₂ = −3 ± j1（欠阻尼）
稳态值：u_C(∞) = 10 V（电容开路后两端电压等于电源电压）
全响应形式：u_C(t) = 10 + e^{−3t}(A₁ cos t + A₂ sin t)

初始条件：
u_C(0⁺) = 2 → 10 + A₁ = 2 → A₁ = −8
du_C/dt(0⁺) = (1/C) i_L(0⁺) = 0
du_C/dt = −3e^{−3t}(A₁ cos t + A₂ sin t) + e^{−3t}(−A₁ sin t + A₂ cos t)
代入 t=0：du_C/dt(0⁺) = −3A₁ + A₂ = 0 → A₂ = 3A₁ = −24
∴ u_C(t) = 10 + e^{−3t}(−8 cos t − 24 sin t) V

#### 解题思路

1. 稳态值由直流稳态确定（电容开路）；
2. 特征根决定暂态响应形式；
3. 全响应 = 稳态响应 + 暂态响应；
4. 用 u_C(0⁺) 和 du_C/dt(0⁺) 确定系数。

#### 易错点

- 漏写稳态强迫项（只写暂态部分）；
- 导数初值 du_C/dt(0⁺) 没有通过 i_L(0⁺)/C 计算；
- 欠阻尼响应中待定系数只用一个初值确定。

#### 可用于诊断的学生错误表现

- 若 u_C(t) 只有衰减振荡项、没有 +10 V 稳态项，可判断为 second_order_forced_response_error；
- 若只用一个初值条件求两个系数，可判断为 second_order_initial_condition_error。

#### 推荐反馈语

直流激励下的二阶全响应必须包含稳态强迫项。请先求换路后电容开路时的稳态值 u_C(∞)，再叠加暂态项。系数则需要两个初始条件联立求解。

### QCT-09-006 二阶电路冲激响应

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T12（冲激响应与状态跳变）
- 难度：综合
- 主要知识点：
  - 冲激响应
  - 状态跳变
  - 冲激后的自然响应
- 关联错因标签：
  - impulse_response_error
  - impulse_state_jump_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 串联电路中 R = 2 Ω，L = 1 H，C = 1 F。初始状态均为零。t = 0 时施加单位冲激电压源 u_s(t) = δ(t) V。求 0⁺ 时刻的状态跳变。

#### 标准答案

对串联 RLC，KVL：L·di_L/dt + R·i_L + u_C = δ(t)
在 t = 0 附近积分：∫ L·di_L = ∫ δ(t) dt → L·[i_L(0⁺) − i_L(0⁻)] = 1
i_L(0⁻) = 0 → i_L(0⁺) = 1/L = 1 A

电容电压：i_L = C·du_C/dt → du_C/dt = i_L/C
在 t=0⁺ 时：du_C/dt(0⁺) = i_L(0⁺)/C = 1/1 = 1 V/s
u_C 本身连续：u_C(0⁺) = u_C(0⁻) = 0

冲激后（t > 0⁺）为零输入响应，初始条件：
u_C(0⁺) = 0，du_C/dt(0⁺) = 1 V/s

#### 解题思路

1. 冲激在无限短时间内注入能量，导致电感电流跳变；
2. 对微分方程两边在 0⁻ 到 0⁺ 积分；
3. 冲激后的响应转化为初始条件确定的零输入响应。

#### 易错点

- 认为冲激下所有状态变量都不跳变；
- 把冲激直接作为普通幅值代入微分方程求解；
- 只是用阶跃响应求导得到冲激响应，但漏掉不连续点处的冲激项。

#### 可用于诊断的学生错误表现

- 若认为 i_L(0⁺) = i_L(0⁻) = 0，可判断为 impulse_state_jump_error；
- 若直接用 δ(t) 代入微分方程求解而不考虑积分跳变，可判断为 impulse_response_error。

#### 推荐反馈语

冲激响应不能把 δ(t) 当作普通函数代入方程。正确的做法是在 0⁻ 到 0⁺ 的无限小区间上对微分方程两边积分，求解状态变量的跳变值，再将跳变后的值作为初始条件求零输入响应。

### QCT-10-002 相位超前滞后判断

- 所属章节：第10章 正弦稳态分析
- 对应题型：CT10-T02（相位关系判断）
- 难度：基础
- 主要知识点：
  - 相位差计算
  - 超前滞后判断
  - 统一三角函数基准
- 关联错因标签：
  - phase_lead_lag_error
  - phase_reference_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 题目

已知 u(t) = 10 cos(100t + 60°) V，i(t) = 5 sin(100t − 30°) A。求 u 与 i 的相位差，并判断谁超前谁滞后。

#### 标准答案

先将 i(t) 统一为 cos 形式：sin(ωt + θ) = cos(ωt + θ − 90°)
i(t) = 5 cos(100t − 30° − 90°) = 5 cos(100t − 120°) A
相位差 φ = φ_u − φ_i = 60° − (−120°) = 180°
u 超前 i 180°（即反相），或说 u 与 i 反相。

#### 解题思路

1. 必须将两者统一为同一种三角函数（都用 cos 或都用 sin）；
2. 相位差 = 电压初相 − 电流初相；
3. Δφ > 0 表示 u 超前 i（若基准一致）。

#### 易错点

- 不统一三角函数基准就直接减初相位；
- 把 sin 和 cos 的 90° 关系搞反（sin → cos 是减 90°，cos → sin 是加 90°）；
- 认为 u 总是超前 i。

#### 可用于诊断的学生错误表现

- 若 φ = 60° − (−30°) = 90°，可判断为 phase_lead_lag_error（未统一三角函数基准）；
- 若将 sin(100t − 30°) = cos(100t − 30° + 90°)，可判断为 sin-cos 转换方向错误。

#### 推荐反馈语

比较相位前必须把两个量统一成同一种三角函数。sin 转 cos 是减 90°，cos 转 sin 是加 90°。你先统一后再算相位差试试。

### QCT-10-003 RLC阻抗计算

- 所属章节：第10章 正弦稳态分析
- 对应题型：CT10-T08（阻抗与导纳计算）
- 难度：标准
- 主要知识点：
  - RLC相量模型
  - 阻抗计算
  - 感抗与容抗
- 关联错因标签：
  - impedance_model_error
  - reactance_sign_error
  - complex_impedance_simplification_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 题目

R = 30 Ω，L = 0.2 H，C = 50 μF 串联，接于 ω = 100 rad/s 的正弦电源。求：
（1）感抗 X_L、容抗 X_C；
（2）总阻抗 Z（复数形式）；
（3）判断电路呈感性还是容性。

#### 标准答案

（1）X_L = ωL = 100 × 0.2 = 20 Ω
X_C = 1/(ωC) = 1/(100 × 50 × 10⁻⁶) = 1/0.005 = 200 Ω
（2）Z = R + j(X_L − X_C) = 30 + j(20 − 200) = 30 − j180 Ω
（3）阻抗虚部为负 → 电路呈容性。

#### 解题思路

1. 感抗 X_L = ωL，容抗 X_C = 1/(ωC)；
2. 串联总阻抗 Z = R + j(X_L − X_C)；
3. 电压滞后电流（容性）或超前电流（感性）由阻抗虚部符号决定。

#### 易错点

- 电容阻抗写成 +j/(ωC) 而不是 −j/(ωC)；
- X_L 和 X_C 公式混淆（X_L = 1/(ωL) 等错误）；
- 串联阻抗计算中把 X_L + X_C 直接相加而不考虑符号差异；
- 只计算阻抗的模值，丢掉相位信息。

#### 可用于诊断的学生错误表现

- 若 X_C = ωC = 100 × 50 × 10⁻⁶ = 0.005 Ω，可判断为 reactance_sign_error（容抗公式写错）；
- 若 Z = 30 + j220 Ω（将感抗容抗相加），可判断为 impedance_model_error。

#### 推荐反馈语

电容的阻抗是 −j/(ωC)，不是 +jωC。请先分别计算 X_L = ωL 和 X_C = 1/(ωC)，求串联阻抗时用 Z = R + j(X_L − X_C) 得到复数。

### QCT-10-004 相量法求解正弦稳态

- 所属章节：第10章 正弦稳态分析
- 对应题型：CT10-T09（串联正弦稳态电路分析）
- 难度：标准
- 主要知识点：
  - 相量域 KVL
  - 分压计算
  - 时域与相量转换
- 关联错因标签：
  - phasor_kcl_kvl_error
  - phasor_conversion_error
  - impedance_model_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 题目

RC 串联电路，R = 40 Ω，C = 50 μF，正弦电压 u_s(t) = 100 cos(500t + 45°) V。求电流 i(t) 和电容电压 u_C(t)。

#### 标准答案

相量域：U̇_s = (100/√2)∠45° V（或 U̇_s = 100∠45° V 用幅值相量，此处用有效值）
此处使用有效值相量：U̇_s = (100/√2)∠45° ≈ 70.7∠45° V
X_C = 1/(ωC) = 1/(500 × 50 × 10⁻⁶) = 40 Ω
Z = R − jX_C = 40 − j40 = 40√2∠−45° Ω
İ = U̇_s/Z = (70.7∠45°)/(56.57∠−45°) = 1.25∠90° A
i(t) = 1.25√2 cos(500t + 90°) = 1.77 cos(500t + 90°) A

U̇_C = İ · (−jX_C) = 1.25∠90° × 40∠−90° = 50∠0° V
u_C(t) = 50√2 cos(500t) ≈ 70.7 cos(500t) V

#### 解题思路

1. 将时域信号转换为相量；
2. 计算容抗 X_C，写出复阻抗 Z；
3. 用相量形式欧姆定律求电流相量；
4. 用分压或乘阻抗求电容电压相量；
5. 将相量转换回时域余弦表达。

#### 易错点

- 时域→相量转换中漏掉 √2（有效值与幅值混淆）；
- 电容阻抗符号写错（应为 −jX_C）；
- 相量除法时角度计算错误。

#### 可用于诊断的学生错误表现

- 若 i(t) 写为 1.25 cos(500t + 90°)（漏掉 √2），可判断为 rms_value_error；
- 若 Z = 40 + j40，可判断为 reactance_sign_error（电容阻抗符号错误）。

#### 推荐反馈语

你的相量计算中电容阻抗符号反了。电容阻抗是 −jX_C，所以 Z = R − jX_C。另外要注意相量域中使用的是有效值，回到时域时要乘以 √2。

### QCT-10-005 交流电桥平衡条件

- 所属章节：第10章 正弦稳态分析
- 对应题型：CT10-T17（交流电桥平衡）
- 难度：综合
- 主要知识点：
  - 交流电桥平衡条件
  - 复数阻抗的幅值和相位
  - 电桥设计
- 关联错因标签：
  - ac_bridge_balance_error
  - complex_impedance_simplification_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 题目

一个电容电桥用于测量未知电容 C_x 和损耗电阻 R_x。已知标准桥臂：R₂ = 1 kΩ，C₂ = 0.1 μF 串联；R₃ = 500 Ω；R₄ = 2 kΩ。电源角频率 ω = 1000 rad/s。求电桥平衡时的 C_x 和 R_x。

#### 标准答案

交流电桥平衡条件：Z₁·Z₄ = Z₂·Z₃
Z₂ = R₂ + 1/(jωC₂)，Z₃ = R₃，Z₄ = R₄，Z_x = R_x + 1/(jωC_x)
(R_x + 1/(jωC_x)) · R₄ = (R₂ + 1/(jωC₂)) · R₃
实部相等：R_x · R₄ = R₂ · R₃ → R_x = R₂·R₃/R₄ = 1000×500/2000 = 250 Ω
虚部相等：R₄/(ωC_x) = R₃/(ωC₂) → C_x = C₂·R₄/R₃ = 0.1×2000/500 = 0.4 μF

#### 解题思路

1. 电桥平衡条件为对臂阻抗乘积相等；
2. 代入四个桥臂的复阻抗；
3. 实部和虚部分别相等，得到两个方程；
4. 解出待测元件参数。

#### 易错点

- 只写实部条件（电阻平衡），忽略虚部条件（电容/电感平衡）；
- 平衡条件公式写反（写成 Z₁·Z₃ = Z₂·Z₄ 等其他形式）；
- 虚部符号计算错误。

#### 可用于诊断的学生错误表现

- 若只根据 R_x·R₄ = R₂·R₃ 求解而忽略电容条件，可判断为 ac_bridge_balance_error；
- 若平衡条件写成 Z₁·Z₂ = Z₃·Z₄，可判断为 bridge_balance_condition_error。

#### 推荐反馈语

交流电桥平衡是复数条件，必须同时满足实部和虚部两个方程。你只分析了电阻部分，还需要检查电容/电感引起的虚部平衡条件。

### QCT-10-006 移相电路分析

- 所属章节：第10章 正弦稳态分析

- 难度：综合
- 主要知识点：

  - 相位超前与滞后判断
  - 相量法
- 关联错因标签：

  - phasor_conversion_error
- 推荐复习文件：chapters/10_sinusoidal_steady_state_analysis.md

#### 题目

RC 串联电路中，R = 10 kΩ，C = 0.01 μF，输入电压 u_i(t) = 5 cos(ωt) V，ω = 10⁴ rad/s。求输出电压 u_o(t)（取电容两端电压），并求相移。

X_C = 1/(ωC) = 1/(10⁴ × 0.01 × 10⁻⁶) = 1/(10⁻⁴) = 10 kΩ

#### 解题思路

1. 电容电压作为输出时，该电路为滞后移相；
2. 用相量分压公式求输出；
3. 相移由阻抗分压得到。

- 认为电容输出一定超前（取决于取电阻还是电容两端电压）；

RC 移相电路中，取电容电压为输出时得到滞后移相，取电阻电压为输出时得到超前移相。请重新用相量分压公式推导传输函数，确认相角的正负号。

### QCT-11-001 有功、无功、视在功率计算

- 所属章节：第11章 正弦稳态电路的功率
- 对应题型：CT11-T01（有功功率、无功功率、视在功率计算）
- 难度：基础
- 主要知识点：
  - 有功功率 P
  - 无功功率 Q
  - 视在功率 S
  - 功率因数
- 关联错因标签：
  - active_power_error
  - reactive_power_sign_error
  - apparent_power_error
  - power_factor_error
- 推荐复习文件：chapters/11_sinusoidal_steady_state_power.md

#### 题目

某负载端电压 U̇ = 220∠0° V，电流 İ = 5∠−30° A（有效值）。求：
（1）有功功率 P、无功功率 Q、视在功率 S；
（2）功率因数 λ 及负载性质（感性/容性）。

#### 标准答案

（1）电压电流相位差 φ = φ_u − φ_i = 0° − (−30°) = 30°
P = UI cos φ = 220 × 5 × cos 30° = 1100 × 0.866 ≈ 952.6 W
Q = UI sin φ = 220 × 5 × sin 30° = 1100 × 0.5 = 550 var
S = UI = 220 × 5 = 1100 V·A
（2）λ = cos φ = cos 30° ≈ 0.866（滞后，因为 I 滞后 U → 感性负载）

#### 解题思路

1. 由电压、电流相量求相位差；
2. P = UI cos φ，Q = UI sin φ，S = UI；
3. 感性负载电流滞后电压（φ > 0），功率因数滞后。

#### 易错点

- 无功功率公式漏掉 sin φ 直接写 Q = UI；
- 功率因数 λ 只写数值不注明滞后/超前；
- I 滞后 U 时误判为容性。

#### 可用于诊断的学生错误表现

- 若 P = 220 × 5 = 1100 W（漏掉 cos φ），可判断为 active_power_error；
- 若 λ = 0.866（超前），可判断为 power_factor_error（负载性质判断反）；
- 若 Q = 220 × 5 × sin(−30°)，可判断为 reactive_power_sign_error（相位差符号取错）。

#### 推荐反馈语

计算有功功率时一定要乘功率因数 cos φ，不能直接用 UI。另外电流相位 −30° 滞后于电压 0°，说明负载是感性的，功率因数是滞后的。

### QCT-11-002 复功率计算

- 所属章节：第11章 正弦稳态电路的功率
- 对应题型：CT11-T03（复功率计算）
- 难度：标准
- 主要知识点：
  - 复功率定义
  - 共轭关系
  - 复功率守恒
- 关联错因标签：
  - complex_power_conjugate_error
- 推荐复习文件：chapters/11_sinusoidal_steady_state_power.md

#### 题目

某负载端电压 U̇ = 200∠45° V，电流 İ = 4∠15° A（均为有效值）。求复功率 S̅，并写出有功功率和无功功率。

#### 标准答案

S̅ = U̇ · İ^* = 200∠45° × 4∠−15° = 800∠30° V·A
P = Re[S̅] = 800 cos 30° ≈ 692.8 W
Q = Im[S̅] = 800 sin 30° = 400 var

#### 解题思路

1. 复功率公式：S̅ = U̇ · İ^*（电流取共轭）；
2. 复数乘法：模相乘，角度相加（注意共轭后角度变号）；
3. 实部为 P，虚部为 Q。

#### 易错点

- 忘记对电流相量取共轭（直接用 İ 而不是 İ^*）；
- 复功率的单位是 V·A，不是 W 或 var；
- 复功率的虚部符号由负载性质决定。

#### 可用于诊断的学生错误表现

- 若 S̅ = 200∠45° × 4∠15° = 800∠60°，可判断为 complex_power_conjugate_error（忘记电流共轭）；
- 若将复功率模值 800 直接当作 P，可判断为 apparent_power_error。

#### 推荐反馈语

复功率计算中电流相量必须取共轭。你直接用了 İ 而不是 İ^*，导致 Q 的符号错误。建议记住：S̅ = U̇ · İ^* = P + jQ。

### QCT-11-003 功率因数校正（并联电容）

- 所属章节：第11章 正弦稳态电路的功率
- 对应题型：CT11-T05（功率因数校正）
- 难度：综合
- 主要知识点：
  - 功率因数校正原理
  - 并联电容补偿
  - 无功功率计算
- 关联错因标签：
  - power_factor_correction_error
  - capacitor_compensation_error
  - reactive_power_sign_error
- 推荐复习文件：chapters/11_sinusoidal_steady_state_power.md

#### 题目

某感性负载接 220 V、50 Hz 电源。负载有功功率 P = 1 kW，功率因数 λ₁ = 0.6（滞后）。现要将功率因数提高到 λ₂ = 0.95（滞后）。求需并联的电容 C 的值。

#### 标准答案

λ₁ = 0.6 → cos φ₁ = 0.6 → φ₁ = arccos 0.6 ≈ 53.13°，tan φ₁ ≈ 1.333
λ₂ = 0.95 → cos φ₂ = 0.95 → φ₂ ≈ 18.19°，tan φ₂ ≈ 0.329

所需无功补偿：Q_C = P(tan φ₁ − tan φ₂) = 1000 × (1.333 − 0.329) = 1004 var

电容提供无功：Q_C = U²ωC → C = Q_C/(U²·2πf)
= 1004/(220² × 2π × 50) = 1004/(48400 × 314)
= 1004/15197600 ≈ 66.1 μF

#### 解题思路

1. 功率因数从感性（滞后）校正到目标感性（滞后）；
2. 并联电容不改变负载有功，只补偿负载的无功；
3. 补偿无功 Q_C = P(tan φ₁ − tan φ₂)；
4. C = Q_C/(ωU²)。

#### 易错点

- 并联电容后误认为负载有功功率也会改变；
- 功率因数角 φ 搞混（用 cos φ 的值直接代入，而不是通过 tan 计算无功）；
- 补偿公式中的符号搞反（实际是减无功不是加无功）；
- C 的单位换算错误（μF 写成 F 等）。

#### 可用于诊断的学生错误表现

- 若补偿后计算出更大的 P，可判断为 power_factor_correction_error（并联电容不改变有功）；
- 若 C 计算比合理值大 10⁶ 倍（单位错误），可判断为 unit_conversion_error；
- 若 Q_C = P(tan φ₂ − tan φ₁)（符号反了），可判断为 capacitor_compensation_error。

#### 推荐反馈语

功率因数校正是通过并联电容提供无功补偿，不改变负载本身的有功功率。注意公式 Q_C = P(tan φ₁ − tan φ₂) 是减号。你先算补偿前后的无功差值，再求电容值。

### QCT-11-004 交流最大功率传输

  - thevenin_resistance_error
- 推荐复习文件：chapters/11_sinusoidal_steady_state_power.md

#### 题目

正弦稳态电路中，有源二端网络的戴维南等效为：U̇_th = 50∠0° V，Z_th = 10 + j20 Ω。负载 Z_L 可调。求 Z_L 为何值时获得最大有功功率，最大功率 P_max 为多少？

共轭匹配条件：Z_L = Z_th^* = 10 − j20 Ω
最大有功功率：P_max = |U̇_th|²/(4·Re[Z_th]) = 50²/(4 × 10) = 2500/40 = 62.5 W

交流最大功率传输要求共轭匹配：负载电抗与源电抗大小相等、符号相反。公式中分母是 4 倍的等效电阻实部，不是阻抗模值。请重新计算 Z_th 的共轭。

（2）线电压 u_AB(t)、u_BC(t)、u_CA(t) 的表达式。

（2）U̇_AB = U̇_A − U̇_B = 220∠0° − 220∠−120° = 220√3∠30° V

Y 形连接中线电压是相电压的 √3 倍，且超前对应的相电压 30°。你先写出 A、B 相电压相量，再用 U̇_AB = U̇_A − U̇_B 验证一下。

### QCT-12-002 对称三相电路功率计算

- 所属章节：第12章 三相正弦稳态电路

- 难度：标准
- 主要知识点：

  - three_phase_power_formula_error
  - power_factor_error

- 推荐复习文件：chapters/12_three_phase_sinusoidal_circuits.md

#### 题目

对称三相负载接成 Y 形，线电压 U_l = 380 V，线电流 I_l = 10 A，功率因数 λ = 0.8（滞后）。求三相总有功功率 P 和无功功率 Q。

≈ 1.732 × 380 × 10 × 0.8 ≈ 5265.3 W ≈ 5.27 kW

φ = arccos 0.8 ≈ 36.87°，sin φ ≈ 0.6
Q = √3 · U_l · I_l · sin φ = √3 × 380 × 10 × 0.6 ≈ 3949 var ≈ 3.95 kvar

#### 解题思路

1. 对 Y 连接对称三相负载可直接使用线-线公式；
2. P = √3 U_l I_l cos φ；
3. φ 是每相负载的阻抗角（即相电压与相电流的相位差）。

#### 易错点

- 忘了 √3 系数（写成 P = U_l I_l cos φ）；
- 把线电压 380 V 当作相电压使用；
- 将功率因数角 φ 理解为线电压与线电流的相位差而不是相电压与相电流。

#### 可用于诊断的学生错误表现

- 若 P = 380 × 10 × 0.8 = 3040 W（漏√3），可判断为 three_phase_power_formula_error；
- 若 φ 直接取 arccos 0.8 但理解成线电压与线电流夹角，可判断为 power_factor_error。

#### 推荐反馈语

三相功率公式中有一个 √3 因子：P = √3 U_l I_l cos φ。另外功率因数角 φ 是每相的阻抗角（相电压与相电流的夹角），不能理解成线电压与线电流的夹角。

### QCT-12-003 不对称Y形电路分析（中性点位移）

- 所属章节：第12章 三相正弦稳态电路
- 对应题型：CT12-T07（不对称三相电路）
- 难度：综合
- 主要知识点：
  - 不对称负载
  - 中性点位移
  - 中线电流
- 关联错因标签：
  - unbalanced_three_phase_error
  - neutral_shift_error
  - phase_line_quantity_confusion
- 推荐复习文件：chapters/12_three_phase_sinusoidal_circuits.md

#### 题目

三相四线制 Y-Y 电路中，对称电源相电压 U_p = 220 V（有效值），A、B、C 三相负载分别为 Z_A = 10 Ω（纯阻），Z_B = 10 Ω（纯阻），Z_C = 20 Ω（纯阻），中线阻抗 Z_N = 0。求各相电流和中线电流。

#### 标准答案

由于 Z_N = 0，负载中性点电位等于电源中性点电位，各相独立。
I_A = U_A/Z_A = 220∠0°/10 = 22∠0° A
I_B = U_B/Z_B = 220∠−120°/10 = 22∠−120° A
I_C = U_C/Z_C = 220∠120°/20 = 11∠120° A
中线电流：İ_N = İ_A + İ_B + İ_C
= 22∠0° + 22∠−120° + 11∠120°
= 22 + (−11 − j19.05) + (−5.5 + j9.53)
= 5.5 − j9.52
= 11∠−60° A

#### 解题思路

1. 当 Z_N = 0 时，各相独立，每相可按单相电路计算；
2. 中线电流等于三相电流的相量之和。

#### 易错点

- 认为不对称时仍可直接使用分相计算法而不考虑中线阻抗；
- 中线电流用标量相加而非相量相加；
- 将 Z_N = 0 时各相独立的概念遗忘，错误地使用节点法求中性点电压。

#### 可用于诊断的学生错误表现

- 若中线电流 I_N = 22 + 22 + 11 = 55 A（标量相加），可判断为 unbalanced_three_phase_error；
- 若直接使用分相计算法但忽略 Z_N=0 条件，可判断为 balanced_three_phase_condition_error。

#### 推荐反馈语

不对称电路中，中线电流是三相电流的相量和，不是代数和。你需要将三个电流相量在复平面上相加，不能直接作标量加法。本题中 Z_N = 0，各相独立计算，最后相量求和。

### QCT-12-004 三相功率测量——二表法

- 所属章节：第12章 三相正弦稳态电路
- 对应题型：CT12-T09（三相功率测量）
- 难度：综合
- 主要知识点：
  - 二表法测量三相功率
  - 功率表读数
  - 对称负载
- 关联错因标签：
  - three_phase_power_measurement_error
  - wattmeter_connection_error
- 推荐复习文件：chapters/12_three_phase_sinusoidal_circuits.md

#### 题目

对称三相负载（感性，功率因数 0.5）接于线电压 380 V 的三相电源。线电流 20 A。采用二表法测量三相功率，求两个功率表的读数 W₁ 和 W₂。

#### 标准答案

φ = arccos 0.5 = 60°
W₁ = U_l I_l cos(30° − φ) = 380×20×cos(30°−60°) = 380×20×cos(−30°)
= 380×20×0.866 ≈ 6582 W
W₂ = U_l I_l cos(30° + φ) = 380×20×cos(30°+60°) = 380×20×cos 90° = 0 W
总功率 P = W₁ + W₂ = 6582 W（也可验证：P = √3×380×20×0.5 ≈ 6582 W）

#### 解题思路

1. 二表法：W₁ = U_l I_l cos(30° − φ)，W₂ = U_l I_l cos(30° + φ)；
2. 两表读数之和等于总有功功率；
3. φ 是负载阻抗角。

#### 易错点

- 二表法的接线方式搞错（电流线圈应该串联在对应相线）；
- 直接认为两表读数相等或总功率就是两表读数简单相加；
- 二表法中其中一表可能读数为负（当 φ > 60° 时），此时应正确读取。

#### 可用于诊断的学生错误表现

- 若 W₁ = W₂ = 3291 W（平均分配），可判断为 three_phase_power_measurement_error；
- 若 cos(30° − φ) 和 cos(30° + φ) 的角度符号搞反，可判断为功率表接线相序混。

#### 推荐反馈语

二表法中两个功率表的读数一般不相等。公式是 W₁ = U_l I_l cos(30° − φ)，W₂ = U_l I_l cos(30° + φ)。当 φ = 60° 时 W₂ 恰好为零。请重新代入 φ = 60° 试试。

### QCT-13-001 同名端与互感电压极性

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T02（同名端判断）
- 难度：基础
- 主要知识点：
  - 同名端概念
  - 互感电压极性
  - 耦合电感 KVL
- 关联错因标签：
  - dot_convention_error
  - mutual_voltage_polarity_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

两个耦合电感，L₁ = 4 H，L₂ = 1 H，M = 1 H。已知 i₁(t) 从 L₁ 的同名端流入，i₂(t) 从 L₂ 的同名端流出。电流变化率 di₁/dt = 2 A/s，di₂/dt = 1 A/s。求：
（1）L₁ 两端的总感应电压 u₁（自感 + 互感）；
（2）L₂ 两端的总感应电压 u₂。

#### 标准答案

根据同名端和参考方向，互感电压符号规则：
若电流从同名端流入，该电流在另一线圈同名端产生的互感电压为正（相对于同名端）。

（1）u₁ = L₁·di₁/dt + M·di₂/dt
注意：i₂ 从 L₂ 的同名端流出，因此 i₂ 在 L₁ 中产生的互感电压方向与参考方向相反 → 取负号
→ u₁ = 4×2 + 1×(−1) = 8 − 1 = 7 V
（2）u₂ = L₂·di₂/dt + M·di₁/dt
i₁ 从 L₁ 同名端流入，在 L₂ 同名端产生正极性互感电压
→ u₂ = 1×1 + 1×2 = 1 + 2 = 3 V

#### 解题思路

1. 根据同名端和电流方向判断互感电压的符号；
2. 总电压 = 自感电压 + 互感电压（注意符号）；
3. 如果电流从同名端流入，在另一线圈同名端产生正极性互感电压。

#### 易错点

- 机械记忆：只要看到同名端就取正号，忽略电流方向；
- 互感电压的符号只在电流同时流入同名端时才同号；
- 把一个电流在自身线圈中的自感电压和它在另一线圈的互感电压混淆。

#### 可用于诊断的学生错误表现

- 若 u₁ = 8 + 1 = 9 V（互感项符号取反），可判断为 mutual_voltage_polarity_error；
- 若 u₁ = 4×2 + 1×2 = 10 V（互感项使用了 di₁/dt 而不是 di₂/dt），可判断为 dot_convention_error。

#### 推荐反馈语

互感电压的符号取决于电流方向与同名端的关系，不能只看同名端位置。i₂ 从异名端流出，因此在 L₁ 中的互感电压是负的。建议先画出电流方向和同名端，再逐项判断符号。

### QCT-13-002 去耦等效电路

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T05（去耦等效）
- 难度：标准
- 主要知识点：
  - 去耦等效变换
  - T形等效
  - 耦合电感网孔分析
- 关联错因标签：
  - decoupling_equivalent_error
  - coupled_inductor_kvl_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

两个耦合电感 L₁ = 6 H，L₂ = 4 H，M = 2 H，同名端在同一侧（耦合端相连、非耦合端分别接外电路）。求它们串联后（顺接串联）的总等效电感 L_eq。

#### 标准答案

顺接串联：两个线圈的同名端在同一侧，电流从两线圈的同名端流入。
L_eq = L₁ + L₂ + 2M = 6 + 4 + 2×2 = 14 H

#### 解题思路

1. 顺接（同名端方向一致）：等效电感 L_eq = L₁ + L₂ + 2M；
2. 反接（异名端方向一致）：等效电感 L_eq = L₁ + L₂ − 2M。

#### 易错点

- 顺接和反接的公式混淆（+2M 与 −2M 用反）；
- 忘记了耦合电感串联不是简单相加。

- 若 L_eq = 6 + 4 = 10 H（漏掉 2M 项），可判断为 coupled_inductor_kvl_error（忽略互感影响）；
- 若 L_eq = 6 + 4 − 4 = 6 H（用了反接公式），可判断为 decoupling_equivalent_error（连接类型判断反）。

#### 推荐反馈语

耦合电感串联不能直接相加。顺接时互感加强，等效电感为 L₁ + L₂ + 2M；反接时互感削弱，为 L₁ + L₂ − 2M。请先判断电流是否都从同名端流入。

### QCT-13-003 理想变压器电压电流关系

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T12（理想变压器电压电流关系）
- 难度：基础
- 主要知识点：
  - 理想变压器电压比
  - 理想变压器电流比
  - 同名端与方向
- 关联错因标签：
  - ideal_transformer_ratio_error
  - ideal_transformer_current_direction_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

理想变压器匝数比为 n = N₁/N₂ = 5。原边电压 u₁(t) = 100 cos(ωt) V，副边接纯电阻负载 R_L = 25 Ω。同名端如图所示。求：
（1）副边电压 u₂(t)；
（2）副边电流 i₂(t)；
（3）原边电流 i₁(t)（注意电流方向与同名端的关系）。

#### 标准答案

（1）u₂ = u₁/n = 100/5 = 20 V（有效值 20/√2 V，此处用幅值）
    u₂(t) = 20 cos(ωt) V（根据同名端，u₁ 和 u₂ 同极性）
（2）i₂(t) = u₂/R_L = 20 cos(ωt)/25 = 0.8 cos(ωt) A
（3）理想变压器：i₁·N₁ = i₂·N₂（电流从同名端流入/流出时符号为正）
    i₁(t) = (N₂/N₁)·i₂(t) = (1/5)×0.8 cos(ωt) = 0.16 cos(ωt) A
    若 i₁ 参考方向为流入同名端，i₂ 参考方向为流出同名端，则电流比取正号。

#### 解题思路

1. 理想变压器电压比：U₁/U₂ = N₁/N₂；
2. 理想变压器电流比：I₁/I₂ = N₂/N₁（注意电流参考方向与同名端的关系）；
3. 电流方向约定：若 i₁ 和 i₂ 分别从同名端流入，则 i₁·N₁ + i₂·N₂ = 0。

#### 易错点

- 把电流比也写成 N₁/N₂（与电压比同方向）；
- 未考虑同名端方向对电流符号的影响；
- 误以为理想变压器可以改变功率（实际 P₁ = P₂）。

#### 可用于诊断的学生错误表现

- 若 i₁ = 0.8/5 = 0.16 A 但方向符号写反，可判断为 ideal_transformer_current_direction_error；
- 若 i₁ = i₂·n = 0.8×5 = 4 A，可判断为 ideal_transformer_ratio_error（电流比与电压比混淆）。

#### 推荐反馈语

理想变压器中电压比和电流比互为倒数：u₁/u₂ = n，i₁/i₂ = 1/n。你写的电流比方向反了。另外要检查电流参考方向与同名端的关系来确定符号。

### QCT-13-004 理想变压器阻抗变换

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T13（理想变压器阻抗变换）
- 难度：标准
- 主要知识点：
  - 阻抗变换公式
  - 匝比平方关系
  - 折合到原边
- 关联错因标签：
  - ideal_transformer_impedance_transform_error
  - ideal_transformer_ratio_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

理想变压器匝数比 n = N₁/N₂ = 4。副边接阻抗 Z_L = 8 + j6 Ω。求从原边看入的等效阻抗 Z_in。

#### 标准答案

Z_in = n² · Z_L = 4² × (8 + j6) = 16 × (8 + j6) = 128 + j96 Ω

#### 解题思路

1. 理想变压器的阻抗变换公式：Z_in = n² · Z_L；
2. 其中 n = N₁/N₂ 是原边对副边的匝数比；
3. 阻抗变换不仅改变模值，复数阻抗的相位角不变。

#### 易错点

- 公式写成 Z_in = Z_L/n²（把匝比方向弄反）；
- 忘记平方（直接写 Z_in = n·Z_L）；
- 认为阻抗变换后相位角改变。

#### 可用于诊断的学生错误表现

- 若 Z_in = 4 × (8 + j6) = 32 + j24 Ω（漏平方），可判断为 ideal_transformer_impedance_transform_error；
- 若 Z_in = (8 + j6)/16 = 0.5 + j0.375 Ω（匝比方向反了），可判断为 ideal_transformer_ratio_error。

#### 推荐反馈语

理想变压器的阻抗变换是匝数比的平方关系：Z_in = n² Z_L。你漏了平方。注意从原边看，阻抗变大（n > 1 时）。

### QCT-13-005 反映阻抗计算

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T07（反映阻抗）
- 难度：综合
- 主要知识点：
  - 反映阻抗
  - 线性变压器
  - 耦合电感相量分析
- 关联错因标签：
  - reflected_impedance_error
  - coupled_inductor_phasor_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

线性变压器中，原边自感 L₁ = 2 H，副边自感 L₂ = 0.5 H，互感 M = 0.5 H，ω = 100 rad/s。副边接负载 Z_L = 100 Ω（纯电阻）。求原边的输入阻抗 Z_in。

#### 标准答案

原边阻抗：Z₁₁ = jωL₁ = j100×2 = j200 Ω
副边阻抗：Z₂₂ = jωL₂ + Z_L = j100×0.5 + 100 = 100 + j50 Ω
反映阻抗：Z_ref = (ωM)² / Z₂₂ = (100×0.5)² / (100 + j50)
= 2500 / (100 + j50) = 2500 / (111.8∠26.57°) ≈ 22.36∠−26.57°
= 20 − j10 Ω
Z_in = Z₁₁ + Z_ref = j200 + (20 − j10) = 20 + j190 Ω

#### 解题思路

1. 反映阻抗 Z_ref = (ωM)² / Z₂₂，其中 Z₂₂ 是副边总阻抗；
2. 原边输入阻抗 Z_in = Z₁₁（原边自阻抗）+ Z_ref；
3. 反映阻抗将副边负载"反映"到原边。

#### 易错点

- 反映阻抗公式写成 (ωM)²·Z₂₂（分子分母颠倒）；
- 忘记 Z₂₂ 包含副边自阻抗和负载阻抗两部分；
- 反映到原边后仍保留 jωL₁ 项，忘记原边还有自阻抗。

#### 可用于诊断的学生错误表现

- 若 Z_ref = (ωM)² × Z₂₂ = 2500 × (100 + j50)，可判断为 reflected_impedance_error（公式写反）；
- 若 Z_in = Z_ref（忽略原边自阻抗 jωL₁），可判断为 linear_transformer_model_error。

#### 推荐反馈语

反映阻抗是 (ωM)² 除以副边总阻抗，不是乘以。另外输入阻抗还要加上原边的自阻抗。请写出从原边端看入的完整阻抗表达式。

  - 开路电压
  - 等效电阻
  - 戴维南定理
- 关联错因标签：
  - thevenin_resistance_error
  - voltage_polarity_error
- 推荐复习文件：chapters/04_circuit_theorems.md

#### 题目

已知电路中，U_s = 20 V，R₁ = 5 Ω，R₂ = 5 Ω，受控源类型未知（此题为独立源电路）。求 ab 端口的戴维南等效电路。

#### 标准答案

（此处给出标准流程）
开路电压：U_oc = U_s × R₂/(R₁ + R₂) = 20 × 6/(4 + 6) = 12 V
等效电阻（独立源置零）：R_eq = R₁ ∥ R₂ = 4∥6 = 2.4 Ω

1. 断开负载，求端口开路电压；
2. 独立源置零（电压源短路），求端口等效电阻；
3. 电压源串联电阻组成戴维南等效电路。

- 开路电压的极性（参考方向）标反。

求戴维南等效电阻时，独立电压源要短路、独立电流源要开路。请确认你的端口参考方向与开路电压极性一致。

### QCT-08-008 分段换路——先充电再放电

- 难度：标准
- 主要知识点：

- 关联错因标签：
  - piecewise_switching_error
  - dynamic_initial_condition_error
- 推荐复习文件：chapters/08_first_order_transient_analysis.md

#### 题目

RC 串联电路：R=1kΩ，C=100μF。t=0 时开关闭合接入直流电源 12V；t=1s 时开关断开，电容通过 R 放电。

### QCT-09-007 RLC 串联零输入响应——临界阻尼

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T05（临界阻尼响应）
- 难度：标准
- 主要知识点：
  - 临界阻尼响应
  - 重根情况
  - 边界条件
- 关联错因标签：
  - damping_case_judgment_error
  - characteristic_root_error
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 串联电路：R=4Ω，L=1H，C=0.25F。初始条件：uC(0⁺)=10V，iL(0⁺)=0A。
（1）求阻尼系数 α 和固有角频率 ω₀，判断阻尼类型；
（2）写出 uC(t) 的表达式（t≥0）。

#### 标准答案

（1）α = R/(2L) = 4/2 = 2，ω₀ = 1/√(1×0.25) = 2，α=ω₀→临界阻尼
（2）特征根 s₁=s₂=-2。uC(t) = (A+Bt)e^{-2t}V
由 uC(0⁺)=10→A=10
iC=C·duC/dt→uC'(0⁺)=iC(0⁺)/C=iL(0⁺)/C=0
uC'(t) = Be^{-2t} - 2(A+Bt)e^{-2t} → uC'(0⁺)=B-20=0→B=20
uC(t) = (10+20t)e^{-2t}V

#### 解题思路

1. 计算 α 和 ω₀，比较大小
2. 临界阻尼响应形式：uC(t) = (A+Bt)e^{-αt}
3. 用 uC(0⁺) 和 uC'(0⁺) 确定系数

#### 易错点

- 临界阻尼响应形式漏写 t 前面的系数项（写成 Ae^{-αt}）
- 求 uC'(0⁺) 时忘记用 iC 关系

#### 可用于诊断的学生错误表现

- 若学生写 uC(t)=10e^{-2t}，判断为 damping_case_judgment_error（缺 t 项）
- 若学生判成了过阻尼，判断为 characteristic_root_error

#### 推荐反馈语

临界阻尼的特征根是重根，响应形式必须包含 (A+Bt)e^{-αt}。只写一个指数项会漏掉 Bt 部分的贡献。系数由初始值和导数初值共同确定。

### QCT-09-008 RLC 并联零输入响应

- 所属章节：第9章 二阶电路的暂态分析
- 对应题型：CT09-T03（RLC并联零输入响应）
- 难度：标准
- 主要知识点：
  - RLC 并联
  - 特征方程
  - 对偶关系
- 关联错因标签：
  - conductance_resistance_confusion
  - rlc_duality_confusion
- 推荐复习文件：chapters/09_second_order_transient_analysis.md

#### 题目

RLC 并联电路：G=0.5S（等效于 R=2Ω），L=1H，C=0.5F。初始条件：uC(0⁺)=5V，iL(0⁺)=0A。
（1）求 α 和 ω₀，判断阻尼类型；
（2）写出 uC(t) 表达式。

#### 标准答案

（1）并联 RLC：α = G/(2C) = 0.5/(2×0.5) = 0.5，ω₀ = 1/√(LC) = 1/√(0.5) ≈ 1.414
α < ω₀ → 欠阻尼
（2）ωd = √(ω₀²-α²) = √(2-0.25) = √1.75 ≈ 1.323
uC(t) = e^{-0.5t}[A cos(1.323t) + B sin(1.323t)]V
uC(0⁺)=5→A=5
uC'(0⁺) = -iL(0⁺)/C = 0 → B = (α·A)/ωd = (0.5×5)/1.323 ≈ 1.89
uC(t) = e^{-0.5t}[5cos(1.323t) + 1.89sin(1.323t)]V

#### 解题思路

1. 并联 RLC 的 α = G/(2C)，区别于串联的 α = R/(2L)
2. 特征根 → 欠阻尼 → 衰减振荡形式
3. 系数由 uC(0⁺) 和 uC'(0⁺) 确定

#### 易错点

- 并联 RLC 的 α 公式误用串联公式（R/(2L) 而非 G/(2C)）
- 导数初值 uC'(0⁺) 的符号（注意 iC 参考方向）

#### 可用于诊断的学生错误表现

- 若学生用 R/(2L)=2/(2×1)=1 作为 α，判断为 conductance_resistance_confusion
- 若学生并联电路用了串联的特征方程，判断为 rlc_duality_confusion

#### 推荐反馈语

RLC 并联的分析要注意与串联的对偶关系。并联中阻尼系数是 G/(2C) 而不是 R/(2L)，两者不能混用。建议对照串联和并联的对偶映射表检查参数。

Z₄ = Z₂×Z₃/Z₁ = 200×(50-j159.2)/100 = 100 - j318.4Ω
即 R₄=100Ω，X₄=-318.4Ω（容性），或 L₄=318.4/314≈1.014H

#### 解题思路

1. 写出每个桥臂的复阻抗
2. 代入电桥平衡条件 Z₁Z₄=Z₂Z₃
3. 解得未知阻抗

#### 易错点

- 只考虑阻抗幅值平衡，忽略相位条件
- 电容阻抗虚部符号写反

#### 可用于诊断的学生错误表现

- 若学生只写 100×Z₄=200×50→Z₄=100Ω，判断为 ac_bridge_balance_error（漏相位）
- 若学生把 Z₃ 的虚部符号写为正，判断为 impedance_model_error

#### 推荐反馈语

交流电桥的平衡条件是复数方程，必须同时满足幅值条件和相位条件。仅考虑实部的平衡是不够的，还要检查虚部是否也为零。

### QCT-11-007 最大有功功率传输——共轭匹配

某线性网络的开路电压 Ůoc=100∠0°V，等效阻抗 Z₀=40+j30Ω。负载 ZL 可调，求 ZL 为多少时获得最大有功功率，并求该最大功率。

### QCT-12-006 三相功率测量——二表法

- 所属章节：第12章 三相正弦稳态电路
- 对应题型：CT12-T09（三相功率测量）
- 难度：标准

  - three_phase_power_measurement_error
- 推荐复习文件：chapters/12_three_phase_sinusoidal_circuits.md

#### 题目

对称三相三线制电路，线电压 380V，Y 形负载每相阻抗 Z=20∠60°Ω。用二表法（两功率表法）测量总有功功率。两个功率表的读数分别是多少？总功率是多少？

#### 标准答案

I线 = U线/(√3·|Z|) = 380/(√3×20) ≈ 10.97A
P₁ = U线I线cos(30°-60°) = 380×10.97×cos(-30°) = 380×10.97×0.866 ≈ 3610W
P₂ = U线I线cos(30°+60°) = 380×10.97×cos(90°) = 0W
总功率 P = P₁+P₂ = 3610W

1. 二表法公式：P₁ = U线I线cos(30°-φ)，P₂ = U线I线cos(30°+φ)
2. 总功率 P = P₁+P₂

- 二表法读数公式中的相位角（30°±φ）弄反
- 遇到某个表读数为负时直接取绝对值

#### 可用于诊断的学生错误表现

- 若学生用 P₁=UₚIₚcosφ 直接算单个功率，未用线电压公式，判断为 three_phase_power_measurement_error
- 若学生遇到 P₂=0 时认为仪表坏了，需要解释这是正常现象（当 φ=60°时）

#### 推荐反馈语

二表法中每个功率表的读数不是某相功率，而是线电压和线电流的乘积再乘以 cos(30°±φ)。当 φ=60° 时一个表读数为零是正常情况。

### QCT-13-006 耦合电感网孔分析

两个耦合线圈，L₁=2H，L₂=3H，M=1H。线圈1接电压源 Ůs=100∠0°V（ω=100rad/s），线圈2短路。两线圈的同名端同侧。写出网孔方程并求 İ₁。

j100·İ₁ + j300·İ₂ = 0 → İ₂ = -İ₁/3
代入：j200·İ₁ + j100·(-İ₁/3) = 100
j(200-33.3)İ₁ = 100 → İ₁ = 100/(j166.7) = -j0.6A = 0.6∠-90°A

#### 解题思路

1. 按网孔法列 KVL，注意互感电压项
2. 同名端决定互感电压的符号
3. 解复数方程组

#### 易错点

- 互感电压项 jωMİ₂ 漏写
- 同名端判断反了导致互感电压符号错误

#### 可用于诊断的学生错误表现

- 若学生漏写互感项（只写 jωL₁İ₁=Ůs），判断为 coupled_inductor_kvl_error
- 若学生互感符号写反，判断为 mutual_voltage_polarity_error

#### 推荐反馈语

含耦合电感的网孔分析中，每个网孔的 KVL 都要包含自感电压和互感电压。先标出同名端和电流参考方向，再判断互感电压的正负号。

### QCT-13-007 反映阻抗计算

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T08（反映阻抗）
- 难度：标准
- 主要知识点：
  - 反映阻抗
  - 输入阻抗
  - 互感耦合
- 关联错因标签：
  - reflected_impedance_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

耦合电感参数：L₁=4H，L₂=1H，M=0.5H。原边接电源 Ůs=50∠0°V（ω=100rad/s），副边接负载 ZL=10+j10Ω。求从原边看入的输入阻抗 Zin 和原边电流 İ₁。

#### 标准答案

副边总阻抗 Z₂₂ = R₂ + jωL₂ + ZL = 0 + j100 + 10 + j10 = 10 + j110Ω
反映阻抗 Zr = (ωM)²/Z₂₂ = (100×0.5)²/(10+j110) = 2500/(10+j110) = 2500/(110.45∠84.8°) = 22.63∠-84.8° = 2.0-j22.5Ω
原边总阻抗 Z₁₁ = jωL₁ = j400Ω
Zin = Z₁₁ + Zr = j400 + 2.0 - j22.5 = 2.0 + j377.5Ω
İ₁ = Ůs/Zin = 50/(2.0+j377.5) ≈ 50/(377.5∠89.7°) ≈ 0.132∠-89.7°A

#### 解题思路

1. 计算副边总阻抗 Z₂₂
2. 反映阻抗 Zr = (ωM)²/Z₂₂
3. 输入阻抗 Zin = Z₁₁ + Zr
4. 求原边电流

#### 易错点

- 反映阻抗公式中的 ωM 忘记平方
- 反映阻抗与副边阻抗串联的概念混淆

#### 可用于诊断的学生错误表现

- 若学生写 Zr = ωM/Z₂₂（漏了平方），判断为 reflected_impedance_error
- 若学生把副边负载直接搬到原边（不通过反映阻抗公式），同样为 reflected_impedance_error

#### 推荐反馈语

反映阻抗的计算公式是 Zr = (ωM)²/Z₂₂，注意分子是 (ωM)²，不是 ωM。反映阻抗是一种映射效果，不是实际串入的阻抗。

### QCT-13-008 理想变压器阻抗变换应用

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T13（理想变压器阻抗变换）
- 难度：标准
- 主要知识点：
  - 阻抗变换
  - 最大功率传输
  - 变比设计
- 关联错因标签：
  - ideal_transformer_impedance_transform_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

音频放大器输出阻抗 Z₀=200Ω（纯电阻），需驱动一个 8Ω 的扬声器。通过理想变压器实现阻抗匹配。
（1）求变压器的匝数比 n = N₁/N₂；
（2）若扬声器获得最大功率时输出功率为 10W，求放大器输出电压有效值。

#### 标准答案

（1）阻抗匹配条件：从原边看入的等效电阻 = 200Ω
n² × 8 = 200 → n² = 25 → n = 5（取正）
（2）Pmax = Uoc²/(4R₀) = Uoc²/(4×200) = 10W
Uoc² = 10×800 = 8000 → Uoc = 89.4V

#### 解题思路

1. 扬声器阻抗通过变压器变换到原边：Z' = n²×8Ω
2. 匹配条件：n²×8 = 200 → n=5
3. 最大功率公式 Pmax = Uoc²/(4R₀)

#### 易错点

- 阻抗变换公式写反（用 n 而不是 n²）
- 最大功率传输条件误用成 R_L = R_th（直流条件）

#### 可用于诊断的学生错误表现

- 若学生写 n×8=200→n=25，判断为 ideal_transformer_impedance_transform_error
- 若学生写 P=U²/R 而不是最大功率公式，需提示

#### 推荐反馈语

理想变压器的阻抗变换按匝比的平方进行：Z₁ = n²·Z₂。注意是 n²，不是 n。最大功率传输的共轭匹配条件在纯电阻电路中简化为 R_L = R_s。

### QCT-13-009 耦合系数与互感计算

- 所属章节：第13章 含磁耦合的电路
- 对应题型：CT13-T01（互感与耦合系数）
- 难度：基础
- 主要知识点：
  - 互感计算
  - 耦合系数
  - 自感互感关系
- 关联错因标签：
  - mutual_inductance_definition_error
- 推荐复习文件：chapters/13_magnetically_coupled_circuits.md

#### 题目

两个线圈的自感分别为 L₁=0.5H，L₂=1.2H，耦合系数 k=0.6。
（1）求互感 M；
（2）若两线圈顺向串联（同名端相连），求总等效电感。

#### 标准答案

（1）M = k√(L₁L₂) = 0.6√(0.5×1.2) = 0.6√0.6 = 0.6×0.775 = 0.465H
（2）顺串（同名端相连）：L_eq = L₁+L₂+2M = 0.5+1.2+2×0.465 = 2.63H

#### 解题思路

1. 互感公式 M = k√(L₁L₂)
2. 顺串（磁通加强）：L_eq = L₁+L₂+2M
3. 反串（磁通削弱）：L_eq = L₁+L₂-2M

#### 易错点

- 互感公式中的 k 是乘在根号外面
- 顺串和反串的 ±2M 符号搞反

#### 可用于诊断的学生错误表现

- 若学生写 M = k(L₁+L₂)/2，判断为 mutual_inductance_definition_error
- 若学生顺串写成 L₁+L₂-2M，判断为耦合方式判断错误

#### 推荐反馈语

互感 M 与自感 L₁、L₂ 的关系是 M = k√(L₁L₂)。顺向串联（同名端相连）时磁通加强，等效电感为 L₁+L₂+2M。建议通过画磁通方向来辅助判断。
