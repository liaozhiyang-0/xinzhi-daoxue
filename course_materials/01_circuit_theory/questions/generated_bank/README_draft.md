# 原创题库草稿 README (draft)

## 目录说明

本目录存放《电路理论》课程的**原创题库草稿**，用于支撑学生智能答疑、错题诊断和学习路径推荐。

## 当前状态

- 当前版本：**v0.1 draft**
- 文件：`question_bank_v0_1_draft.md`
- 总题量：40～60 道题目
- 重点覆盖章节：第8～13章（一阶暂态、二阶暂态、正弦稳态分析、正弦稳态功率、三相电路、磁耦合电路）
- 第1～7章：少量补充基础题

## 内容说明

- 所有题目均为原创或基于知识点的改编型题目；
- 不包含教材原题全文和完整例题解析；
- 每道题包含章节信息、题型编号、难度、知识点、错因标签、答案、解题思路、易错点和诊断信息；
- 题型编号与 `questions/question_type_index.md` 保持一致；
- 错因标签与 `wrong_cases/wrong_reason_tags.md` 保持一致。

## 使用方式

1. 人工审核：逐题检查知识点覆盖、题型编号和标签引用的准确性；
2. 合并到正式版：审核通过后将 `question_bank_v0_1_draft.md` 内容合并到正式题库；
3. 扩展：后续可为每道题补充参考答案详解和分步提示。

## 注意事项

- 当前为草稿，需人工审核后再合并为正式版本；
- 题目中的电路图目前使用文字描述，正式版本可考虑补充 ASCII 简图或图片链接；
- 关键公式的上下标和符号仍需复核。
