# 电路理论错因标签字典

## 1. 文件说明

本文件用于统一《电路理论》课程 Agent 的错因识别标签，服务于错题诊断、学习画像、薄弱知识点统计和教师端薄弱知识点分析。

本版本在标签瘦身后的基础上，进一步精修中文名称、适用场景、典型表现和诊断提示，使标签能够直接支持 Agent 判断学生答案中的具体错误。

## 2. 标签命名规则

- 使用英文 snake_case；
- 一个标签只表达一种主要错误；
- 跨章节共用标签只保留一个全局定义；
- 标签定义包括中文名称、适用章节、适用场景、典型表现、诊断提示；
- 中文名称使用自然、规范、专业的中文表达，避免中英文混杂。

## 3. 合并说明

- `virtual_short_misuse` 合并为 `virtual_short_error`，`virtual_open_misuse` 合并为 `virtual_open_error`；
- `capacitor_vi_relation_error` 合并为 `capacitor_iv_relation_error`，`inductor_vi_relation_error` 合并为 `inductor_iv_relation_error`；
- 负载线斜率、截距等细节标签合并为 `load_line_error`；
- 冲激系数、区间、面积等细节标签合并为 `impulse_sampling_error` 或 `impulse_response_error`；
- 相量定义、时域相量混用、微分积分相量关系统一归入 `phasor_conversion_error`；
- 三相线量/相量、Y/Δ、功率和磁耦合相关标签保留关键工程错误层次，不做过度合并。

## 4. 全局错因标签

### ac_bridge_balance_error

中文名称：交流电桥平衡错误

适用章节：
- 第10章 正弦稳态分析

适用场景：
正弦量、相量、复阻抗、相量图、交流节点/网孔法、电桥和振荡移相电路。

典型表现：
- 把交流电桥平衡的适用前提、变量方向或公式形式套错，导致方程项或结果与题图不一致。
- 阻抗/导纳的虚部符号、相量方向或复数运算出错。
- 交流平衡、移相或振荡条件只检查幅值不检查相位。

诊断提示：
先把交流电桥平衡对应的相量、阻抗或相位条件写清楚，再检查幅值与相位是否同时满足题目约束。

### capacitor_voltage_continuity_error

中文名称：电容电压连续性错误

适用章节：
- 第7章 电容、电感及动态电路

适用场景：
换路瞬间、电容初始条件、冲激电流和暂态分析。

典型表现：
- 无冲激电流条件下让电容电压突变。

### characteristic_root_error

中文名称：特征根计算错误

适用章节：
- 第9章 二阶电路的暂态分析

适用场景：
二阶电路零输入响应、直流激励全响应和阻尼类型判断。

典型表现：
- 特征方程系数由回路方程或节点方程推导错误。
- 串联和并联二阶电路的特征方程混用。
- 求根时漏掉平方项或判别式。
- 把实根、重根、共轭复根对应的响应形式混用。

诊断提示：
先写出标准二阶特征方程，再计算特征根，并根据根的类型选择过阻尼、临界阻尼或欠阻尼形式。
