# 错题诊断样例草稿 README (draft)

## 目录说明

本目录存放《电路理论》课程的**错题诊断样例草稿**，用于展示 Agent 对学生典型错误的识别与诊断能力。

## 当前状态

- 当前版本：**v0.1 draft**
- 文件：`wrong_case_examples_v0_1_draft.md`
- 总样例数：20～30 个错题案例
- 重点覆盖章节：第8～13章
- 第1～7章：少量补充样例

## 内容说明

- 所有错题样例均为原创或基于典型学生错误模拟生成的诊断案例；
- 每个样例包含：题目简述、学生错误答案、错误分析、正确思路、Agent 诊断反馈语和推荐复习内容；
- 样例来源题目编号与 `questions/generated_bank/question_bank_v0_1_draft.md` 一致；
- 错因标签与 `wrong_cases/wrong_reason_tags.md` 保持一致。

## 使用方式

1. 人工审核：检查每个样例的诊断逻辑是否合理，反馈语是否准确；
2. 验证 Agent 响应：将样例输入到课程 Agent 中测试其诊断能力；
3. 补充：后续可为更多错因标签补充诊断样例。

## 注意事项

- 当前为草稿，学生错误答案为模拟生成，不代表真实学生数据；
- 反馈语模板可根据实际 Agent 表现调整；
- 建议在实际教学数据中验证诊断规则的准确性。
