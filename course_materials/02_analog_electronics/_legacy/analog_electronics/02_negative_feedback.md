# 负反馈判断

## 1. 基本概念

负反馈是指将输出信号的一部分反馈到输入端，并使净输入量减小的反馈方式。负反馈通常会降低闭环增益，但能改善增益稳定性、线性度、带宽、失真和输入输出阻抗，是模拟电子技术中的核心分析对象。

判断负反馈不能只看“有没有反馈支路”，还必须判断反馈信号回到输入端后是否削弱原输入作用。

## 2. 核心公式或判断条件

闭环增益的一般形式为：

```text
Af = A / (1 + Aβ)
```

其中：

- `A`：基本放大电路的开环增益。
- `β`：反馈网络的反馈系数。
- `1 + Aβ`：反馈深度。

若反馈信号使净输入减小，通常表现为分母为 `1 + Aβ`，属于负反馈；若反馈信号使净输入增大，则属于正反馈。

反馈类型常按两个维度判断：

| 判断维度 | 类型 | 判断依据 |
| --- | --- | --- |
| 输出取样 | 电压反馈 / 电流反馈 | 反馈量与输出电压还是输出电流相关 |
| 输入混合 | 串联反馈 / 并联反馈 | 反馈信号与输入信号以电压串联还是电流并联方式混合 |

## 3. 分析方法

1. 找反馈路径：确认输出端是否有信号返回输入端。
2. 判反馈极性：采用瞬时极性法，假设输入增加，观察反馈信号是否削弱净输入。
3. 判输出取样：令输出电压短路，若反馈消失，通常为电压反馈；若仍与输出电流相关，考虑电流反馈。
4. 判输入混合：反馈信号与输入信号串接比较为串联反馈，在同一输入节点汇合为并联反馈。
5. 分析性能影响：结合反馈类型判断输入电阻、输出电阻、增益和带宽变化。

## 4. 常见易错点

- 看到反馈支路就认定为负反馈，没有判断极性。
- 将“反馈到反相端”机械等同于负反馈，忽略多级放大电路相位变化。
- 混淆电压反馈和电流反馈的取样判断。
- 只记住负反馈降低增益，忽略其提升稳定性和减小非线性失真的意义。
- 深度负反馈条件下仍用开环增益直接估算闭环增益。

## 5. 典型例题

### 例题 1：闭环增益估算

某放大器开环增益 `A = 1000`，反馈系数 `β = 0.01`，则：

```text
Af = 1000 / (1 + 1000 * 0.01) = 1000 / 11 ≈ 90.9
```

虽然增益从 1000 降到约 90.9，但闭环增益对开环增益波动不再敏感。

### 例题 2：错误诊断

学生回答：“输出端有电阻接回输入端，所以一定是负反馈。”

诊断：反馈路径存在不等于负反馈成立。还需要判断反馈信号对净输入的作用方向。如果反馈信号增强净输入，则为正反馈。

## 6. 推荐复习建议

- 按“找路径、判极性、判取样、判混合、看影响”的顺序做题。
- 重点训练瞬时极性法，不要只依赖电路外观。
- 将典型运放反相、同相、射极负反馈电路整理成对比表。
- 错题整理时标注错在“极性判断”“取样判断”还是“混合判断”。
