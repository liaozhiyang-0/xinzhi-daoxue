# CMOS 反相器

## 1. 基本概念

CMOS 反相器由一个 PMOS 上拉管和一个 NMOS 下拉管组成，是 CMOS 数字集成电路中最基本的逻辑门。它实现逻辑非功能：输入为低电平时输出高电平，输入为高电平时输出低电平。

从课程学习角度看，CMOS 反相器同时连接了 MOS 管工作区、开关模型、电压传输特性、噪声容限、传播延迟和功耗等多个知识点。

## 2. 核心公式或判断条件

| 输入状态 | PMOS 状态 | NMOS 状态 | 输出状态 |
| --- | --- | --- | --- |
| `Vin = 0` | 导通 | 截止 | `Vout ≈ VDD` |
| `Vin = VDD` | 截止 | 导通 | `Vout ≈ 0` |
| `Vin` 位于翻转区 | PMOS 与 NMOS 可能同时部分导通 | 存在短路电流 | 输出快速翻转 |

动态功耗近似：

```text
Pdynamic = α Cload VDD^2 f
```

其中 `α` 为翻转活动因子，`Cload` 为负载电容，`VDD` 为供电电压，`f` 为时钟或信号翻转频率。

## 3. 分析方法

1. 根据输入电平判断 PMOS 和 NMOS 的导通状态。
2. 确认输出节点通过哪条路径连接到 `VDD` 或 `GND`。
3. 结合电压传输特性曲线分析开关阈值和噪声容限。
4. 结合负载电容和等效导通电阻分析上升延迟、下降延迟和传播延迟。
5. 区分静态功耗、动态功耗和短路功耗。

## 4. 常见易错点

- 记反 PMOS 与 NMOS 的导通条件。
- 只用理想开关模型解释逻辑功能，忽略翻转区两个管子可能同时导通。
- 将 CMOS 理想静态功耗接近 0 理解为实际功耗为 0。
- 混淆上升时间、下降时间和传播延迟。
- 分析噪声容限时不结合电压传输特性。

## 5. 典型例题

### 例题 1：逻辑状态判断

当 `Vin = 0` 时，PMOS 的栅源电压使其导通，NMOS 截止，输出节点被上拉到 `VDD`，因此输出逻辑 1。

当 `Vin = VDD` 时，PMOS 截止，NMOS 导通，输出节点被下拉到 `GND`，因此输出逻辑 0。

### 例题 2：功耗判断

若某 CMOS 反相器负载电容增大，且输入翻转频率不变，根据 `Pdynamic = α Cload VDD^2 f`，动态功耗会随负载电容增大而增大。

## 6. 推荐复习建议

- 先掌握 PMOS 上拉、NMOS 下拉的开关模型。
- 再结合电压传输曲线理解翻转阈值、噪声容限和稳定输出。
- 最后复习延迟和功耗，将反相器作为理解 CMOS 逻辑门的基础模板。
- 将“逻辑功能、传输特性、延迟、功耗”四类问题分别整理错题。
