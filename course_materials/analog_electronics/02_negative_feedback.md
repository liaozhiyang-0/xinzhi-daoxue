# 负反馈

## 1. 基本概念

负反馈是指将输出信号的一部分以削弱输入作用的方式反馈到输入端。负反馈会降低闭环增益，但能改善放大电路的稳定性、线性度、带宽和输入输出阻抗特性。

## 2. 核心公式或判断条件

闭环增益常写为：

```text
Af = A / (1 + Aβ)
```

其中 `A` 为开环增益，`β` 为反馈系数，`1 + Aβ` 为反馈深度。当 `Aβ > 0` 且反馈信号削弱净输入时，电路形成负反馈。

## 3. 分析方法

1. 找到输出端取样方式，判断是电压取样还是电流取样。
2. 找到输入端混合方式，判断是串联反馈还是并联反馈。
3. 判断反馈信号对净输入的影响，确认正反馈或负反馈。
4. 分析负反馈对增益、带宽、失真和阻抗的影响。

## 4. 常见易错点

- 只看反馈路径，不判断反馈极性。
- 混淆电压反馈和电流反馈。
- 忽略深度负反馈条件下闭环增益主要由反馈网络决定。
- 将“增益降低”误认为负反馈没有价值。

## 5. 典型例题

若某放大器开环增益 `A = 1000`，反馈系数 `β = 0.01`，则闭环增益约为 `1000 / (1 + 10) = 90.9`。负反馈降低了增益，但提升了增益稳定性。

## 6. 推荐复习建议

建议按“取样方式、混合方式、反馈极性、性能影响”的顺序复习，并结合典型运放负反馈电路训练闭环增益计算。
