# MOS 管工作区

## 1. 基本概念

MOS 管工作区用于描述器件在不同端口电压下的导通状态和电流控制关系，是分析放大电路静态工作点、开关电路状态和 CMOS 基本单元的基础。电子信息课程中通常先以增强型 NMOS 为例掌握截止区、线性区和饱和区，再推广到 PMOS 和实际电路。

在模拟电路中，MOS 管常被希望偏置在饱和区，以便作为受控电流源实现电压放大；在数字电路中，MOS 管更多被抽象为开关，重点关注导通和截止状态。

## 2. 核心公式或判断条件

以增强型 NMOS 为例：

| 工作区 | 判断条件 | 物理含义 |
| --- | --- | --- |
| 截止区 | `VGS <= Vth` | 沟道未形成，漏极电流近似为 0 |
| 线性区 | `VGS > Vth` 且 `0 <= VDS < VGS - Vth` | 沟道形成，器件近似受控电阻 |
| 饱和区 | `VGS > Vth` 且 `VDS >= VGS - Vth` | 漏端夹断，器件近似受控电流源 |

常见长沟道近似模型：

```text
线性区：ID = μn Cox (W/L) [(VGS - Vth)VDS - VDS^2/2]
饱和区：ID = 1/2 μn Cox (W/L) (VGS - Vth)^2
```

其中 `VGS` 为栅源电压，`VDS` 为漏源电压，`Vth` 为阈值电压，`W/L` 为宽长比。

## 3. 分析方法

1. 明确器件类型：先判断是 NMOS 还是 PMOS，并统一端口电压方向。
2. 判断是否导通：对 NMOS 检查 `VGS > Vth`；对 PMOS 需换成源栅、源漏电压关系。
3. 判断工作区：导通后比较 `VDS` 与 `VGS - Vth`。
4. 结合电路目标解释结果：放大电路关注是否在饱和区，开关电路关注导通电阻和截止状态。
5. 若题目给出偏置网络，先求直流工作点，再判断工作区，最后再进行小信号分析。

## 4. 常见易错点

- 只判断 `VGS > Vth`，直接得出“饱和区”结论。
- 将线性区条件 `VDS < VGS - Vth` 与饱和区条件写反。
- 用 NMOS 的电压方向直接套到 PMOS 上，导致符号错误。
- 在放大电路中跳过静态工作点判断，直接套用小信号模型。
- 忽略题目中的体效应、沟道长度调制等附加条件；第一版课程问答中默认使用基础长沟道模型，除非题目明确给出高级模型参数。

## 5. 典型例题

### 例题 1：判断 NMOS 工作区

已知增强型 NMOS 的 `Vth = 1 V`，`VGS = 3 V`，`VDS = 2.5 V`，判断工作区。

分析：

1. `VGS = 3 V > Vth = 1 V`，器件导通。
2. `VGS - Vth = 2 V`。
3. `VDS = 2.5 V >= 2 V`。

结论：MOS 管工作在饱和区。

### 例题 2：常见错误诊断

学生回答：“`VGS > Vth`，所以该 MOS 管在饱和区。”

诊断：该答案遗漏了 `VDS >= VGS - Vth` 条件。正确流程应先判断导通，再判断漏源电压是否满足饱和条件。

## 6. 推荐复习建议

- 先用表格记忆三个工作区的判断条件。
- 做题时固定写出两步：导通判断、工作区判断。
- 将 MOS 工作区与“放大电路静态工作点”“CMOS 开关模型”联系起来复习。
- 将错题按“条件遗漏”“符号方向错误”“区间边界错误”归类，便于后续导入错题诊断 Agent。
