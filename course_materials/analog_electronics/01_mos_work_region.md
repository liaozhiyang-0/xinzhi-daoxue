# MOS 管工作区

## 1. 基本概念

MOS 管工作区描述 MOS 管在不同栅源电压和漏源电压条件下的导通状态。模拟电子课程中常用工作区包括截止区、线性区和饱和区。判断工作区是分析放大电路、开关电路和偏置点的基础。

## 2. 核心公式或判断条件

以增强型 NMOS 为例：

- 截止区：`VGS <= Vth`
- 线性区：`VGS > Vth` 且 `0 <= VDS < VGS - Vth`
- 饱和区：`VGS > Vth` 且 `VDS >= VGS - Vth`

其中 `VGS` 为栅源电压，`VDS` 为漏源电压，`Vth` 为阈值电压。

## 3. 分析方法

1. 先判断 `VGS` 是否超过阈值电压。
2. 若未超过阈值电压，则 MOS 管截止。
3. 若超过阈值电压，再比较 `VDS` 与 `VGS - Vth`。
4. 根据比较结果判断线性区或饱和区。

## 4. 常见易错点

- 只判断 `VGS > Vth`，忽略 `VDS` 条件。
- 将线性区和饱和区的边界条件写反。
- 未区分 NMOS 与 PMOS 的电压方向。
- 在小信号分析前没有先确认直流工作点。

## 5. 典型例题

已知增强型 NMOS 的 `Vth = 1 V`，`VGS = 3 V`，`VDS = 2.5 V`。因为 `VGS - Vth = 2 V`，且 `VDS >= 2 V`，所以 MOS 管工作在饱和区。

## 6. 推荐复习建议

建议先熟记三个工作区的判断条件，再通过直流偏置例题训练判断流程。复习时应将工作区判断与放大电路静态工作点联系起来。
