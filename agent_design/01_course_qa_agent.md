# 课程问答 Agent

## Agent 目标

面向电子信息课程群，为学生提供基于课程资料和知识库的概念解释、公式说明、分析步骤和易错提醒。

## 输入

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| user_id | string | 学生标识 |
| course_id | string | 课程标识 |
| knowledge_point_id | string | 可选知识点标识 |
| question | string | 学生问题 |
| history | array | 可选历史问答上下文 |

## 处理流程

1. 识别问题所属课程与知识点。
2. 检索课程资料、题库和错题样例。
3. 组织概念、公式、步骤和易错点。
4. 生成结构化答案。
5. 返回引用知识点和后续学习建议。

## 调用资源

- 课程资料 Markdown。
- 知识点目录。
- 题库样例。
- 历史问答记录。

## 输出格式

```json
{
  "answer": "面向学生的直接回答",
  "key_points": ["关键点1", "关键点2"],
  "formula_or_rules": ["公式或判断条件"],
  "common_mistakes": ["常见错误"],
  "recommended_review": ["复习建议"],
  "references": ["知识点编号或资料路径"]
}
```

## Prompt 初稿

```text
你是电子信息课程群智能导学助手。请基于课程资料回答学生问题，回答应准确、清晰、适合本科课程学习。请优先给出概念解释、公式或判断条件、分析步骤、常见易错点和复习建议。若资料不足，请明确说明不确定范围，不要编造结论。
```

## 与自研系统的接口关系

自研后端通过 `/api/v1/student/qa` 接收学生问题，补充课程、知识点和历史上下文后调用星辰 Agent。Agent 返回结构化结果，后端保存 `qa_record` 并返回前端。
