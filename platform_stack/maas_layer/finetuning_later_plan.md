# 后续微调计划

## 1. 当前阶段结论

当前阶段不优先做微调。项目应先完成识图解题 Agent、课程知识库检索问答、批量推理和模型评估闭环。

## 2. 微调前置条件

只有在满足以下条件后再考虑微调：

- 已积累足够高质量图文解题样例；
- 图像识别、结构还原和解题推理错误类型已经清楚分类；
- Prompt 工程无法继续明显提升效果；
- 已明确微调目标是多模态识图、文本解题，还是教学解释风格优化；
- 已有评估集和回归测试集。

## 3. 可选微调方向

- 多模态图文解题微调；
- 文本结构化解题微调；
- 错因提示和教学反馈语微调；
- 课程问答风格微调。

## 4. 数据来源

- 经人工审核的识图解题样例；
- v0.2 结构化题库样例；
- 课程问答高质量对话；
- 错因标签和简易纠错反馈；
- 教师确认的标准解析。

## 5. 风险

- 样例质量不足会放大错误；
- 题图和题干版权需确认；
- 微调成本高于 Prompt 调整；
- 若没有稳定评估集，无法证明微调有效。
