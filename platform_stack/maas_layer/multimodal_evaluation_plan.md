# MaaS 多模态图文解题评估计划

## 1. 定位变化

MaaS 层重点从“错题诊断数据增强”调整为“图文解题评估”。第一阶段不做微调，先使用图文解题样例、批量推理和人工/模型辅助评估验证识图解题能力。

## 2. 评估维度

1. 图片文字识别是否准确；
2. 电路元件识别是否准确；
3. 节点/支路关系是否还原；
4. 参考方向是否提取；
5. 章节和方法选择是否正确；
6. 关键方程是否正确；
7. 计算结果是否正确；
8. 解释是否适合学生理解；
9. 不确定信息是否主动标注；
10. 是否出现幻觉或编造电路结构。

## 3. 建议测试集结构

完整测试集建议：

- 10 道电路理论题图；
- 5 道模拟电子技术题图；
- 5 道数字电子技术题图；
- 5 道信号与系统波形/框图题。

第一阶段只做电路理论，优先覆盖：

- RC/RL 一阶换路；
- RLC 二阶暂态；
- 正弦稳态相量网络；
- 正弦稳态功率；
- 三相 Y/Delta 连接；
- 磁耦合同名端。

## 4. 评估输出格式

```json
{
  "case_id": "VS-CT-001",
  "ocr_score": 0,
  "circuit_structure_score": 0,
  "method_selection_score": 0,
  "equation_score": 0,
  "calculation_score": 0,
  "teaching_explanation_score": 0,
  "uncertainty_handling_score": 0,
  "hallucination_flag": false,
  "review_comment": "简短评价"
}
```

## 5. 第一阶段通过标准

- 不编造看不清的电路结构；
- 能稳定输出节点表、支路表或 netlist；
- 电路理论样例中关键方程正确率达到可演示水平；
- 对不确定图片主动要求补图或补充条件。
