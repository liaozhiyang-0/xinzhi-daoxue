# 学习规划 Agent 设计

## 1. 定位

学习规划 Agent 是辅助展示功能，不再作为第一优先级核心能力。它根据学生问答记录、识图解题记录、简单纠错提示和薄弱知识点，为学生生成短周期复习建议。

## 2. 输入

- 学生最近问题记录；
- 识图解题历史；
- 简易错因提示；
- 课程章节掌握情况；
- 学生目标，例如“期末复习”“补第8章暂态分析”。

## 3. 输出

- 重点复习章节；
- 每日任务；
- 推荐题型；
- 易错点提醒；
- 掌握检查方式。

## 4. 与其他 Agent 的关系

- 接收课程问答 Agent 的知识点记录；
- 接收识图解题 Agent 的题型和章节判断；
- 使用错因标签作为薄弱点辅助信号；
- 为教师分析 Agent 提供个人层面的学习建议摘要。

## 5. 第一阶段边界

第一阶段只做演示级学习规划，不做复杂自适应学习系统，不引入额外推荐算法依赖。
