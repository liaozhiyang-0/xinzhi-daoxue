# 识图解题 Agent 设计

## 1. Agent 定位

识图解题 Agent 是系统核心智能体，面向电子信息类课程中的图形化题目，尤其是电路理论、模拟电子技术、数字电子技术、信号与系统等课程。

在《电路理论》中，它主要处理：

- 电路图截图；
- 题目截图；
- 手写或打印题干；
- 元件参数；
- 电压电流参考方向；
- 节点和支路关系；
- 开关状态；
- 相量关系；
- 三相连接；
- 互感同名端；
- 变压器端口。

识图解题 Agent 的核心目标不是只给最终答案，而是将题图转换为可解释的结构化电路描述，再完成分步推理。

## 2. 输入

| 输入项 | 是否必填 | 说明 |
|---|---|---|
| 题目截图 | 是 | 可以包含题干、图、表格或手写批注。 |
| 电路图截图 | 是 | 可与题目截图合并，也可单独上传。 |
| 学生补充文字 | 否 | 用于说明题目要求、已知条件或自己的疑问。 |
| 课程章节范围 | 否 | 例如“电路理论第8章”。 |
| 答案详细程度 | 否 | 可选“详细讲解”或“只要答案”。 |

## 3. 输出

统一输出：

- 题目识别；
- 电路结构提取；
- 已知条件；
- 待求量；
- 所属章节；
- 解题方法选择；
- 关键方程；
- 分步计算；
- 最终答案；
- 易错提醒；
- 推荐复习章节。

## 4. 内部流程

### 节点1：图片内容识别

识别题干文字、图中元件、数值、单位、节点标记、方向箭头、开关状态和端口标注。

输出：

- OCR 题干；
- 图中元件清单；
- 可读参数；
- 看不清或有歧义的位置。

### 节点2：电路结构标准化

将图像信息转为可检索、可推理的结构化表示。

必须输出：

- 节点表；
- 支路表；
- netlist；
- 参考方向表；
- 不确定信息清单。

### 节点3：章节/题型判断

根据电路结构和题目目标判断所属课程、章节、题型和可调用知识库文件。

示例：

- RC/RL 换路题：`chapters/08_first_order_transient_analysis.md`
- RLC 二阶暂态题：`chapters/09_second_order_transient_analysis.md`
- 三相 Y/Delta 连接题：`chapters/12_three_phase_sinusoidal_circuits.md`

### 节点4：解题推理

选择合适方法并列出关键方程。对电路理论题，常见方法包括节点法、网孔法、三要素法、相量法、功率法、分相计算和互感方程。

### 节点5：答案整理与教学解释

将推理结果整理为学生可读输出，包括最终答案、步骤解释、易错提醒和复习建议。若图像不清或条件不足，应主动标注不确定信息，不猜测电路结构。

## 5. 标准输出模板

```markdown
# 识图解题结果

## 1. 题目识别

## 2. 电路结构

### 2.1 节点说明

### 2.2 支路参数表

### 2.3 参考方向

### 2.4 不确定信息

## 3. 解题方法选择

## 4. 关键方程

## 5. 分步计算

## 6. 最终答案

## 7. 易错提醒

## 8. 推荐复习内容
```

## 6. 适用课程扩展

- 模拟电子技术：放大电路识别、反馈判断、频率响应；
- 数字电子技术：逻辑图、时序图、卡诺图；
- 信号与系统：系统框图、波形图、频谱图；
- 通信原理：调制框图、星座图、频谱图；
- 嵌入式系统：接口连接图、时序图、流程图。

## 7. 质量约束

- 不猜测看不清的元件、数值或连接关系；
- 先还原电路结构，再列方程；
- 对参考方向、单位和相量有效值约定必须显式说明；
- 对不确定信息给出补图或补充条件请求；
- 最终答案必须能追溯到关键方程。
