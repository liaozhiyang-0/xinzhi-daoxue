# 教师分析 Agent 设计

## 1. 定位

教师分析 Agent 是辅助展示功能，用于将学生问答、识图解题、简单纠错和学习规划数据聚合为班级层面的教学观察结果。

## 2. 输入

- 班级问题统计；
- 题图解题题型分布；
- 高频知识点；
- 高频错因标签；
- 学生学习计划完成情况；
- 教师选择的课程和时间范围。

## 3. 输出

- 高频问题摘要；
- 薄弱章节分布；
- 易错点分布；
- 可补充讲解的题型；
- 作业或课堂活动建议；
- 后续观察指标。

## 4. 与其他 Agent 的关系

- 使用识图解题 Agent 的章节和题型识别结果；
- 使用课程问答 Agent 的高频概念问题；
- 使用简易纠错模块输出的错因标签；
- 使用学习规划 Agent 的个人建议汇总。

## 5. 第一阶段边界

第一阶段使用 mock 数据和聚合统计展示，不直接连接真实教务系统，不做学生排名和高风险预测。
