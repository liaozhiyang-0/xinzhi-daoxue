# 课程问答案例

## 案例 1：公式适用条件

- 用户问题：三要素法是不是所有换路电路都能用？
- 期望输出：说明一阶线性电路、直流或阶跃激励等适用条件，并提示二阶电路不能直接套用。
- 对应 Agent：课程问答 Agent。

## 案例 2：简单纠错

- 用户问题：我求时间常数时把受控源也关掉，为什么错？
- 期望输出：解释独立源置零、受控源保留，并提示 `dependent_source_zeroed_error`。
- 对应 Agent：课程问答 Agent 简易纠错模块。

## 案例 3：章节检索

- 用户问题：复功率为什么要对电流取共轭？
- 期望输出：解释 `S=UI*`，说明不取共轭会导致无功符号错误，并推荐第11章。
- 对应 Agent：课程问答 Agent。

## 案例 4：转交识图解题

- 用户问题：这张电路图怎么列节点方程？
- 期望输出：如果包含图片，转交识图解题 Agent；如果没有图片，要求补充电路图或节点连接关系。
- 对应 Agent：工作流分流。
