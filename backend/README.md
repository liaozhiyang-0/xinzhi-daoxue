# 后端说明

## 技术选型

第一版后端可采用 Spring Boot 或 FastAPI。若团队更熟悉 Java 生态，可优先选择 Spring Boot；若需要快速原型和 Agent API 调用，可优先选择 FastAPI。

## 主要职责

- 提供学生端和教师端 REST API。
- 管理课程、知识点、题库、问答记录、错题记录和学习画像。
- 封装星辰 Agent API 调用。
- 生成教师端统计指标。
- 统一处理鉴权、日志、异常、限流和重试。

## 运行配置

后端配置从环境变量读取，参考根目录 `.env.example`。禁止在代码中硬编码数据库密码或平台 API Key。
