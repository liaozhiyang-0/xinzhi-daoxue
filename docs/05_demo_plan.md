# 第一版演示计划

## 演示链路

### 1. 识图解题

学生上传电路图或题目截图，例如 RC 换路题、RLC 暂态题或三相电路题。系统调用识图解题 Agent，返回题目识别、电路结构、关键方程、分步计算、最终答案和易错提醒。

### 2. 学生智能答疑

学生选择课程和知识点，输入问题，例如“MOS 管饱和区如何判断？”。系统调用课程问答 Agent，返回概念解释、判断条件、分析步骤和易错提醒。

### 3. 简单纠错

学生提交某一步推导或公式使用疑问。课程问答 Agent 结合错因标签给出轻量错因提示，例如“受控源不能置零”“线电压不能直接当相电压使用”。

### 4. 学习建议

系统汇总学生近期问答、题图解题记录和错因提示，生成针对薄弱知识点的复习建议，包括复习顺序、重点公式、例题类型和练习建议。

### 5. 教师端看板

教师查看班级高频问题 Top 10、错因标签分布、薄弱知识点统计、识图解题失败点和学生活跃度统计，快速定位共性教学难点。

### 6. 教学建议生成

系统基于班级学情生成教学建议，例如建议增加 MOS 工作区判断专题讲解，补充 CMOS 反相器噪声容限例题，或调整触发器时序分析练习比例。

## 演示准备

- 准备 3 到 5 个典型知识点问题。
- 准备 6 到 10 个题图识别与分步解题样例。
- 准备 2 到 3 条简单纠错样例。
- 准备一份模拟班级统计数据。
- 准备学生端和教师端页面截图或原型说明。
