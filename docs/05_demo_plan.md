# 第一版演示计划

## 演示链路

### 1. 学生智能答疑

学生选择课程和知识点，输入问题，例如“MOS 管饱和区如何判断？”。系统调用课程问答 Agent，返回概念解释、判断条件、分析步骤和易错提醒。

### 2. 错题诊断

学生提交题目、自己的答案和标准答案。系统调用错题诊断 Agent，识别错误类型，例如“判断条件遗漏”“公式适用范围错误”或“步骤缺失”，并给出修正建议。

### 3. 学习建议

系统汇总学生近期问答和错题记录，生成针对薄弱知识点的复习建议，包括复习顺序、重点公式、例题类型和练习建议。

### 4. 教师端看板

教师查看班级高频问题 Top 10、错题类型分布、薄弱知识点统计和学生活跃度统计，快速定位共性教学难点。

### 5. 教学建议生成

系统基于班级学情生成教学建议，例如建议增加 MOS 工作区判断专题讲解，补充 CMOS 反相器噪声容限例题，或调整触发器时序分析练习比例。

## 演示准备

- 准备 3 到 5 个典型知识点问题。
- 准备 2 到 3 条错题诊断样例。
- 准备一份模拟班级统计数据。
- 准备学生端和教师端页面截图或原型说明。
