# 第一版范围

## 第一版目标

第一版目标是完成“学生提问 - Agent 回答 - 错题诊断 - 学习建议 - 教师看板”的基础闭环，形成可演示、可答辩、可继续迭代的项目雏形。

## 第一版实现功能

- 学生智能答疑：支持电子信息课程知识点问答。
- 错题诊断：支持学生提交题目、作答和标准答案，返回错误原因与改进建议。
- 学习建议：基于错题和薄弱知识点生成复习计划。
- 教师端学情看板：展示高频问题、错题类型、薄弱知识点和教学建议。

## 第一版暂不实现内容

- 完整教务系统、排课系统和成绩管理系统。
- 大规模多租户权限体系。
- 复杂在线考试与自动阅卷系统。
- 高并发生产级部署和 Kubernetes 集群。
- 全量模型训练流水线，仅保留 MaaS 微调数据格式和评估方案。

## 第一版验收标准

- 仓库结构清晰，文档、课程资料、接口、数据库和部署模板完整。
- 学生端和教师端原型说明覆盖核心交互。
- 后端 API 设计能支撑第一版功能闭环。
- Agent 设计文档明确输入、流程、资源、输出和接口关系。
- 不包含真实密钥，不提交 `.env` 文件。
