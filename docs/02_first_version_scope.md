# 第一版范围

## 第一版目标

第一版目标是完成“题图上传 - 识图解题 - 课程问答/简单纠错 - 学习建议 - 教师看板”的基础闭环，形成可演示、可答辩、可继续迭代的项目雏形。

## 第一版实现功能

- 识图解题：支持学生上传题目截图或电路图，返回结构提取、关键方程、分步解题和易错提醒。
- 学生智能答疑：支持电子信息课程知识点问答、公式条件解释和学习追问。
- 简单纠错：支持学生提交某一步推导或公式使用疑问，返回轻量错因提示和复习建议。
- 学习建议：基于问答、题图解题记录和薄弱知识点生成复习计划。
- 教师端学情看板：展示高频问题、错因标签、薄弱知识点和教学建议。

## 第一版暂不实现内容

- 完整教务系统、排课系统和成绩管理系统。
- 大规模多租户权限体系。
- 复杂在线考试与自动阅卷系统。
- 高并发生产级部署和 Kubernetes 集群。
- 全量模型训练流水线，仅保留 MaaS 图文解题评估、批量推理和后续微调计划。

## 第一版验收标准

- 仓库结构清晰，文档、课程资料、接口、数据库和部署模板完整。
- 学生端和教师端原型说明覆盖核心交互。
- 后端 API 设计能支撑第一版功能闭环。
- `platform_stack/` 明确 Agent 层、MaaS 层、工作流分流和演示案例。
- 不包含真实密钥，不提交 `.env` 文件。
